- 检索中国裁判文书网（wenshu.court.gov.cn）与最高法官网典型案例，按案由/案号/关键词/法院/年份查询裁判文书
- 面向律师、法官、检察官、企业法务等法律实务场景，定位「相关裁判文书是否存在、案号/法院/裁判日期」
- 输入关键词、案由、案号、法院、年份等结构化条件，返回匹配的裁判文书记录列表
- 每条结果包含：标题、案号、法院、案由、裁判日期、来源链接
- 中国裁判文书网无开放 JSON 接口、受强风控保护：本工具尽力请求，返回非 JSON 或不可达时优雅降级为「权威深链 + 检索指引」，而非报错

使用要点：
- 引用案例前务必核对案号、法院与裁判日期，并区分生效裁判与一审未生效裁判
- 本工具返回检索记录与摘要；如需裁判要旨全文，请对来源链接调用 webfetch 抓取后再逐份分析
- 同案同判类论证应交叉多份案例，避免依赖单一孤例
