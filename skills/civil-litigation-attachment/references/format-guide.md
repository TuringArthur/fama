# 附带民事起诉状格式与示例参考

## 一、概念界定

### 1.1 什么是附带民事诉讼

附带民事诉讼是一种依附于刑事诉讼的特殊民事诉讼，是指因受犯罪行为侵害的被害人在刑事诉讼过程中，以犯罪行为人为被告人提起的要求其承担赔偿责任的民事诉讼。

### 1.2 当事人称谓

| 角色 | 称谓 |
|------|------|
| 提起诉讼的被害人 | 附带民事诉讼原告人 |
| 被诉承担赔偿责任的犯罪行为人 | 附带民事诉讼被告人 |

### 1.3 文书定义

附带民事起诉状，是指刑事案件的被害人及其法定代理人或者近亲属，在刑事诉讼进行过程中，向人民法院提出诉讼请求，要求人民法院判令被告人赔偿因其犯罪行为导致附带民事诉讼原告人遭受的物质损失的法律文书。

### 1.4 适用范围

**适用情况**：
- 被害人因被告人的犯罪行为而遭受物质损失
- 被害人请求人民法院判令行为人承担经济赔偿责任

**不适用情况**：
- 精神损害赔偿（应另行提起民事诉讼）
- 逾期未提起（应在刑事诉讼期间提出）

## 二、格式规范（8.3.1）

### 2.1 首部

#### （1）标题
在文书顶端居中书写"附带民事起诉状"。

#### （2）当事人基本情况

**自然人当事人**：
```
原告人：XXX，男/女，XXXX年XX月XX日出生，X族，籍贯XX省XX市，
职业：XX，工作单位及职务：XXXX，住所：XXXXXX

被告人：XXX，男/女，XXXX年XX月XX日出生，X族，籍贯XX省XX市，
职业：XX，工作单位及职务：XXXX，住所：XXXXXX
```

**无民事行为能力人或限制民事行为能力人**：
在当事人基本情况后，另起一行写明：
```
法定代理人：XXX（系原告人/被告人XXX之父/母/其他关系），男/女，
职业：XX，工作单位及职务：XXXX，住址：XXXXXX
```

**法人或其他组织**：
```
原告人/被告人：XXXX（单位名称）
所在地：XXXXXX（邮政编码：XXXXXX）
企业性质：XXXXXX
统一社会信用代码：XXXXXX
经营范围和方式：XXXXXX
开户银行和账号：XXXXXX

法定代表人（主要负责人）：XXX，职务：XXXXXX
```

**诉讼代理人**：
在委托人项下另起一行写明：
```
委托诉讼代理人：XXX，男/女，工作单位：XXXXXX
```

### 2.2 正文

#### （1）诉讼请求

```
诉讼请求

一、请求人民法院依法追究被告人XXX的刑事责任；
二、请求人民法院依法判令被告人XXX赔偿因其犯罪行为给原告人造成的
物质损失共计人民币XXXXX元（￥XXXXX.XX元）。
```

#### （2）事实

```
事实与理由

一、被告人犯罪事实

XXXX年XX月XX日XX时许，在XXXX（地点），被告人XXX因XXXX（动机），
采用XXXX（手段），实施了XXXX（具体经过），导致XXXX（后果）。

二、原告人物质损失情况

被告人XXX的犯罪行为给原告人造成了以下物质损失：

（一）积极损失：
1. 医疗费：人民币XXXXX元；
2. 护理费：人民币XXXXX元；
3. 误工费：人民币XXXXX元；
4. 住院伙食补助费：人民币XXXXX元；
5. 残疾赔偿金：人民币XXXXX元（如涉及）；
6. 财产损失：人民币XXXXX元（如涉及）；
7. 其他损失：人民币XXXXX元。

（二）消极损失：
1. 后续治疗费：人民币XXXXX元；
2. 康复费：人民币XXXXX元；
3. 残疾辅助器具费：人民币XXXXX元（如涉及）；
4. 其他预期损失：人民币XXXXX元。

上述物质损失共计：人民币XXXXX元（￥XXXXX.XX元）。

三、因果关系

原告人的上述物质损失与被告人XXX的犯罪行为存在直接的因果关系，
有XXXX（证据名称）等证据予以证实。
```

#### （3）理由

```
理由

被告人XXX的行为已触犯《中华人民共和国刑法》第XXX条之规定，
构成XXX罪，依法应当追究其刑事责任。

同时，被告人XXX的犯罪行为给原告人造成了巨大的物质损失，
共计人民币XXXXX元。根据《中华人民共和国民法典》第XXX条之规定，
被告人XXX应当承担民事赔偿责任。

综上，为维护原告人的合法权益，请求人民法院依法支持原告人的
全部诉讼请求。
```

#### （4）证据

```
证据

原告人为证明其诉讼请求，向人民法院提交以下证据：

1. 书证：
   （1）医疗费票据原件X份，证明原告人支出医疗费XXXXX元；
   （2）误工证明原件X份，证明原告人误工损失XXXXX元；
   （3）伤残鉴定报告原件X份，证明原告人伤残等级为X级；
   （4）财产损失清单原件X份，证明原告人财产损失XXXXX元；
   （5）其他书证XXXXXX。

2. 物证：
   （1）XXXXXX；
   （2）XXXXXX。

3. 证人证言：
   （1）证人XXX（身份证号：XXXXXX）出具的书面证言原件X份；
   （2）证人XXX出庭作证申请书。

4. 鉴定意见：
   （1）司法鉴定意见书原件X份，证明原告人伤残等级及后续治疗费用；
   （2）财产损失评估报告原件X份。

5. 当事人陈述：
   原告人陈述原件X份，陈述遭受侵害及损失的具体情况。

6. 视听资料/电子数据：
   （1）监控录像光盘X张；
   （2）照片X张；
   （3）其他电子数据XXXXXX。

以上证据能够证明：
第一，原告人遭受被告人侵害行为侵害的事实；
第二，因该侵害行为导致原告人物质损失的事实；
第三，该损失的程度及数额的情况。
```


## 三、写作注意事项（8.3.2）

### 3.1 时间限制

- 只能在检察机关向人民法院提起公诉之后，人民法院立案受理至第一审终结前制作和提交
- 逾期不得提起附带民事诉讼，应当单独提起民事诉讼

### 3.2 赔偿范围限制

| 可以主张 | 不可主张 |
|----------|----------|
| 医疗费 | 精神损害赔偿 |
| 护理费 | 死亡赔偿金（交通肇事罪等特定罪名除外） |
| 误工费 | 营养费（除非有医嘱） |
| 残疾赔偿金 | |
| 财产损失 | |
| 后续治疗费 | |

### 3.3 精神损害赔偿的处理

如果被害人请求精神损害赔偿：
- 不能在附带民事起诉状中提出
- 附带民事诉讼只限于赔偿物质损失
- 可以另行制作民事起诉状单独提起民事诉讼

## 四、常见问题解答

### Q1：谁可以作为附带民事诉讼原告人？

- 被害人本人
- 被害人的法定代理人（被害人为无民事行为能力人或限制民事行为能力人时）
- 被害人的近亲属（被害人死亡的等情况下）

### Q2：哪些犯罪类型可以提起附带民事诉讼？

原则上，所有犯罪行为造成物质损失的案件都可以提起附带民事诉讼，但部分罪名有特殊规定。

### Q3：损失金额如何确定？

- 有票据的以票据为准
- 无法提供票据的，可以申请司法鉴定或评估
- 误工费需要提供收入证明
- 后续治疗费需要提供医嘱或鉴定意见

### Q4：被告人无赔偿能力怎么办？

可以在判决中确认赔偿数额，待被告人具备执行能力时再申请强制执行。
