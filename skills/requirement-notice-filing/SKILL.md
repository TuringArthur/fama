# 要求说明立案理由通知书

## 概述

本技能用于生成人民检察院向公安机关发出的《要求说明立案理由通知书》，适用于检察机关认为公安机关立案决定不当，需要其书面说明立案理由的法律监督场景。

## 法律依据

- **《人民检察院刑事诉讼规则》** 第五百五十九条、第五百六十条
- **《最高人民检察院、公安部关于刑事立案监督有关问题的规定（试行）》** 第六条

## 适用情形

人民检察院经审查，认为需要公安机关说明立案理由的，应当要求说明。具体包括：
- 有证据证明公安机关可能存在违法动用刑事手段插手民事、经济纠纷
- 利用立案实施报复陷害、敲诈勒索
- 谋取其他非法利益等违法立案情形
- 尚未提请批准逮捕或者移送起诉的，应当要求公安机关书面说明立案理由

## 文书结构

### 1. 首部 (Header)

| 要素 | 说明 |
|------|------|
| 标题 | **要求说明立案理由通知书**（居中，黑体字） |
| 文号 | `××检刑立通〔××××〕××号`（位于标题右下角） |

### 2. 正文 (Body)

| 要素 | 说明 |
|------|------|
| 受文单位 | 接收通知书的公安机关名称 |
| 案件信息 | 案件发生时间、案件名称 |
| 法律依据 | 引用具体法律法规条款 |
| 监督理由 | 阐述立案决定不当的具体原因 |
| 具体要求 | 要求在**7日以内**提交书面说明 |

### 3. 尾部 (Footer)

| 要素 | 说明 |
|------|------|
| 落款日期 | 20××年×月×日 |
| 印章 | 加盖人民检察院院印 |

## 核心要求

1. **时限要求**：公安机关必须在收到通知书后的 **7日以内** 作出答复
2. **形式要求**：答复必须是 **书面说明**，不能仅口头回复
3. **内容要求**：必须详细说明针对特定案件进行 **立案的理由**
4. **材料要求**：应连同有关证据材料一并回复

## 标准模板

```
要求说明立案理由通知书

××检刑立通〔20××〕×号

________公安局：

根据《人民检察院刑事诉讼规则》第五百五十九条、第五百六十条和《最高人民检察院、公安部关于刑事立案监督有关问题的规定（试行）》第六条的规定，现要求你们对________一案说明立案理由。请在收到本通知书之日起七日以内，向本院提交书面说明，并连同有关证据材料一并回复。

                    ××××人民检察院（院印）

                        20××年×月×日
```

## 使用说明

1. 填写标题中的检察机关名称
2. 填写文号（年份使用四位数字）
3. 填写接收通知书的公安机关全称
4. 填写具体的案件名称
5. 加盖人民检察院院印
6. 注明准确的发出日期

## 注意事项

- 通知书内容应当事实清楚、法律依据充分
- 监督理由应当具体明确，不得含糊不清
- 严格按照法定时限要求公安机关答复
- 收到书面说明后应当及时审查并作出相应处理
