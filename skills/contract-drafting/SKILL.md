# 合同起草助手

## 核心能力

### 1. 合同类型识别与生成

根据用户需求，自动识别并生成以下类型的合同文书：

| 合同类型 | 主要条款 | 适用场景 |
|---------|---------|---------|
| **劳动合同** | 合同期限、工作内容、工作地点、工作时间、劳动报酬、社会保险、劳动保护、解除终止等 | 雇主与员工建立劳动关系 |
| **购销合同** | 产品明细、质量要求、交货方式、验收标准、付款方式、违约责任等 | 产品买卖双方 |
| **合伙合同** | 出资方式、财务管理、盈余分配、债务承担、入伙退伙、权利义务等 | 多人合伙经营 |
| **技术合同** | 技术内容、交付方式、验收标准、保密条款、知识产权归属等 | 技术开发、转让、咨询 |
| **租赁合同** | 租赁物、租金、期限、维修责任、违约责任等 | 房屋、设备等租赁 |
| **委托合同** | 委托事项、权限范围、费用承担、报告义务、违约责任等 | 代理、经纪等委托事务 |

### 2. 合同结构框架

标准合同应包含以下基本结构：

```
1. 合同标题（居中）
2. 当事人信息（甲方/乙方）
3. 鉴于条款（合同背景）
4. 定义与解释
5. 合同主体条款
   - 权利义务
   - 履行方式
   - 期限约定
   - 费用支付
6. 违约责任
7. 争议解决
8. 其他约定
9. 签署栏
   - 甲方（盖章）________ 乙方（盖章）________
   - 日期：____年____月____日
10. 附件说明
```

### 3. 合同条款标准表述

#### 合同期限
```
甲、乙双方选择以下第____种形式确定本合同期限：
（一）固定期限：自____年____月____日起至____年____月____日止。
（二）无固定期限：自____年____月____日起至法定的终止条件出现时止。
（三）以完成一定工作任务为期限：自____年____月____日起至工作任务完成时止。
```

#### 工作内容
```
根据甲方工作需要，乙方同意从事____岗位（工种或职务）工作。
经甲、乙双方协商同意，可以变更工作岗位（工种或职务）。
乙方应按照甲方的要求，按时完成规定的工作数量，达到规定的质量标准。
```

#### 劳动报酬
```
甲方应根据本单位的工资制度，确定乙方实行以下第____种工资形式：
（一）计时工资：由以下几部分组成：____、____、____；其标准分别为____元/月。
（二）计件工资：甲方应制定科学合理的劳动定额标准，计件单价约定为____。
（三）其他工资形式：具体约定在本合同____中明确。
```

#### 违约责任
```
（一）甲方违约责任：
1. 未按照合同约定提供劳动保护或劳动条件的；
2. 未及时足额支付劳动报酬的；
3. 未依法为乙方缴纳社会保险费的；
（二）乙方违约责任：
1. 违反本合同约定解除合同的；
2. 违反保密义务或竞业限制约定的；
3. 给甲方造成其他损失的。
```

#### 争议解决
```
因本合同的履行或与本合同有关的一切纠纷，甲乙双方应协商解决；
协商不成的，可提交____人民法院仲裁/诉讼。
```

### 4. 合同审查要点

#### 形式审查
- [ ] 合同名称与内容是否一致
- [ ] 当事人信息是否完整准确
- [ ] 合同编号和日期是否正确
- [ ] 签署栏是否完整（盖章、签名、日期）

#### 实质审查
- [ ] 合同条款是否完整（必备条款是否齐全）
- [ ] 权利义务是否对等
- [ ] 违约责任是否明确
- [ ] 争议解决方式是否可行
- [ ] 生效条件是否明确
- [ ] 附件是否完整

#### 风险审查
- [ ] 是否存在霸王条款
- [ ] 是否存在显失公平的条款
- [ ] 违约金是否过高或过低
- [ ] 免责条款是否合理
- [ ] 知识产权归属是否明确
- [ ] 保密条款是否完善

### 5. 常用法律依据

#### 劳动合同相关
- 《中华人民共和国劳动法》
- 《中华人民共和国劳动合同法》
- 《中华人民共和国社会保险法》
- 《工资支付暂行规定》
- 《职工带薪年休假条例》

#### 合同通用条款
- 《中华人民共和国民法典》（合同编）
- 《最高人民法院关于适用〈中华人民共和国民法典〉合同编通则若干问题的解释》

### 6. 输出格式

根据用户需求，可输出以下格式：

**完整合同文本**：包含所有条款的正式合同文档

**合同条款清单**：分条列项的合同条款大纲

**合同审查报告**：对现有合同的审查意见和改进建议

**合同要点摘要**：合同的核心条款和关键信息提取

### 7. 使用示例

```
用户：请帮我起草一份劳动合同
助手：好的，请问您需要以下信息：
- 雇主（甲方）名称和基本信息
- 员工（乙方）姓名和基本信息
- 合同期限（固定期限/无固定期限）
- 工作岗位和地点
- 工资待遇和支付方式
- 其他特殊约定

用户提供信息后，助手将生成完整的劳动合同文本。
```

```
用户：请审查这份购销合同（粘贴合同内容）
助手：根据审查要点，我将从以下几个方面进行审查：
1. 形式审查
2. 实质审查
3. 风险审查
[提供详细的审查报告和改进建议]
```

## 注意事项

1. **信息完整**：生成合同前，请尽可能提供完整的基本信息
2. **条款定制**：可根据具体交易情况调整和增删条款
3. **合规性**：确保合同条款符合现行法律法规要求
4. **审慎使用**：本合同模板仅供参考，正式合同建议咨询专业律师
5. **版本控制**：重要合同建议保留历史版本和修改记录
