# 仲裁裁决书格式规范详解

## 一、仲裁裁决书概念与功能

仲裁裁决书是仲裁机构对当事人申请仲裁的纠纷争议所作的书面决定。

**功能作用**：
1. 有助于解决纠纷，维护当事人合法权益
2. 有效地约束双方当事人，具有法律效力
3. 不服可向法院申请强制执行

**裁决类型**：
- 中间裁决
- 部分裁决
- 最终裁决（本节主要介绍最终裁决）

最终裁决标志着仲裁程序的终结，对争议双方具有约束力。

## 三、裁决金额计算示例

### 【案例】病假工资争议

**案情**：申请人月工资14,720.00元，2018年1月6日受伤住院。

**计算依据**：
- 《山东省企业工资支付规定》：停工医疗累计不超过180天的，发给本人工资70%的病假工资

**计算公式**：
```
病假工资 = 月工资 × 70% × 病假月数
```

### 【案例】违法解除劳动合同赔偿金

**法律依据**：《中华人民共和国劳动合同法》第四十七条、第八十七条

**计算公式**：
```
经济补偿 = 月平均工资 × 工作年限（月）
赔偿金 = 经济补偿 × 2倍
```

**工作年限认定**：
- 每满一年支付一个月工资
- 六个月以上不满一年按一年计
- 不满六个月按半个月计

**月工资标准**：劳动合同解除前12个月的平均工资

## 四、常见法律依据

### 劳动争议相关

1. **《中华人民共和国劳动争议调解仲裁法》**
   - 第三十一条：仲裁庭的组成
   - 第四十七条：终局裁决

2. **《中华人民共和国劳动合同法》**
   - 第十条：劳动关系的建立
   - 第四十条：医疗期解除限制
   - 第四十七条：经济补偿标准
   - 第八十七条：违法解除赔偿金

3. **《企业职工患病或非因工负伤医疗期规定》**（劳部发〔1994〕479号）
   - 医疗期计算标准

4. **《山东省企业工资支付规定》**
   - 病假工资支付标准

## 五、制作注意事项

### 1. 关于争议事实和理由
一般应写明，但若申请人不愿写明则可以不写，这是与民事判决书的区别。

### 2. 引言部分的依据
必须写明受理案件的依据，即仲裁协议和仲裁申请，否则裁决可能被法院撤销。

### 3. 裁决结果
- 必须针对双方请求事项作出明确、具体的答复（支持或驳回）
- 不能超出申请请求的范围
- 便于执行

### 4. 语言表达
要求清晰明确，行文有条有理，表达准确。

## 六、证据认定规范

### 证据分类
1. 书证（劳动合同、工资条、考勤记录等）
2. 物证
3. 视听资料（录音、录像等）
4. 电子数据（邮件、OA记录、短信等）
5. 证人证言
6. 当事人陈述
7. 鉴定意见
8. 勘验笔录

### 质证要点
1. **真实性**：证据是否真实
2. **关联性**：与争议事实的关联程度
3. **合法性**：取证程序是否合法
4. **证明力**：综合认定证据的证明力
