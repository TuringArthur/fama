# 仲裁裁决书撰写技能

## 使用场景

当用户请求起草劳动争议仲裁裁决书时使用本技能。本技能基于《法律文书写作（第五版）》标准格式，提供完整的裁决书结构和内容指导。

## 仲裁裁决书基本结构

仲裁裁决书由**首部**、**正文**、**裁决**和**尾部**四部分组成。

### 一、首部

```
××仲裁委员会
裁 决 书
(××)×仲裁字第××号
```

当事人信息格式：
- **申请人**：姓名/名称、住所地、联系方式
- **被申请人**：姓名/名称、住所地、法定代表人
- **委托代理人**（如有）：姓名、性别、单位及职务

引言部分需包含：
1. 仲裁机构受理案件的依据（仲裁协议和仲裁申请）
2. 仲裁庭组成情况
3. 审理程序的合法性说明

### 二、正文

#### （一）事实认定

记录仲裁庭查明的事实和认定的证据：
- 劳动关系的建立时间
- 争议发生的时间、经过
- 申请人主张的事实
- 被申请人答辩意见
- 双方提交的证据及质证情况
- 仲裁庭对证据的认定

#### （二）仲裁庭意见（本委认为）

针对双方争议焦点进行分析：
1. 法律关系认定
2. 适用法律依据
3. 对申请人仲裁请求的评析

**写作要点**：
- 法律关系要明晰
- 适用法律要论证充分
- 针对诉讼请求逐一评析

### 三、裁决

根据事实认定和法律分析，作出裁决：

```
根据以上事实及有关法律规定，本委裁决如下：

1. 自本裁决书生效之日起十日内，由被申请人向申请人支付病假工资×××元；
2. 自本裁决书生效之日起十日内，由被申请人向申请人支付违法解除劳动合同赔偿金×××元；
```

**撰写要求**：
- 裁决结果必须明确、具体
- 不得超过申请人仲裁请求的范围
- 便于执行

### 四、尾部

#### （一）生效时间
- 终局裁决：自裁决书作出之日起发生法律效力

#### （二）署名

```
首席仲裁员：×××
仲裁员：×××
仲裁员：×××
××年××月××日
书记员：×××
```

## 裁决金额计算规范

### 病假工资计算

病假工资不得低于当地最低工资标准的80%。

**参考依据**：
- 《山东省企业工资支付规定》：停工医疗累计不超过180天的，发给本人工资70%的病假工资
- 计算时需考虑医疗期长短

### 违法解除劳动合同赔偿金

**计算公式**：
- 经济补偿 = 月平均工资 × 工作年限
- 赔偿金 = 经济补偿 × 2倍

**工作年限认定**：
- 每满一年支付一个月工资
- 六个月以上不满一年按一年计
- 不满六个月按半个月计

**月工资标准**：劳动合同解除前12个月的平均工资

## 证据认定要点

1. **真实性审查**：书证、物证、视听资料的真实性
2. **关联性审查**：证据与争议事实的关联程度
3. **合法性审查**：取证程序是否合法
4. **证明力判断**：综合认定证据的证明力

## 撰写注意事项

1. **事实描述**：一般应写明争议事实和理由，但申请人自愿放弃的可以不写
2. **引言依据**：必须写明受理案件的依据（仲裁协议和仲裁申请）
3. **裁决明确**：必须针对双方请求事项作出明确、具体的答复
4. **语言规范**：清晰明确，行文有条有理，表达准确

## 输出模板

```markdown
# 仲裁裁决书

## 基本信息
- 案号：(年份)×仲裁字第×号
- 申请人：[姓名/名称]
- 被申请人：[姓名/名称]
- 受理日期：[日期]
- 裁决日期：[日期]

## 正文内容

### 一、当事人信息
[详细列出申请人和被申请人信息]

### 二、事实认定
[仲裁庭查明的事实]

### 三、本委认为
[仲裁庭的法律分析和意见]

### 四、裁决
[具体裁决内容]

### 五、尾部
[署名、日期]
```

## 质量检查清单

- [ ] 首部信息完整（机构名称、文号、当事人）
- [ ] 事实认定清楚、证据列举完整
- [ ] 法律分析有据、论证充分
- [ ] 裁决结果明确、具体、可执行
- [ ] 金额计算准确、依据充分
- [ ] 尾部署名规范、日期完整
- [ ] 语言表达准确、无歧义
