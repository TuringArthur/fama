# 民事上诉状撰写技能

## 触发识别

当用户提供以下信息时触发本技能：
- 需要对第一审判决/裁定提起上诉
- 请求帮助撰写上诉状
- 需要准备二审法律文书

## 工作流程

### 步骤1：收集案件信息

收集以下关键信息：

**基础信息**
- 上诉人信息（姓名/名称、身份）
- 被上诉人信息（姓名/名称、身份）
- 原审法院名称
- 原审案号
- 原审判决/裁定日期
- 案件类型/案由

**上诉核心要素**
- 原审裁判结果
- 上诉人的具体诉求（撤销/变更/改判）
- 上诉的主要理由（事实认定/法律适用/程序违法）

### 步骤2：确认上诉资格与期限

在撰写前确认：
1. 上诉人是否为法定上诉权人
2. 是否在法定上诉期限内（判决书送达后15日内）
3. 上诉途径（向原审法院或直接向上级法院提交）

### 步骤3：撰写上诉状正文

按照以下结构撰写：

#### 3.1 首部

```
民事上诉状

上诉人（原审原告/被告/第三人）：[姓名]，[性别]，[出生日期]，[民族]，[职业]，住[住址]。联系方式：[电话]
（法定代理人）：[姓名]，[与上诉人关系]，[职业]，住[住址]
（诉讼代理人）：[姓名]，[律所名称]，[代理权限]

被上诉人（原审原告/被告/第三人）：[同上格式]
```

#### 3.2 上诉事由

```
上诉人因[被上诉人姓名]与[案由]一案，不服[原审法院名称]于[年月日]作出的（[年度]）[民初字第X号]民事判决（裁定），现提出上诉。
```

#### 3.3 上诉请求

分条列项书写，例如：
```
上诉请求：

一、撤销[原审法院名称]（[年度]）[民初字第X号]民事判决第X项；

二、依法改判[具体诉求]；

三、本案诉讼费用由被上诉人承担。
```

#### 3.4 上诉理由

针对原判决错误进行反驳：
1. **事实认定错误**：指出错误事实，提出正确事实和证据
2. **法律适用错误**：指出错误法律条款，正确法律依据
3. **程序违法**：指出违反的程序规定及影响

格式要求：
- 引用原判决具体内容
- 提出相反证据或事实
- 援引正确法律条款
- 分条列项，层次分明

#### 3.5 尾部

```
为此，特向你院上诉，请求撤销原判决（裁定），依法予以改判（重新审理）。

此致
[上诉法院名称]人民法院

附：本上诉状副本 X 份

上诉人（签名或盖章）
[年] 年 [月] 月 [日] 日
```

## 格式要求

### 份数要求
- 上诉状副本份数 = 被上诉人数量 + 1

### 语言要求
- 使用规范的法律用语
- 表述准确、简洁
- 避免口语化表达

### 逻辑要求
- 上诉请求与上诉理由相互呼应
- 证据与事实认定相互印证
- 法律适用与案件性质相匹配

## 质量检查清单

- [ ] 当事人信息完整准确
- [ ] 原审案号和判决日期正确
- [ ] 上诉请求明确具体
- [ ] 上诉理由针对原判决错误
- [ ] 证据材料充分支持上诉理由
- [ ] 法律条款引用准确
- [ ] 格式规范，层次分明
- [ ] 附项完整（副本份数正确）
- [ ] 落款日期在法定期限内

## 常见上诉情形

| 原判决情形 | 处理方式 |
|-----------|---------|
| 事实清楚，适用法律正确 | 驳回上诉，维持原判 |
| 适用法律错误 | 依法改判 |
| 认定事实错误/不清/证据不足 | 撤销原判，发回重审或查清后改判 |
| 违反法定程序 | 撤销原判，发回重审 |
