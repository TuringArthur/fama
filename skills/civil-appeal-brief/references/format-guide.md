# 民事上诉状格式指南（871格式）

## 文书定义

民事上诉状，即民事诉讼当事人不服第一审人民法院作出的判决或裁定，在法定期限内向上一级人民法院提出上诉，请求撤销或变更原裁判而制作的法律文书。

## 法律意义

- 上诉是诉讼当事人享有的一项重要诉讼权利
- 是维护受不公正第一审判决影响的诉讼当事人合法权益的重要手段
- 是法院启动第二审程序、查明案件事实、公正合法处理案件的基础
- 是纠正第一审人民法院错误裁判、确保司法公正的重要法律途径

## 效力说明

> 最高人民法院作出的第一审判决或裁定，各级人民法院作出的第二审判决或裁定，都是立即发生法律效力的判决或裁定，当事人不能再上诉。

## 完整格式结构

### 一、首部

#### （一）标题
在文书顶端居中书写文书名称"民事上诉状"。

#### （二）当事人基本情况

**书写顺序**：按上诉人、被上诉人顺序分别书写。

**标注方式**：用括号在后面分别注明各自在原审中的诉讼地位
- 上诉人（原审原告）
- 上诉人（原审被告）
- 上诉人（原审第三人）
- 被上诉人（原审原告）
- 被上诉人（原审被告）
- 被上诉人（原审第三人）

**共同诉讼特别规定**：
> 共同诉讼的案件，没有提出上诉且他人上诉的内容与其无关时，仍列原审中的诉讼称谓，而不列为被上诉人。

#### 自然人当事人信息
依次写明：
1. 姓名
2. 性别
3. 出生日期
4. 民族
5. 职业
6. 工作单位
7. 住址
8. 联系方式

**特别情形**：上诉人属于未成年人或精神病人的，应在上诉人下面另起一行书写法定代理人的基本情况。

#### 法人或其他组织当事人信息
依次写明：
1. 单位名称
2. 住所地
3. 法定代表人或主要负责人的姓名、职务

#### 诉讼代理人
上诉人或被上诉人及其法定代理人已委托诉讼代理人的，应另起一行书写诉讼代理人的基本情况。

---

### 二、正文

#### （一）上诉事由

**书写位置**：正文开头

**包含要素**：
1. 案由
2. 原审法院名称
3. 案件编号

**表述示例**：

```
上诉人因××××（写明对方当事人姓名/名称和案由）一案，不服××××人民法院××××年××月××日（××××）×民初字第××号民事判决（裁定），现提出上诉。上诉的请求和理由如下：
```

#### （二）上诉请求

**撰写要求**：
- 针对性指明第一审判决或裁定有何错误
- 明确提出要求第二审人民法院撤销或变更原判决的具体请求
- 可明确提出上诉人对该民事争议在实体或程序上的具体诉讼权利主张
- 多项请求应分条列项写明

**表述要求**：
> 明确、具体，不能含糊其词、模棱两可

#### （三）上诉理由

**撰写方法**：

1. **找出错误**：先找出原判决中能够影响原判决结果的错误
2. **提炼论点**：提炼出上诉理由的论点
3. **论证反驳**：针对这些论点，提出有确凿证据证明的事实，结合正确的法律条款进行分析论证，反驳原判决的错误

**论证方法**：主要采取反驳的方法书写

**写作要点**：
- 从具体案情出发
- 紧紧围绕原判决的错误进行有理、有力、有据的驳斥
- 注意一定的条理性
- 做到层次分明、章法有序
- 有针对性地对原判决的事实认定、证据采纳和法律适用情况进行缜密分析

**分条列项要求**：
> 如果原判决有几处错误，则应用分条列项的方法，将驳斥的内容逐一表述出来进行反驳，避免将几个不同性质的问题纠缠在一起进行驳斥，造成毫无头绪、含混不清的现象出现。

**四种情形处理**：

| 情形 | 处理方式 |
|-----|---------|
| 原判决认定事实清楚，适用法律正确 | 判决驳回上诉，维持原判 |
| 原判决适用法律错误 | 依法改判 |
| 原判决认定事实错误，或者原判决认定事实不清，证据不足 | 裁定撤销原判决，发回原审人民法院重审，或者查清事实后改判 |
| 原判决违反法定程序，可能影响案件正确判决 | 裁定撤销原判决，发回原审人民法院重审 |

**正文结尾**：

上诉理由阐述完毕后，接着续写：

```
为此，特向你院上诉，请求撤销原判决（或裁定），依法予以改判（或重新审理）。
```

---

### 三、尾部

#### （一）致送机关

**书写位置**：正文左下角，分两行书写

**书写内容**：
```
此致
××××人民法院
```

#### （二）附项

写明本上诉状副本的份数。

**计算方式**：副本份数 = 被上诉人数量 + 1

#### （三）落款

由上诉人签名或盖章，并注明上诉日期。

## 注意事项汇总

### 基本要求
1. 必须以有法定上诉权的当事人的名义制作
2. 在法定的上诉期限内提出
3. 通过法定的上诉途径向原审人民法院或第二审人民法院提出

### 上诉请求要求
- 应写得明确、具体
- 不能含糊其词、模棱两可

### 上诉理由要求
1. 主要采取反驳的方法书写
2. 从具体案情出发
3. 紧紧围绕原判决的错误进行有理、有力、有据的驳斥
4. 注意一定的条理性
5. 做到层次分明、章法有序
6. 有针对性地对原判决的事实认定、证据采纳和法律适用情况进行缜密分析
