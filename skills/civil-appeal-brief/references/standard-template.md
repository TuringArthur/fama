# 民事上诉状标准模板

## 完整模板

---

**民事上诉状**

上诉人（原审[原告/被告/第三人]）：[姓名]，[性别]，[×年×月×日出生]，[民族]族，[职业]，住[详细住址]。联系方式：[电话号码]。

[法定代理人（如有）]：[姓名]，[与上诉人关系]，[职业]，住[住址]。

[诉讼代理人（如有）]：[姓名]，[所属律所或单位]，[代理权限]。

被上诉人（原审[原告/被告/第三人]）：[姓名]，[性别]，[×年×月×日出生]，[民族]族，[职业]，住[详细住址]。联系方式：[电话号码]。

上诉人因[被上诉人姓名]与[案由]一案，不服[原审法院全称]于[年]年[月]月[日]日作出的（[年度]）[民初字第×号]民事判决（裁定），现提出上诉。上诉的请求和理由如下：

**上诉请求**：

一、撤销[原审法院名称]（[年度]）[民初字第×号]民事判决第[×]项；

二、依法改判[具体诉讼请求，如：支持上诉人全部诉讼请求/被上诉人支付上诉人款项×××元/变更子女抚养权归上诉人等]；

三、本案一审、二审诉讼费用由被上诉人承担。

**上诉理由**：

一、关于原判决认定事实错误

[原审认定]：原审法院认定"[引述原审关于事实认定的具体内容]"。

[上诉人主张]：该认定与事实不符。实际情况是[详细陈述正确事实]。

[证据支持]：[列出支持上诉主张的证据，如：证据一：......；证据二：......；证人[姓名]可证明......]。

[正确认定]：根据上述证据，应当认定[正确的事实认定]。

二、关于原判决适用法律错误

[原审适用]：原审法院适用[法律名称]第×条第×款认定[相关内容]。

[错误分析]：该法律适用存在错误。[详细分析为何适用错误]。

[正确适用]：根据[正确法律名称]第×条第×款之规定[正确法律内容]。

三、[如有]关于原判决违反法定程序

[程序违法]：原审法院在审理过程中存在[具体程序违法情形，如：未依法送达/未保障当事人质证权利/遗漏必须共同诉讼当事人等]。

[法律规定]：[引用程序法的相关规定]

[影响分析]：该程序违法导致[具体影响，如：剥夺了上诉人的辩论权利/影响了案件的公正判决]。

综上所述，原审判决存在事实认定错误（或：适用法律错误，或：违反法定程序），依法应予撤销（或改判）。为此，特向你院上诉，请求撤销原判决（裁定），依法予以改判（重新审理）。

此致

## 模板使用说明

### 空格填写说明

| 空格 | 填写内容 | 示例 |
|-----|---------|-----|
| [姓名] | 当事人姓名 | 张三 |
| [性别] | 男/女 | 男 |
| [×年×月×日出生] | 出生日期 | 1980年5月15日出生 |
| [民族] | 民族 | 汉 |
| [职业] | 职业 | 农民/教师/个体工商户 |
| [详细住址] | 实际居住地址 | 北京市朝阳区建国路1号 |
| [电话号码] | 联系电话 | 13800138000 |
| [与上诉人关系] | 与上诉人关系 | 父子/母子/配偶 |
| [所属律所或单位] | 律所或单位名称 | 某某律师事务所 |
| [代理权限] | 一般代理/特别授权 | 一般代理 |
| [被上诉人姓名] | 对方当事人姓名 | 李四 |
| [案由] | 案件类型 | 民间借贷纠纷/离婚纠纷/合同纠纷 |
| [原审法院全称] | 原审法院完整名称 | 北京市朝阳区人民法院 |
| [年][月][日] | 具体日期 | 2024年1月15日 |
| [年度] | 年份 | 2024 |
| [民初字第×号] | 案号 | 民初字第1234号 |
| [上诉法院名称] | 上级法院名称 | 北京市第三中级人民法院 |

### 上诉请求填写要点

1. **撤销项**：写明需要撤销的原判决第几项
2. **改判项**：明确具体的改判请求（支付金额/变更内容/承担义务等）
3. **诉讼费**：一般要求对方承担

### 上诉理由填写要点

1. **事实认定错误**
   - 引用原审认定的具体内容
   - 陈述正确事实
   - 列出证据支持

2. **法律适用错误**
   - 指出原审适用的法律
   - 分析错误原因
   - 提出正确法律依据

3. **程序违法**
   - 指明具体程序问题
   - 引用程序法规定
   - 说明对判决的影响

### 份数计算

- 副本份数 = 被上诉人数量 + 1份（上诉人留存）
- 证据材料按实际数量准备

### 日期要求

- 上诉日期必须在收到判决书之日起15日内
- 计算方式：从收到判决书次日起算，第15日为最后期限
