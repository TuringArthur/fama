# 通知立案书生成器

## 概述

通知立案书是人民检察院在审查公安机关的不立案理由后，认为理由不能成立或公安机关未说明不立案理由但符合立案条件时，依法制作的法律文书。根据《中华人民共和国刑事诉讼法》第113条及相关司法解释制作。

## 法律依据

- 《中华人民共和国刑事诉讼法》第113条
- 《人民检察院刑事诉讼规则》第559条、第561条、第563条

## 文书份数

该文书一式三份：
1. 一份存档
2. 一份送达公安机关
3. 一份报送上一级人民检察院备案

## 文书结构

通知立案书包含四个部分：
1. 文头（院名、文种、案号）
2. 正文（受文机关、审查意见、理由、法律指令）
3. 落款（日期、印章）
4. 附注

### 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| 检察院名称 | 发文检察院的全称 | 北京市人民检察院 |
| 案号 | 检察院案件编号格式 | ×检刑申通立〔2024〕15号 |
| 侦查机关名称 | 接收文书的公安机关 | 北京市公安局 |
| 原案当事人姓名 | 涉嫌犯罪的人员 | 张三 |
| 罪名 | 涉嫌的罪名 | 诈骗罪 |
| 收到不立案理由日期 | 收到不立案理由说明书的日期 | 2024年3月15日 |
| 审查意见 | 说明不立案理由不能成立的原因 | 经审查，你局认定张三不构成诈骗罪的理由不能成立 |
| 事实与法律依据 | 立案的事实根据和法律依据 | 根据现有证据，张三的行为符合诈骗罪构成要件 |
| 立案期限 | 法律规定的期限 | 十五日 |

## 生成逻辑

### 输入信息

1. **基本信息**（必填）
   - 检察院名称
   - 案号
   - 侦查机关名称

2. **案件信息**（必填）
   - 原案当事人姓名
   - 涉嫌罪名
   - 收到不立案理由说明书日期

3. **审查意见**（必填）
   - 不立案理由不能成立的原因
   - 应当立案的事实根据
   - 应当立案的法律依据

### 处理流程

1. 验证必填参数是否完整
2. 按标准格式组装文书内容
3. 计算立案期限（收到通知后15日）
4. 生成完整文书

### 输出结果

生成完整的通知立案书文本，包含：
- 规范的中文法律文书格式
- 完整的法律依据引用
- 明确的立案要求和期限
- 标准的落款格式

## 使用场景

- 检察院审查公安不立案决定
- 检察院认为公安不立案理由不成立
- 检察院认为符合立案条件需要通知立案
- 制作立案监督法律文书

## 注意事项

1. 通知书须在法定期限内制作送达
2. 理由陈述须具体充分
3. 立案期限从收到通知书之日起算
4. 须加盖检察院印章生效
5. 需报送上一级检察院备案
