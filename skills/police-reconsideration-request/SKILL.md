# 公安机关提请复核意见书

本技能帮助您撰写规范的提请复核意见书。提请复核意见书是公安机关认为同级人民检察院复议决定存在错误时，向上一级人民检察院请求复核的法律文书。

## 工作流程

### 第一步：收集基本信息

请提供以下信息：

1. **案件基本信息**
   - 案由/案件名称
   - 案件编号
   - 当事人信息

2. **复议历程**
   - 原不批准逮捕/不起诉决定的日期和文号
   - 公安机关申请复议的日期和文号
   - 同级人民检察院复议决定的日期和文号
   - 复议决定的具体内容

3. **复核理由**
   - 认为复议决定错误的具体理由
   - 支持己方观点的事实依据
   - 支持己方观点的法律依据
   - 公安机关的处理意见

4. **法律依据**
   - 引用《刑事诉讼法》第70条（逮捕条件）
   - 引用《刑事诉讼法》第144条（复核程序）

### 第二步：生成文书结构

提请复核意见书由以下三部分组成：

#### 一、首部

```
[标题] 提请复核意见书
[编号] [年份]×字第×号
[主送机关] ××省/市人民检察院
```

#### 二、正文

**（一）提请复核的原因**

1. 原不批准逮捕/不起诉决定的情况
2. 公安机关申请复议的情况
3. 同级人民检察院复议决定的情况
4. 公安机关认为复议决定错误之处

**（二）提请复核的理由和意见**

1. 针对复议决定中的具体事项逐条反驳
2. 引用证据说明事实认定问题
3. 引用法律条文说明法律适用问题
4. 明确提出公安机关的处理意见

**（三）提请复核的法律根据和要求**

1. 引用法律依据
2. 提出复核请求
3. 说明时间、内容和未被采纳情况

#### 三、尾部

**（一）附项**
- 案卷册数及页数

**（二）落款**
- 发文机关：××公安局
- 日期：××××年××月××日
- 加盖公章

### 第三步：写作注意事项

1. **目的明确**：复核原因必须写清楚
2. **论据充分**：针对复议决定提出具体论点和论据
3. **事实具体**：扼要写明犯罪事实，证据确凿
4. **引用准确**：法律条文引用必须准确
5. **语气平和**：语言风格平和、委婉、据理力争
6. **理由充分**：指出错误时理由充分，不含糊

### 第四步：输出模板

```
                        ××公安局
                    提请复核意见书

[年份]×公复核字第×号

××省/市人民检察院：

一、提请复核的原因
（详述复议历程和认为复议决定错误之处）

二、提请复核的理由和意见
（逐条反驳复议决定，提出处理意见）

三、提请复核的法律根据和要求
（引用法律依据，提出复核请求）

附：案卷×册，共×页

                    ××公安局
                ××××年××月××日
               （加盖公章）
```

## 常见问题

**问：什么情况下可以提请复核？**
答：公安机关对同级人民检察院不批准逮捕或不起诉的复议决定不服时，可以向上一级人民检察院提请复核。

**问：复核与复议有何区别？**
答：复议是向原作出决定的检察院申请复核；复核是向上一级检察院请求审查复议决定。

**问：复核意见书的时效要求是什么？**
答：应在收到复议决定书后法定期限内提出，具体时限参照相关法律规定。
