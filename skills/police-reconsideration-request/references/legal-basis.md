# 提请复核意见书法律依据与格式规范

## 一、法律依据

### 《中华人民共和国刑事诉讼法》相关条款

#### 第七十条（逮捕条件）
公安机关要求逮捕犯罪嫌疑人的时候，应当写出提请批准逮捕书，连同案卷材料、证据，一并移送同级人民检察院审查批准。逮捕的条件及程序。

#### 第一百四十四条（复核程序）
对人民检察院不批准逮捕、不起诉的决定，公安机关认为错误的时候，可以要求复议，但是必须将被拘留的人立即释放。如果意见不被接受，可以向上一级人民检察院提请复核。

## 二、提请复核意见书的结构

### 首部
1. **标题**：「提请复核意见书」
2. **编号**：「[年份]×公复核字第×号」
3. **主送机关**：「××省/市人民检察院」

### 正文（核心部分）

#### （一）提请复核的原因

**必须包含以下内容：**

1. **原决定情况**
   - 说明同级人民检察院不批准逮捕/不起诉决定的日期
   - 文号
   - 决定的具体内容

2. **复议申请情况**
   - 公安机关申请复议的日期
   - 复议申请的具体内容
   - 复议申请书的编号

3. **复议决定情况**
   - 同级人民检察院复议决定的日期
   - 发文机关
   - 复议决定的具体内容

4. **提请复核的原因**
   - 公安机关认为复议决定错误的具体之处
   - 指明分歧焦点

#### （二）提请复核的理由和意见

**写作要求：**

1. **逐条反驳**：针对复议决定书中的决定事项和理由，逐条予以反驳

2. **事实论证**：
   - 引用具体证据
   - 重申原请求复议意见书中所列举的事实
   - 指出复议决定对事实认定的错误

3. **法律论证**：
   - 指出复议决定在法律适用上的错误
   - 明确反驳人民检察院不批准逮捕/不起诉的理由
   - 运用驳论手法表明不同意见

4. **提出处理意见**：
   - 明确提出公安机关对案件的处理意见
   - 说明应当如何处理

#### （三）提请复核的法律根据和要求

**必须包含：**

1. **法律依据引用**
   - 《刑事诉讼法》第七十条
   - 《刑事诉讼法》第一百四十四条

2. **复核请求**
   - 请求上一级人民检察院对案件进行复核
   - 提出复核的具体要求

3. **说明事项**
   - 提请复议的时间
   - 复议的内容
   - 未被采纳的情况
   - 公安机关认定的结论

### 尾部

#### （一）附项
- 案卷册数
- 案卷页数

#### （二）落款
- 发文机关名称：「××公安局」
- 发文日期：「××××年××月××日」
- 加盖公章

## 三、写作注意事项

### 内容要求

1. **复核原因必须明确**
   - 清楚写明为什么提请复核
   - 目的要明确
   - 要有针对性

2. **正文部分**
   - 针对复议决定书提出具体论点和论据
   - 具体、扼要地写明犯罪事实
   - 证据必须确凿
   - 引用法律条文必须准确

3. **指出错误时**
   - 理由必须充分
   - 不能含糊其辞
   - 要有理有据

4. **语言风格**
   - 平和
   - 委婉
   - 据理力争
   - 与复议申请书的语言风格保持一致

### 格式要求

1. 结构完整，层次分明
2. 编号规范，引用准确
3. 用词严谨，表达准确
4. 格式统一，排版规范

## 五、术语解释

| 术语 | 含义 |
|------|------|
| 不批准逮捕 | 检察院认为不符合逮捕条件，不批准公安机关的逮捕请求 |
| 复议 | 公安机关向原作出决定的检察院申请重新审查 |
| 复核 | 公安机关向上级检察院申请审查复议决定 |
| 案由 | 案件的法律性质和类型 |
