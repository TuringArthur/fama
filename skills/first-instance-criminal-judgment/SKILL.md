# 第一审刑事判决书撰写技能

## 触发条件

当用户请求以下操作时激活本技能：
- 撰写第一审刑事判决书
- 生成公诉案件判决书
- 创建一审刑事判决文书
- 制作刑事判决书模板
- 撰写法院判决书

## 工作流程

### 1. 收集案件基本信息

在开始撰写前，必须收集以下信息：

**案件基本信息：**
- 法院名称
- 案号（年份+刑初字+编号）
- 公诉机关名称
- 被告人基本信息：
  - 姓名
  - 性别
  - 出生年月日
  - 民族
  - 出生地
  - 文化程度
  - 职业或工作单位和职务
  - 住址
  - 强制措施情况及现羁押场所
- 辩护人信息（姓名、工作单位和职务）

### 2. 撰写审理经过部分

描述以下内容：
- 检察院提起公诉的情况
- 法院受理情况
- 合议庭组成
- 开庭方式（公开或不公开）
- 参加庭审人员：公诉人、被告人及其法定代理人、辩护人、证人、鉴定人、翻译人员等

### 3. 撰写控辩双方意见

**公诉机关指控：**
- 简述指控的犯罪事实
- 列举主要证据
- 阐述法律意见

**被告人及辩护人辩护意见：**
- 简述被告人的辩解
- 辩护人的辩护意见
- 提交的证据

### 4. 撰写事实认定（经审理查明）

详细写明：
- 法庭认定的事实
- 证据及其来源
- 对控辩双方有异议的事实和证据进行分析认证

### 5. 撰写判决理由（本院认为）

包含以下内容：
- 根据认定的事实和法律规定
- 论证指控的犯罪是否成立
- 确定罪名
- 确定量刑情节（从轻、减轻、免除或从重处罚）
- 对控辩双方关于法律适用的意见进行分析
- 阐明判决理由
- 写明判决所依据的具体法律条款

### 6. 撰写判决结果

根据案件情况选择以下格式：

**定罪判刑格式：**
```
被告人XXX犯XX罪，判处......（写明主刑、附加刑），刑期......（写明刑期起止日期及折抵说明）。
```

**定罪免刑格式：**
```
被告人XXX犯XX罪，免予刑事处分。
```

**宣告无罪格式：**
```
依照《中华人民共和国刑事诉讼法》第一百六十二条第（二）项的规定，宣告被告人XXX无罪。
```

### 7. 撰写尾部内容

包含以下要素：
- 上诉权利告知（十日内向上一级人民法院提出上诉）
- 审判人员署名（审判长、审判员）
- 判决日期（年 月 日）
- 院印
- 书记员署名
- 核对章标注（本件与原本核对无异）

## 格式规范

### 文书结构
```
××××人民法院
刑事判决书
(××××)×刑初字第××号

公诉机关：×××人民检察院

被告人：[基本信息]

辩护人：[信息]

审理经过
...

经审理查明
...

本院认为
...

判决结果
...

审判长 ×××
审判员 ×××
审判员 ×××
××××年××月××日
（院印）

书记员 ×××
本件与原本核对无异
```

### 排版要求
- 标题居中加粗
- 各部分标题使用加粗标记
- 正文采用标准公文格式
- 页面标注页码

## 法律依据参考

撰写时需引用以下法律依据：
- 《中华人民共和国刑法》相关条款
- 《中华人民共和国刑事诉讼法》第一百六十二条
- 最高人民法院司法解释

## 注意事项

1. **事实清楚**：判决书认定的事实必须有充分证据支持
2. **证据确实**：引用的证据必须经过法庭质证
3. **法律准确**：引用的法律条款必须准确无误
4. **说理充分**：判决理由必须充分阐述法律适用依据
5. **格式规范**：严格遵循判决书格式规范
6. **用语严谨**：使用法律专业术语，表达准确严谨

## 输出格式

生成的判决书应包含完整的所有要素，格式规范，内容完整，可直接用于司法实践。
