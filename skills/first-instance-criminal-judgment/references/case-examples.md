# 第一审刑事判决书实例分析

## 实例一：盗窃罪案件

---

```
                              北京市朝阳区人民法院
                                刑事判决书

                            (2024)朝刑初字第1234号

公诉机关：北京市朝阳区人民检察院

被告人：王某某，男，1990年5月15日出生，汉族，出生地河南省郑州市，文化程度初中，无职业，住河南省郑州市某某区某某路某某号。因涉嫌犯盗窃罪，于2024年2月10日被北京市公安局朝阳分局刑事拘留，同年2月17日经朝阳区人民检察院批准逮捕，现羁押于朝阳区看守所。

辩护人：赵某某，北京市某某律师事务所律师。

北京市朝阳区人民法院于2024年3月5日接到北京市朝阳区人民检察院京朝检刑诉〔2024〕123号起诉书后，以(2024)朝刑初字第1234号立案，依法组成合议庭，于2024年4月12日公开开庭审理了本案。北京市朝阳区人民检察院指派检察员张某某出庭支持公诉，被告人王某某及其辩护人赵某某到庭参加诉讼。本案现已审理终结。

                                经审理查明

2024年2月8日14时许，被告人王某某在本市朝阳区某商场一层服装专柜处，趁营业员李某某不备，将其放在柜台上的一部苹果手机（型号：iPhone 15 Pro，内存256G）盗走。后被告人王某某在逃离商场时被商场保安当场抓获。

经北京市朝阳区价格认证中心鉴定，被盗手机价值人民币8999元。被盗手机已发还被害人。

上述事实，有下列证据予以证实：
1. 被害人李某某的陈述，证实其手机被盗的时间、地点及手机特征；
2. 证人孙某某（商场保安）的证言，证实目击被告人盗窃并将其抓获的经过；
3. 证人周某某（商场营业员）的证言，证实被害人手机被盗的情况；
4. 被告人王某某的供述和辩解，供认了上述盗窃事实；
5. 被盗手机照片及发票，证实被盗手机的品牌、型号和价值；
6. 价格鉴定意见，证实被盗手机价值人民币8999元；
7. 现场监控视频，证实被告人实施盗窃的经过；
8. 发还清单，证实被盗手机已发还被害人。

辩护人对上述事实和证据不持异议。

                                本院认为

本院认为，被告人王某某以非法占有为目的，采用秘密窃取的手段盗窃他人财物，数额较大，其行为已构成盗窃罪，依法应予惩处。公诉机关指控被告人王某某犯盗窃罪的事实清楚，证据确实充分，指控的罪名成立，本院予以确认。

关于量刑情节：被告人王某某系初犯，归案后能够如实供述自己的罪行，认罪态度较好，有悔罪表现；被盗手机已发还被害人，未造成实际损失。综合上述情节，依法可对其从轻处罚。辩护人所提相关辩护意见成立，本院予以采纳。

综上所述，为惩罚犯罪，保护公民的财产权利，维护社会秩序，依照《中华人民共和国刑法》第二百六十四条、第五十二条、第五十三条之规定，判决如下：

                                判决结果

被告人王某某犯盗窃罪，判处有期徒刑七个月，并处罚金人民币一万元。

刑期从判决执行之日起计算，判决执行以前先行羁押的，羁押一日折抵刑期一日，即自2024年2月10日起至2024年9月9日止。

如不服本判决，可在接到判决书的第二日起十日内，通过本院或者直接向北京市第三中级人民法院提出上诉。书面上诉的，应当提交上诉状正本一份，副本二份。

                                 审判长  陈某某
                                 审判员  刘某某
                                 审判员  马某某

                               2024年5月20日

                                 （院印）

                                 书记员  黄某某

                               本件与原本核对无异
```

## 实例分析

### 1. 结构完整性
本判决书包含判决书的所有基本要素：眉首、公诉机关、被告人基本情况、辩护人信息、审理经过、事实认定、判决理由、判决结果、尾部。

### 2. 事实认定规范性
- 时间、地点、人物、行为、结果等要素齐全
- 证据分项列举，类型完整
- 证据之间形成完整的证据链

### 3. 说理充分性
- 对犯罪构成进行了分析
- 对量刑情节进行了评价
- 对辩护意见进行了回应
- 引用了具体的法律条款

### 4. 判决结果明确性
- 罪名准确
- 刑期明确
- 罚金数额具体
- 刑期起止日期清楚

## 实例二：故意伤害罪案件（从轻处罚）

---

```
                              上海市浦东新区人民法院
                                刑事判决书

                            (2024)浦刑初字第567号

公诉机关：上海市浦东新区人民检察院

被告人：李某某，男，1985年8月20日出生，汉族，出生地江苏省苏州市，文化程度高中，个体工商户，住上海市浦东新区某某镇某某路某某号。因涉嫌犯故意伤害罪，于2024年1月5日被上海市公安局浦东分局刑事拘留，同年1月12日经浦东新区人民检察院批准逮捕，现羁押于上海市浦东新区看守所。

辩护人：钱某某，上海市某某律师事务所律师。

上海市浦东新区人民法院于2024年2月15日接到上海市浦东新区人民检察院沪浦检刑诉〔2024〕89号起诉书后，以(2024)浦刑初字第567号立案，依法组成合议庭，于2024年3月20日公开开庭审理了本案。上海市浦东新区人民检察院指派检察员王某出庭支持公诉，被告人李某某及其辩护人钱某某到庭参加诉讼。本案现已审理终结。

                                经审理查明

2024年1月3日20时许，被告人李某某在本市浦东新区某酒吧内，因琐事与被害人张某某发生口角。后双方发生肢体冲突，被告人李某某用拳头击打被害人张某某面部，致其鼻骨骨折。经鉴定，被害人张某某的伤势构成轻伤二级。

案发后，被告人李某某明知他人报警而在现场等待，抓捕时无抗拒行为，到案后如实供述了上述犯罪事实。

案发后，被告人李某某与被害人张某某达成和解协议，赔偿被害人经济损失人民币15万元，被害人对被告人的行为表示谅解。

上述事实，有下列证据予以证实：
1. 被害人张某某的陈述，证实其被殴打的时间、地点、经过及受伤情况；
2. 证人陈某某（酒吧服务员）的证言，证实目击双方发生冲突的经过；
3. 证人周某某（酒吧保安）的证言，证实被告人打伤被害人的经过；
4. 被告人李某某的供述和辩解，供认了上述犯罪事实；
5. 法医鉴定意见，证实被害人张某某的伤势构成轻伤二级；
6. 和解协议及收条，证实被告人已赔偿被害人并取得谅解。

辩护人提出以下辩护意见：
1. 被告人明知他人报警而在现场等待，到案后如实供述，构成自首；
2. 被告人认罪态度好，有悔罪表现；
3. 被告人已赔偿被害人全部经济损失并取得谅解；
4. 本案系民间纠纷引发，被害人存在一定过错。

                                本院认为

本院认为，被告人李某某故意伤害他人身体，致人轻伤二级，其行为已构成故意伤害罪，依法应予惩处。公诉机关指控被告人李某某犯故意伤害罪的事实清楚，证据确实充分，指控的罪名成立，本院予以确认。

关于量刑情节：第一，被告人明知他人报警而在现场等待，抓捕时无抗拒行为，到案后如实供述犯罪事实，依法构成自首，可以从轻或减轻处罚。第二，被告人认罪态度较好，有悔罪表现。第三，被告人已赔偿被害人全部经济损失并取得谅解，可酌情从轻处罚。第四，本案系民间纠纷引发，被害人对矛盾的激化负有一定责任，可对被告人酌情从轻处罚。

综上所述，为惩罚犯罪，保护公民的人身权利，维护社会秩序，同时考虑被告人具有自首情节，认罪态度较好，已赔偿被害人损失并取得谅解，有悔罪表现，没有再犯罪的危险，宣告缓刑对所居住社区没有重大不良影响，依法可以对其宣告缓刑。依照《中华人民共和国刑法》第二百三十四条第一款、第六十七条第一款、第七十二条第一款、第七十三条第二款、第三款之规定，判决如下：

                                判决结果

被告人李某某犯故意伤害罪，判处有期徒刑一年，缓刑一年六个月。

缓刑考验期限，从判决确定之日起计算，即自2024年4月15日起至2025年10月14日止。

如不服本判决，可在接到判决书的第二日起十日内，通过本院或者直接向上海市第一中级人民法院提出上诉。书面上诉的，应当提交上诉状正本一份，副本二份。

                                 审判长  吴某某
                                 审判员  郑某某
                                 审判员  孙某某

                               2024年4月15日

                                 （院印）

                                 书记员  冯某某

                               本件与原本核对无异
```

## 实例分析

### 1. 自首情节的认定
- 被告人明知他人报警
- 在现场等待
- 抓捕时无抗拒行为
- 到案后如实供述
- 符合自首的构成要件

### 2. 缓刑的适用
- 具有自首情节
- 认罪态度好
- 已赔偿并取得谅解
- 有悔罪表现
- 没有再犯罪危险
- 对社区无不良影响

### 3. 多情节的综合考量
- 自首（法定从轻或减轻）
- 赔偿损失（酌定从轻）
- 取得谅解（酌定从轻）
- 被害人有过错（酌定从轻）

## 实例三：受贿罪案件（定罪免刑）

---

```
                              广州市中级人民法院
                                刑事判决书

                            (2024)穗刑初字第789号

公诉机关：广东省广州市人民检察院

被告人：陈某某，男，1965年3月10日出生，汉族，出生地广东省广州市，文化程度大学本科，原系广州市某某局副局长（正处级），住广州市天河区某某路某某号。因涉嫌犯受贿罪，于2023年11月20日被广东省监察委员会留置，2024年2月1日被逮捕，现羁押于广州市第一看守所。

辩护人：林某某，广东省某某律师事务所律师。

广东省广州市中级人民法院于2024年2月20日接到广东省广州市人民检察院穗检刑诉〔2024〕56号起诉书后，以(2024)穗刑初字第789号立案，依法组成合议庭，于2024年3月25日公开开庭审理了本案。广东省广州市人民检察院指派检察员徐某某出庭支持公诉，被告人陈某某及其辩护人林某某到庭参加诉讼。本案现已审理终结。

                                经审理查明

一、2018年5月，被告人陈某某在担任广州市某某局副局长期间，利用其负责工程招标的职务便利，为请托人王某某在承接该局某工程项目中提供帮助。2018年8月，被告人陈某某收受王某某给予的现金人民币10万元。

二、2019年10月，被告人陈某某利用上述职务便利，为请托人王某某在工程款拨付方面提供帮助。2019年12月，被告人陈某某收受王某某给予的现金人民币5万元。

综上，被告人陈某某利用职务上的便利，为他人谋取利益，先后两次收受他人财物共计人民币15万元。

案发后，被告人陈某某如实供述了上述犯罪事实，并主动退缴了全部赃款人民币15万元。

上述事实，有下列证据予以证实：
1. 被告人陈某某的供述和辩解，供认了上述受贿事实；
2. 证人王某某（请托人）的证言，证实请托事项和送钱经过；
3. 证人李某（工程公司负责人）的证言，证实工程承接和款拨付的情况；
4. 书证，包括工程招标文件、合同、拨款凭证等，证实请托事项的实现；
5. 银行转账记录，证实受贿款项的来源和去向；
6. 退赃凭证，证实被告人已退缴全部赃款。

辩护人提出以下辩护意见：
1. 被告人认罪态度好，如实供述犯罪事实；
2. 被告人已退缴全部赃款；
3. 被告人受贿数额刚过追诉标准，犯罪情节较轻；
4. 被告人在本案提起公诉前退赃，有悔罪表现；
5. 建议对被告人免予刑事处分。

                                本院认为

本院认为，被告人陈某某身为国家工作人员，利用职务上的便利，非法收受他人财物，为他人谋取利益，其行为已构成受贿罪，依法应予惩处。公诉机关指控被告人陈某某犯受贿罪的事实清楚，证据确实充分，指控的罪名成立，本院予以确认。

关于量刑情节：第一，被告人陈某某归案后能够如实供述自己的罪行，认罪态度较好，有悔罪表现，依法可以从轻处罚。第二，被告人已退缴全部赃款，挽回了国家损失，可以酌情从轻处罚。第三，被告人受贿数额为人民币15万元，刚过受贿罪数额较大的追诉标准（3万元），犯罪数额相对较小。综合上述情节，被告人犯罪情节较轻，有悔罪表现，不需要判处刑罚。

综上所述，为惩罚犯罪，维护国家工作人员职务行为的廉洁性，同时考虑被告人犯罪情节较轻，有悔罪表现，不需要判处刑罚。依照《中华人民共和国刑法》第三百八十五条第一款、第三百八十六条、第三百八十三条第一款第（一）项、第三十七条之规定，判决如下：

                                判决结果

被告人陈某某犯受贿罪，免予刑事处分。

如不服本判决，可在接到判决书的第二日起十日内，通过本院或者直接向广东省高级人民法院提出上诉。书面上诉的，应当提交上诉状正本一份，副本二份。

                                 审判长  何某某
                                 审判员  蒋某某
                                 审判员  沈某某

                               2024年5月10日

                                 （院印）

                                 书记员  朱某某

                               本件与原本核对无异
```

## 实例分析

### 1. 定罪免刑的适用条件
- 犯罪情节较轻
- 有悔罪表现
- 不需要判处刑罚
- 综合全案情况考量

### 2. 量刑情节的认定
- 如实供述（坦白）
- 退缴全部赃款
- 犯罪数额刚过追诉标准

### 3. 法律条款的综合适用
- 受贿罪的构成条款
- 受贿罪的处罚条款
- 定罪免刑的条款

## 三个实例的比较分析

### 1. 判决结果的多样性

| 案件类型 | 判决结果 | 特点 |
|---------|---------|------|
| 盗窃罪 | 有期徒刑+罚金 | 常规刑罚 |
| 故意伤害罪 | 有期徒刑+缓刑 | 适用缓刑 |
| 受贿罪 | 免予刑事处分 | 定罪免刑 |

### 2. 量刑情节的运用

| 案件 | 从轻情节 | 法律依据 |
|-----|---------|---------|
| 盗窃罪 | 初犯、如实供述、发还赃物 | 第264条等 |
| 故意伤害罪 | 自首、赔偿、谅解、被害人过错 | 第234条等 |
| 受贿罪 | 如实供述、退赃、情节较轻 | 第385条等 |

### 3. 判决书的撰写特点

1. **盗窃罪案**：事实清楚，证据充分，说理简洁
2. **故意伤害案**：详细论述了缓刑适用的理由
3. **受贿案**：论证了定罪免刑的正当性
