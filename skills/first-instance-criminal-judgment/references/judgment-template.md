## 不同判决类型示例

### 示例一：盗窃罪判决书（定罪判刑）

```
被告人张某犯盗窃罪，判处有期徒刑六个月，并处罚金人民币三千元。

刑期从判决执行之日起计算，判决执行以前先行羁押的，羁押一日折抵刑期一日，即自2024年3月15日起至2024年9月14日止。
```

### 示例二：故意伤害罪判决书（从轻处罚）

```
被告人李某犯故意伤害罪，判处有期徒刑三年，缓刑四年。

缓刑考验期限，从判决确定之日起计算，即自2024年4月20日起至2028年4月19日止。
```

### 示例三：受贿罪判决书（定罪免刑）

```
被告人王某犯受贿罪，数额较大，鉴于其在案发后主动退缴全部赃款，有悔罪表现，犯罪情节较轻，免予刑事处分。
```

### 示例四：宣告无罪

```
依照《中华人民共和国刑事诉讼法》第一百六十二条第（二）项的规定，宣告被告人赵某无罪。
```
