# 第一审刑事判决书撰写规范指南

## 一、判决书的法律性质

刑事判决书是人民法院依照刑事诉讼法规定的程序，对刑事案件审理后，就案件的实体问题和程序问题作出的书面决定。它是国家的法律文书，具有以下特性：

1. **强制性**：判决书一经生效，即具有法律强制力
2. **稳定性**：非经法定程序不得随意更改
3. **公开性**：除法律规定不公开审理的案件外，一律公开
4. **规范性**：格式和内容必须符合法律规定的样式和要求

---

## 二、判决书的基本要求

### 1. 事实清楚

事实是判决的基础，判决书认定的事实必须达到以下标准：
- 犯罪的时间、地点、动机、目的、手段、情节、后果等要素齐全
- 事实的描述准确、具体，避免模糊笼统
- 事实与证据相互印证，形成完整的证据链

### 2. 证据确凿

证据是认定事实的依据，判决书必须做到：
- 列举的证据经过法庭质证
- 证据与待证事实之间具有关联性
- 证据之间能够相互印证，排除合理怀疑
- 对控辩双方有争议的证据必须进行分析认证

### 3. 说理充分

说理是判决书的灵魂，必须做到：
- 对控辩双方的意见逐一回应
- 对争议的证据逐项分析认证
- 对适用的法律条款逐一说明
- 对量刑的理由充分阐述

### 4. 法律准确

法律适用必须准确：
- 引用的法律条款必须现行有效
- 条款的编号、款项必须准确无误
- 避免引用已废止或修改的条款
- 注意法律竞合时的条款选择

---

## 三、各部分撰写要点

### （一）眉首部分

**法院名称**
- 使用规范的法院全称
- 居中排列，字号适当大于正文

**案号**
- 格式：（年份）+刑初字+编号+号
- 案号应与立案时的案号一致
- 不得随意变更或省略

### （二）公诉机关

- 写明提起公诉的人民检察院全称
- 写明起诉书的编号和日期

### （三）被告人基本情况

必须写明以下信息：
```
姓名        - 与身份证件一致
性别        - 男/女
出生年月日  - 以身份证为准
民族        - 以身份证为准
出生地      - 详细到市/县
文化程度    - 以实际学历为准
职业        - 以实际职业为准
工作单位和职务 - 具体填写
住址        - 以户籍所在地为准
强制措施情况 - 刑事拘留、逮捕、取保候审等
羁押场所    - 看守所、监狱等
```

**注意事项**：
- 被告人曾用名可在括号中注明
- 多名被告人应分别列明
- 被告人的自然情况应当与其在诉讼中的身份相符

### （四）辩护人信息

辩护人信息包括：
- 律师：律所名称 + 律师姓名
- 非律师辩护人：工作单位和职务

### （五）审理经过

这部分应当写明：
1. 起诉机关和起诉日期
2. 法院立案日期和案号
3. 合议庭组成人员
4. 公开或不公开审理
5. 庭审人员到庭情况
6. 审理是否因法定事由中止

**表述示例**：
```
本院于2024年3月15日接到某某人民检察院某某号起诉书后，以（2024）某刑初字第12号立案，依法组成合议庭，于2024年4月8日公开开庭审理了本案。某某人民检察院指派检察员张三出庭支持公诉，被告人李四及其辩护人王五到庭参加诉讼。本案现已审理终结。
```

### （六）控辩双方意见

**公诉机关指控**
- 简述指控的犯罪事实
- 列举主要证据
- 提出明确的指控意见和量刑建议

**被告人及辩护人辩护意见**
- 被告人对自己行为的辩解
- 辩护人的辩护意见
- 提交的新证据
- 量刑方面的意见

**撰写要求**：
- 如实反映控辩双方的意见
- 不加评论和倾向性表述
- 意见较多的应归纳概括
- 被告人的辩解和辩护人的意见应分别写明

### （七）事实认定

**经审理查明**部分应当：
1. 按时间顺序叙述犯罪事实
2. 每一起犯罪事实作为独立的自然段
3. 事实要素齐全：时间、地点、人物、行为、结果
4. 多人共同犯罪的，写明各自的地位和作用

**证据部分应当**：
1. 分项列举证据
2. 证据类型包括：物证、书证、证人证言、被害人陈述、被告人供述和辩解、鉴定意见、勘验笔录、视听资料等
3. 每项证据后简要说明证明内容
4. 证据之间形成完整的证明体系

**对争议事实的认证**：
- 控辩双方对事实有争议的，必须进行分析
- 说明采信或不采信某方意见的理由
- 对关键证据的证明力进行分析判断

**撰写示例**：
```
经审理查明：
2024年1月15日14时许，被告人张三在本市某区某商场一楼服装专柜处，趁营业员不备，将柜台上的一部手机（价值人民币3000元）盗走，后被商场保安当场抓获。

上述事实，有下列证据予以证实：
1. 被害人王五的陈述，证实其手机被盗的时间、地点和经过；
2. 证人李某（商场保安）的证言，证实目击被告人盗窃并将其抓获的经过；
3. 证人赵某（营业员）的证言，证实手机被盗的情况及价值；
4. 被告人张三的供述和辩解，供认了上述盗窃事实；
5. 物证照片，证实被盗手机的外观特征；
6. 价格鉴定意见，证实被盗手机价值人民币3000元；
7. 现场监控视频，证实被告人实施盗窃的经过。

被告人张三对上述事实供认不讳。辩护人提出被告人系初犯、认罪态度好的辩护意见，本院经审查属实，予以采纳。
```

### （八）判决理由

**本院认为**部分应当：
1. 对犯罪事实进行法律评价
2. 认定被告人的行为构成何罪
3. 分析犯罪构成要件
4. 对量刑情节进行评价
5. 回应控辩双方的法律意见
6. 引出判决结果

**撰写要点**：
- 以"本院认为"开头
- 逻辑清晰，层次分明
- 法律分析透彻，说理充分
- 对辩护意见逐条回应
- 最后引用具体的法律条款

**撰写示例**：
```
本院认为，被告人张三以非法占有为目的，采用秘密窃取的手段盗窃他人财物，数额较大，其行为已构成盗窃罪，依法应予惩处。公诉机关指控被告人张三犯盗窃罪的事实清楚，证据确实充分，指控的罪名成立，本院予以确认。

关于量刑情节：被告人张三系初犯，归案后如实供述自己的罪行，认罪态度较好，依法可以从轻处罚。辩护人所提相关辩护意见成立，本院予以采纳。

关于罚金数额：根据《刑法》第二百六十四条的规定，对被告人判处罚金应当根据犯罪情节轻重确定。本院综合考虑被告人盗窃的数额、认罪态度等情节，决定对其判处罚金人民币6000元。

综上所述，为惩罚犯罪，保护公民的财产权利，维护社会秩序，依照《中华人民共和国刑法》第二百六十四条、第五十二条、第五十三条之规定，判决如下：

### （九）判决结果

判决结果必须明确、具体：

**定罪判刑的表述**：
```
主刑：被告人XXX犯XX罪，判处XXX（刑罚种类和刑期）
附加刑：并处XXX（罚金、剥夺政治权利等）
刑期计算：刑期从判决执行之日起计算，判决执行以前先行羁押的，羁押一日折抵刑期一日。即自XXXX年XX月XX日起至XXXX年XX月XX日止。
```

**定罪免刑的表述**：
```
被告人XXX犯XX罪，情节轻微，有悔罪表现，不需要判处刑罚。依照《中华人民共和国刑法》第三十七条之规定，免予刑事处分。
```

**宣告无罪的表述**：
```
依照《中华人民共和国刑事诉讼法》第一百六十二条第（二）项的规定，宣告被告人XXX无罪。
```

```

**院印**：加盖院印，不得压盖正文

## 四、特殊情况的处理

### 1. 多人犯罪案件

- 事实部分按各被告人分别叙述
- 对主从犯分别认定
- 量刑部分分别表述
- 引渡、追诉时效等问题单独说明

### 2. 共同犯罪案件

- 写明各被告人在共同犯罪中的地位和作用
- 对主犯、从犯分别认定
- 对各被告人分别量刑
- 对追缴、退赔等事项分别处理

### 3. 单位犯罪案件

- 对单位判处罚金
- 对直接负责的主管人员和其他直接责任人员判处刑罚
- 分别写明

### 4. 数罪并罚案件

- 认定数罪的罪名和事实
- 分别写明各罪的量刑
- 写明数罪并罚的原则和结果
- 最终确定执行的刑罚

### 5. 附带民事诉讼

- 判决书主文包含刑事和民事两部分
- 民事部分按民事判决书的要求撰写
- 赔偿数额必须明确

## 五、常见问题解答

### Q1：被告人当庭翻供怎么办？

A1：应当客观记明翻供的内容和理由，分析其翻供的理由是否成立。如果原有证据确实充分，能够认定犯罪事实，即使被告人翻供，仍可认定犯罪事实。

### Q2：关键证据存在瑕疵怎么办？

A2：如果关键证据存在瑕疵，应当说明瑕疵的具体表现，分析该瑕疵是否影响证据的证明力。如果瑕疵无法补正或作出合理解释，该证据应当排除，不作为定案依据。

### Q3：量刑情节相互冲突怎么办？

A3：分别认定各个量刑情节的性质（从轻还是从重），然后根据具体情况综合考量。一般而言，从轻和从重情节可以相互抵消，但不能简单相加减。

### Q4：法律条款引用错误怎么办？

A4：法律条款引用错误属于严重错误，必须纠正。发现错误后应当重新撰写或补正裁定。

### Q5：判决书页数过多怎么办？

A5：事实复杂、证据繁多的案件，判决书可以适当详细。但应当注意：
- 避免不必要的重复
- 证据列举可适当简化
- 说理部分抓住重点

## 六、格式检查清单

在完成判决书撰写后，应当逐项检查以下内容：

- [ ] 法院名称准确
- [ ] 案号完整规范
- [ ] 公诉机关名称正确
- [ ] 被告人信息完整准确
- [ ] 辩护人信息正确
- [ ] 审理经过叙述完整
- [ ] 控辩双方意见如实反映
- [ ] 事实认定要素齐全
- [ ] 证据列举规范
- [ ] 争议事实分析透彻
- [ ] 判决理由说理充分
- [ ] 法律条款引用准确
- [ ] 判决结果明确具体
- [ ] 上诉权利告知完整
- [ ] 审判人员署名齐全
- [ ] 日期格式正确
- [ ] 院印位置正确
- [ ] 书记员署名正确
- [ ] 核对章标注正确
