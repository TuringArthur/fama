# 第一审刑事判决书法律依据参考

## 一、判决书结构要素

### 1. 眉首部分
- 法院名称
- 文书种类（刑事判决书）
- 案号

### 2. 当事人基本情况
- 公诉机关
- 被告人
- 辩护人

### 3. 审理经过
- 起诉机关和起诉时间
- 法院立案时间
- 合议庭组成
- 开庭方式（公开/不公开）
- 庭审人员

### 4. 控辩双方意见
- 公诉机关指控
- 被告人及辩护人辩护意见

### 5. 事实认定
- 经审理查明的事实
- 证据清单
- 对争议事实的分析认证

### 6. 判决理由
- 本院认为
- 法律依据
- 量刑理由

### 7. 判决结果
- 定罪判刑
- 定罪免刑
- 宣告无罪

### 8. 尾部
- 上诉权利告知
- 审判人员署名
- 日期
- 院印
- 书记员署名
- 核对章

## 二、刑事诉讼法相关条款

### 判决的种类（第一百六十二条）

人民法院对案件审理后，根据已查明的事实和证据，依据法律认定被告人有罪或无罪，应作出判决：

1. 案件事实清楚，证据确实、充分，依据法律认定被告人有罪的，应当作出有罪判决
2. 依据法律认定被告人无罪的，应当作出无罪判决
3. 证据不足，不能认定被告人有罪的，应当作出证据不足、指控的犯罪不能成立的无罪判决

### 判决书的送达（第一百六十七条）

送达判决书，应当在宣告判决后立即将判决书送达当事人和提起公诉的人民检察院。

### 判决的宣布（第一百七十八条）

宣告判决，一律公开进行。当庭宣告判决的，应当在五日以内将判决书送达当事人和提起公诉的人民检察院；定期宣告判决的，应当在宣告判决后立即送达当事人和提起公诉的人民检察院。

### 上诉期限（第二百一十九条）

不服判决的上诉和抗诉的期限为十日，不服裁定的上诉和抗诉的期限为五日，从接到判决书、裁定书的第二日起算。

## 三、刑法常用条款速查

### 量刑情节

| 情节类型 | 法律条款 | 内容 |
|---------|---------|------|
| 自首 | 第六十七条 | 犯罪后自动投案，如实供述 |
| 立功 | 第六十八条 | 揭发他人犯罪，查证属实 |
| 坦白 | 第六十七条第三款 | 如实供述自己罪行 |
| 从犯 | 第二十七条 | 起次要或辅助作用 |
| 胁从犯 | 第二十八条 | 被胁迫参加犯罪 |
| 未遂 | 第二十三条 | 已着手实行但未得逞 |
| 中止 | 第二十四条 | 自动放弃或有效防止结果 |
| 累犯 | 第六十五条 | 五年内再犯应判有期徒刑以上之罪 |

### 刑罚种类

| 刑罚类型 | 法律条款 | 说明 |
|---------|---------|------|
| 管制 | 第三十八条 | 社区矫正，3月-2年 |
| 拘役 | 第四十二条 | 1-6月，就近执行 |
| 有期徒刑 | 第四十五条 | 6月-15年 |
| 无期徒刑 | 第四十六条 | 终身监禁 |
| 死刑 | 第四十八条 | 立即执行或缓期二年执行 |
| 罚金 | 第五十二条 | 金钱处罚 |
| 剥夺政治权利 | 第五十四条 | 选举权等政治权利 |
| 没收财产 | 第五十九条 | 部分或全部没收 |

### 主刑与附加刑

主刑（只能独立适用）：
- 管制、拘役、有期徒刑、无期徒刑、死刑

附加刑（可以独立适用，也可以附加适用）：
- 罚金、剥夺政治权利、没收财产

## 四、常见犯罪量刑标准

### 盗窃罪（第二百六十四条）

| 情节 | 法定刑 |
|-----|-------|
| 数额较大（1000-3000元以上） | 三年以下有期徒刑、拘役或管制，并处或单处罚金 |
| 数额巨大（3万-10万元以上） | 三年以上十年以下有期徒刑，并处罚金 |
| 数额特别巨大（30万-50万元以上） | 十年以上有期徒刑或无期徒刑，并处罚金或没收财产 |

### 故意伤害罪（第二百三十四条）

| 情节 | 法定刑 |
|-----|-------|
| 轻伤 | 三年以下有期徒刑、拘役或管制 |
| 重伤 | 三年以上十年以下有期徒刑 |
| 致人死亡或特别残忍手段致重伤造成严重残疾 | 十年以上有期徒刑、无期徒刑或死刑 |

### 诈骗罪（第二百六十六条）

| 情节 | 法定刑 |
|-----|-------|
| 数额较大（3000-10000元以上） | 三年以下有期徒刑、拘役或管制，并处或单处罚金 |
| 数额巨大（3万-10万元以上） | 三年以上十年以下有期徒刑，并处罚金 |
| 数额特别巨大（50万元以上） | 十年以上有期徒刑或无期徒刑，并处罚金或没收财产 |

### 受贿罪（第三百八十五条）

| 情节 | 法定刑 |
|-----|-------|
| 数额较大（3万-20万元） | 三年以下有期徒刑或拘役，并处罚金 |
| 数额巨大（20万-300万元） | 三年以上十年以下有期徒刑，并处罚金或没收财产 |
| 数额特别巨大（300万元以上） | 十年以上有期徒刑、无期徒刑或死刑，并处没收财产 |

## 五、判决书撰写注意事项

### 1. 事实认定部分
- 必须写明具体的时间、地点、人物、行为、结果
- 证据必须经法庭质证
- 对争议事实必须进行分析认证
- 事实描述应当客观、准确

### 2. 判决理由部分
- 必须引用具体的法律条款
- 必须分析犯罪构成要件
- 必须说明量刑理由
- 必须回应控辩双方的意见

### 3. 法律适用部分
- 引用法律条款必须准确
- 避免引用已废止或修改的条款
- 注意法律条款的效力范围

### 4. 判决结果部分
- 罪名必须准确
- 刑期必须明确
- 刑期计算必须准确
- 罚金数额必须明确

## 六、常见错误及纠正

### 错误类型

1. **事实认定不清**
   - 问题：事实描述过于笼统
   - 纠正：按六要素详细描述

2. **证据列举不全**
   - 问题：只列举证据类型
   - 纠正：详细列明证据内容和证明目的

3. **法律引用错误**
   - 问题：引用条款不正确或已失效
   - 纠正：核实条款准确性和时效性

4. **说理不充分**
   - 问题：对辩护意见未回应或回应不充分
   - 纠正：逐条分析并说明采纳与否的理由

5. **格式不规范**
   - 问题：结构不完整或用语不规范
   - 纠正：严格遵循判决书格式规范

## 七、相关司法解释

### 最高人民法院关于常见犯罪的量刑指导意见

1. **量刑的基本方法**
   - 根据基本犯罪构成事实在相应的法定刑幅度内确定量刑起点
   - 根据其他影响犯罪构成的犯罪数额、犯罪次数、犯罪后果等犯罪事实，在量刑起点的基础上增加刑罚量
   - 根据量刑情节调节基准刑，并综合考虑全案情况，依法确定宣告刑

2. **常见量刑情节的适用**
   - 对于自首情节，可以减少基准刑的40%以下
   - 对于立功情节，可以减少基准刑的20%-50%
   - 对于当庭自愿认罪情节，可以减少基准刑的10%以下
   - 对于赔偿被害人经济损失情节，可以减少基准刑的30%以下
   - 对于取得被害人或家属谅解情节，可以减少基准刑的20%以下

## 八、文书签发要求

1. **审核把关**：承办法官对判决书内容负全面责任
2. **签发程序**：经审判长审核后，由院长或授权的审判委员会签发
3. **用印规范**：加盖院印，不得压盖正文
4. **送达时限**：宣判后立即送达，不得延误
