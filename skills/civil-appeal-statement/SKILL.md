# 民事申诉状撰写指南

## 一、民事申诉状概述

民事申诉状是民事诉讼当事人不服人民法院作出的**已经发生法律效力**的民事判决书、裁定书和调解书，依法申请人民法院再次审理此案的法律文书。

### 核心作用

1. **维护申诉人合法权益** - 申诉人认为生效裁判有误损害自身权益时，通过申诉请求纠正错误
2. **维护法律尊严** - 坚持实事求是、有错必纠的社会主义法制原则
3. **监督审判质量** - 是人民法院再审案件的来源之一

## 二、申诉法定情形

当事人申诉必须符合以下条件之一：

| 情形 | 说明 |
|------|------|
| 新证据 | 有新的证据，足以推翻原判决、裁定 |
| 事实认定错误 | 原判决、裁定认定事实的主要证据不足 |
| 法律适用错误 | 原判决、裁定适用法律确有错误 |
| 程序违法 | 人民法院违反法定程序，可能影响案件正确判决、裁定 |
| 法官枉法 | 审判人员在审理该案时有贪污受贿、徇私舞弊、枉法裁判行为 |

**重要时限**：当事人的申请应该在判决、裁定生效后**2年内**提出。

## 三、文书格式

### （一）首部

1. **标题**：在文书顶端居中书写"**民事申诉状**"

2. **当事人基本情况**：依次写明申诉人、被申诉人各自的基本情况（姓名、性别、年龄、民族、籍贯、职业、住址）

### （二）正文

1. **申诉事由**
   - 写明申诉的案件名称
   - 作出生效判决、裁定的人民法院名称
   - 判决编号及制作日期
   - 表明对该判决不服，提出申诉

2. **请求事项**
   - 简明扼要写明要求人民法院依法撤销或变更原审错误裁判的具体请求
   - 多项请求按主次顺序分项列明

3. **事实与理由**
   - 从原判决或裁定认定的事实、引用法律、诉讼程序是否合法等方面进行辩驳
   - 写明请求再审的法律依据
   - 必须真实可靠
   - 注意发掘原审时没有发现的新事实、新证据

### （三）尾部

1. **致送机关**："此致"+"××××人民法院"（分两行书写）

2. **附项**：
   - 本申诉书副本份数
   - 应附的其他材料

3. **落款**：申诉人署名 + 制作日期

## 四、写作注意事项

1. **依法申诉**：申诉请求必须根据有关法律条款规定，在陈述事实、归纳理由的基础上，推导出合理合法的申诉请求

2. **针对性强**：针对原审裁判或调解的错误予以集中反驳，从而全部或部分否定原审裁判；切忌无的放矢、东拉西扯

3. **证据真实**：申诉事实和证据必须真实可靠，注意发掘原审时没有发现的新事实、新证据

## 五、标准模板

```
                        民事申诉状

申诉人（写明基本情况）

被申诉人（写明基本情况）

申诉事由：
申诉人因与被申诉人______纠纷一案，不服______人民法院______号民事判决（裁定），现提出申诉。

请求事项：
1. 撤销______人民法院______号民事判决（裁定）；
2. 依法改判（或发回重审）......

事实与理由：
一、原审认定事实错误......
二、原审适用法律不当......
三、有新的证据......（附新证据清单）

此致
______人民法院

    申诉人：______
    ____年____月____日

附：
1. 本状副本____份
2. 证据材料____份
```

## 六、触发条件确认清单

在开始撰写前，请确认以下信息：

- [ ] 生效裁判的法院名称和案号
- [ ] 申诉人、被申诉人的完整信息
- [ ] 申诉的具体理由（属于法定申诉情形之一）
- [ ] 是否有新证据
- [ ] 申诉请求的具体内容
