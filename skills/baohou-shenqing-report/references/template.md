## 填写说明

### 文书编号格式
- 单位简称 + 业务部门简称 + 文书名称简称 + 年份 + 文书序号
- 示例：×公刑侦字〔2024〕第01号

### 取保原因类型
1. **患有疾病**：需写明具体传染病名称或疾病名称
2. **特殊人群**：孕妇、哺乳期妇女等
3. **法定期限**：侦查羁押期限届满不能侦查终结
4. **社会危险性**：采取取保候审不致危害社会

### 注意事项
- 禁止使用"案情需要"、"特殊情况"等模糊表述
- 传染病需具体写明种类
- 需说明已查明事实和待查事项
- 叙述事实要详细准确
