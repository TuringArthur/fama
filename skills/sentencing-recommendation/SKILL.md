# 量刑建议书生成器

根据用户提供的案件信息，生成规范的量刑建议书HTML文档。

## 输入信息收集

使用多页表单收集以下信息：

### 第一页：案件基本信息
- 被告人姓名
- 被告人性别
- 出生日期
- 民族
- 文化程度
- 职业/身份
- 户籍所在地
- 现住址
- 涉嫌罪名
- 拘留/逮捕日期

### 第二页：案件事实
- 案件发生时间
- 案件发生地点
- 主要犯罪事实（简述）
- 造成的损害后果
- 被害人情况

### 第三页：量刑情节
- 法定从重情节（如有）
- 法定从轻/减轻情节（如有）
- 酌定从轻/从重情节
- 认罪认罚情况
- 赔偿/和解情况

### 第四页：建议内容
- 具体量刑建议（主刑）
- 附加刑建议
- 缓刑适用建议
- 其他建议事项

## 输出格式

生成HTML格式的量刑建议书，包含：
- 检察院名称和文号
- 被告人信息
- 案件事实陈述
- 证据概述
- 量刑情节分析
- 量刑建议及理由
- 检察官签章信息
- 附件清单

## 法律依据引用

引用相关法律条款：
- 《中华人民共和国刑法》相关条款
- 《中华人民共和国刑事诉讼法》
- 最高人民法院量刑指导意见

## 样式规范

- 使用宋体字体，正式文书格式
- 每段首行缩进2字符
- 关键信息加粗或突出显示
- 保持法律文书的严肃性和专业性
