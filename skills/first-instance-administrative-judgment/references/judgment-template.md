## 六种判决类型详细模板

### 1. 维持判决

适用于：证据确凿、适用法律法规正确、符合法定程序

```
依照《中华人民共和国行政诉讼法》第六十九条的规定，判决如下：

维持××（行政机关名称）××年××月××日××字第××号处罚决定（复议决定或者其他行政行为）。
```

### 2. 撤销判决

适用于：主要证据不足、适用法律法规错误、违反法定程序、超越职权、滥用职权、明显不当

```
依照《中华人民共和国行政诉讼法》第七十条的规定，判决如下：

一、撤销××（行政机关名称）××年××月××日××字第××号处罚决定（复议决定或者其他行政行为）。

二、责成被告××（行政机关名称）于本判决生效之日起××日内重新作出行政行为。

三、被告××于本判决生效之日起××日内赔偿原告××（金额）及利息（或者：驳回原告的其他诉讼请求）。
```

### 3. 部分撤销判决

适用于：部分合法、部分违法

```
依照《中华人民共和国行政诉讼法》第七十条的规定，判决如下：

一、维持××（行政机关名称）××年××月××日××字第××号处罚决定的第××项。

二、撤销××（行政机关名称）××年××月××日××字第××号处罚决定的第××项。

三、责成被告××（行政机关名称）于本判决生效之日起××日内重新作出行政行为。
```

### 4. 履行判决

适用于：被告拒绝履行或拖延履行法定职责（如拒绝颁发许可证、拒绝保护人身/财产权利、未依法发放抚恤金等）

```
依照《中华人民共和国行政诉讼法》第七十二条的规定，判决如下：

责成被告××（行政机关名称）于本判决生效之日起××日内履行××（具体法定职责）。
```

### 5. 变更判决

适用于：行政处罚明显不当，或涉及款额确定的其他行政行为在事实认定/计算上有错误

```
依照《中华人民共和国行政诉讼法》第七十七条的规定，判决如下：

将被告××（行政机关名称）××年××月××日××字第××号处罚决定中的"××（原处罚内容）"变更为"××（变更后内容）"。
```

### 6. 行政赔偿判决

适用于：行政行为造成损害需赔偿

```
依照《中华人民共和国行政诉讼法》第七十八条的规定，判决如下：

一、被告××于本判决生效之日起××日内赔偿原告××（金额）及利息。

二、被告××于本判决生效之日起××日内返还原物（恢复原状）。
```

## 法律依据条款

根据《法律文书写作（第五版）》教材：

- **《行政诉讼法》第34条第1款**：被告对作出的行政行为负有举证责任
- **《行政诉讼法》第63条**：审理行政案件以法律和行政法规、地方性法规为依据
- **《行政诉讼法》第69条**：行政行为证据确凿，适用法律、法规正确，符合法定程序的，判决维持
- **《行政诉讼法》第70条**：主要证据不足、适用法律法规错误、违反法定程序、超越职权、滥用职权、明显不当的，判决撤销或部分撤销
- **《行政诉讼法》第72条**：被告不履行法定职责的，判决限期履行
- **《行政诉讼法》第77条**：行政处罚明显不当或其他行政行为涉及款额确定的，可以判决变更
- **《行政诉讼法》第78条**：被告违法造成损害的，判决赔偿
