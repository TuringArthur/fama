# 第一审人民法院行政判决书

本技能用于根据案件信息生成规范的第一审行政判决书。

## 工作流程

### 1. 收集案件基本信息

收集以下关键信息：
- 法院名称
- 案号格式：`(年份) ××行初×号`
- 原告信息（姓名/名称、法定代表人、委托代理人）
- 被告信息（行政机关名称、地址、法定代表人）
- 第三人信息（如有）
- 原行政行为（行政机关名称、日期、文号、行为类型）

### 2. 收集案件事实

收集以下事实材料：
- 被诉行政行为的主要内容
- 被告的证据和法律依据
- 原告不服的原因和诉讼请求
- 被告的答辩意见
- 第三人意见（如有）
- 法院认定的事实和证据

### 3. 选择判决类型

根据案件情况选择适当的判决类型：

| 判决类型 | 适用情形 |
|---------|---------|
| 维持判决 | 证据确凿、适用法律正确、符合法定程序 |
| 撤销判决 | 主要证据不足、适用法律错误、违反法定程序、超越职权、滥用职权、明显不当 |
| 部分撤销 | 部分合法、部分违法 |
| 履行判决 | 拒绝履行或拖延履行法定职责 |
| 变更判决 | 行政处罚明显不当、款额确定错误 |
| 赔偿判决 | 行政行为造成损害需赔偿 |

### 4. 生成判决书

按照以下结构生成判决书：

#### 首部
- 法院名称（分行书写）
- 文书标题"行政判决书"
- 案号
- 当事人信息
- 案件由来与审理经过

#### 事实
- 当事人争议内容
- 法院认定的事实和证据

#### 理由
- 判决理由分析
- 适用的法律条款

#### 判决结果
按照六种判决类型选择相应表述。

#### 尾部
- 合议庭成员署名
- 日期
- 书记员
- 上诉权利告知

## 判决结果表述模板

### 维持判决
```
维持XX（行政机关名称）XX年XX月XX日XX字第XX号处罚决定（复议决定或者其他行政行为）。
```

### 撤销判决
```
一、撤销XX（行政机关名称）XX年XX月XX日XX字第XX号处罚决定（复议决定或者其他行政行为）。

二、责成被告XX（行政机关名称）于本判决生效之日起XX日内重新作出行政行为。

三、被告XX于本判决生效之日起XX日内赔偿原告XX（金额）及利息。
```

### 部分撤销判决
```
一、维持XX（行政机关名称）XX年XX月XX日XX字第XX号处罚决定的第XX项。

二、撤销XX（行政机关名称）XX年XX月XX日XX字第XX号处罚决定的第XX项。

三、责成被告XX（行政机关名称）于本判决生效之日起XX日内重新作出行政行为。
```

### 履行判决
```
责成被告XX（行政机关名称）于本判决生效之日起XX日内履行XX（具体法定职责）。
```

### 变更判决
```
将被告XX（行政机关名称）XX年XX月XX日XX字第XX号处罚决定中的"XX（原处罚内容）"变更为"XX（变更后内容）"。
```

### 赔偿判决
```
被告XX于本判决生效之日起XX日内赔偿原告XX（金额）。

或者：被告XX于本判决生效之日起XX日内返还原物（恢复原状）。
```

## 上诉权利告知

```
如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于XX人民法院。
```

## 签署格式

```
审判长 XXX
审判员 XXX
审判员 XXX

XX年XX月XX日

（院印）

书记员 XXX
```

## 证据规则

根据《行政诉讼法》第34条第1款，被告对作出的行政行为负有举证责任。认定事实所依据的证据必须经过庭审质证属实。
