# 检察机关补充侦查决定书制作技能

## 一、补充侦查决定书的概念
人民检察院审查案件，认为犯罪事实不清、证据不足，或者遗漏了罪行，或者还有其他应当追究刑事责任的同案犯没有追究，需要补充侦查时，就可以退回公安机关补充侦查。退回补充侦查时，人民检察院应用书面形式表达，这种文书就叫补充侦查决定书。

## 二、补充侦查决定书的内容构成
补充侦查决定书主要由首部、主文、尾部、附项四部分构成：

### 1. 首部
- 文书名称：写明“××人民检察院补充侦查决定书”，其右下方注明“××检××补侦〔××〕×号”；
- 退回机关：写明退回补充侦查的机关，即报请审查的原公安机关；
- 案由与案件来源：行文格式为“你局____年____月____日____字第____号移送起诉的被告人的案卷材料收悉。经本院审查认为：……”。

### 2. 主文
主文是补充侦查决定书的主体部分，着重阐明退回补充侦查的理由，制作时需注意以下两方面问题：
- 退回问题的界定：必须是案件上的重大问题，对定案有决定性的影响（如主要犯罪事实情节不清、重要证据不足、遗漏了重要罪行或同案犯）；若是一般的、较小的、对定案无重大影响的问题，或检察院自行侦查易解决的问题，无需退回公安机关补充侦查；
- 问题描述要求：必须具体明确地指出问题，便于公安机关有目的地开展补充侦查；若表述笼统含糊，会导致公安机关无从着手，反复退回补充侦查，耽误办案时间、影响办案效率。
- 实践建议：部分检察机关为防止扯皮，会事先与公安机关相关人员交换意见，征得同意后再出具书面补充侦查决定，提升办案效率。

### 3. 尾部
写明作出退回补充侦查决定的检察机关的名称，注明制作文书的年、月、日，并加盖院章。

### 4. 附项
写明“补充侦查的事项”，明确需要补充侦查的具体内容。

## 三、补充侦查决定书的格式

## 四、制作核心要求
1. 理由针对性：主文部分的退回理由需紧扣“重大问题、定案关键”原则，不罗列无关细节；
2. 内容明确性：补充侦查事项需具体可操作，避免模糊表述；
3. 格式规范性：严格按照首部、主文、尾部、附项的结构制作，文书编号、印章、日期等要素完整；
4. 程序合规性：退回补充侦查的适用场景需符合“犯罪事实不清、证据不足、遗漏罪行/同案犯”的法定情形。
