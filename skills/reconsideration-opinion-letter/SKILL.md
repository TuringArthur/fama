# 要求复议意见书技能

根据公安机关办案规范和《刑事诉讼法》要求，生成符合2161标准格式的要求复议意见书。

## 工作流程

### 1. 信息收集

收集以下必要信息（如信息不完整，询问用户补充）：

- **案件基本信息**：案件名称、案号
- **检察院决定**：不批准逮捕/不起诉决定书编号及日期
- **复议理由**：公安机关认为检察院决定有误的具体理由
- **法律依据**：引用的具体法律条款
- **涉案人员**：犯罪嫌疑人基本情况
- **案卷情况**：案卷册数及页数

### 2. 文书结构生成

按照2161格式规范生成三部分内容：

#### 首部
```
××××公安局
要求复议意见书
×公××字〔××××〕第××号
××××人民检察院：
```

#### 正文

**案由（引言）**：
- 检察院决定的文书编号和案件名称
- 公安机关认为决定有误的简要意见

**要求复议的理由和意见**（重点部分）：
- 分析检察院决定的具体错误之处
- 论证公安机关提请逮捕/移送起诉意见的正确性
- 结合案件事实和证据

**法律依据和要求**：
- 引用《刑事诉讼法》第70条（不批准逮捕）
- 或第144条（不起诉）
- 提出复议请求

#### 尾部
```
附：案卷　册　页

（发文机关盖章）
××××年××月××日
```

### 3. 输出规范

- 法律术语使用准确
- 理由阐述逻辑清晰、有据可依
- 格式严格遵循2161标准
- 语言庄重、严谨

## 质量检查清单

- [ ] 首部要素齐全（机关名称、文书名称、编号、受文机关）
- [ ] 案由表述准确引用检察院决定文书
- [ ] 复议理由充分、有针对性
- [ ] 法律依据引用正确
- [ ] 尾部格式规范
- [ ] 日期和盖章位置正确
