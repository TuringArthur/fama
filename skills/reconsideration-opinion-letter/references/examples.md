# 要求复议意见书示例

## 示例一：针对不批准逮捕的复议意见书

```
　　　　　　　　　　　　××县公安局
　　　　　　　　　　要求复议意见书
×公刑复字〔2024〕第8号

××县人民检察院：

　　你院于2024年3月15日以×检刑不批准逮捕字〔2024〕第12号不批准逮捕决定书，对犯罪嫌疑人王××作出不批准逮捕的决定。我局认为该决定确有错误，现依法提出复议意见。

　　一、犯罪嫌疑人王××的犯罪事实
　　（一）2024年1月10日，犯罪嫌疑人王××在××县××路盗窃被害人李××现金人民币15000元及手机一部（价值人民币3000元）。
　　（二）2024年1月15日，犯罪嫌疑人王××在××县××商场盗窃化妆品等物品（价值人民币8000元）。
　　（三）犯罪嫌疑人王××曾因盗窃罪于2020年被判处有期徒刑一年，刑满释放后五年内又犯应当判处有期徒刑以上刑罚之罪，系累犯。

　　二、上述事实的证据
　　（一）被害人李××的陈述，证实被盗现金及手机的情况；
　　（二）被害人张××的陈述，证实被盗化妆品的情况；
　　（三）监控录像，证实犯罪嫌疑人王××实施盗窃的过程；
　　（四）犯罪嫌疑人王××的供述与辩解，其对上述犯罪事实供认不讳；
　　（五）价格认定结论书，证实被盗物品价值；
　　（六）刑事判决书，证实犯罪嫌疑人王××系累犯。

　　三、我局认为应当逮捕的理由
　　（一）犯罪嫌疑人王××盗窃数额较大，共计人民币26000元，已达到盗窃罪数额巨大的标准；
　　（二）犯罪嫌疑人王××系累犯，主观恶性大，有再犯罪的危险性；
　　（三）犯罪嫌疑人王××拒不退还赃款赃物，社会危害性较大；
　　（四）犯罪嫌疑人王××无固定住所，无正当职业，取保候审存在逃跑风险。

　　四、人民检察院决定错误之处
　　（一）你院认定犯罪嫌疑人王××犯罪情节轻微与事实不符，其盗窃数额巨大且系累犯；
　　（二）你院未充分考虑犯罪嫌疑人王××的人身危险性和社会危害性；
　　（三）你院对累犯应当从重处罚的法律规定未予充分考量。

　　综上所述，犯罪嫌疑人王××的行为已涉嫌盗窃罪，且系累犯，依法应当从重处罚。根据《中华人民共和国刑事诉讼法》第七十条之规定，特请你院进行复议，依法撤销原决定，对犯罪嫌疑人王××作出批准逮捕的决定。

　　附：案卷　1　册　86　页

　　　　　　　　　　　　　　××县公安局（印）

　　　　　　　　　　　　　　2024年3月25日
```

## 示例二：针对不起诉决定的复议意见书

```
　　　　　　　　　　　　××市公安局
　　　　　　　　　　要求复议意见书
×公诉复字〔2024〕第15号

××市人民检察院：

　　你院于2024年4月20日以×检刑不诉字〔2024〕第25号不起诉决定书，对犯罪嫌疑人张××作出不起诉的决定。我局认为该决定确有错误，现依法提出复议意见。

　　一、犯罪嫌疑人张××的犯罪事实
　　（一）2023年11月5日，犯罪嫌疑人张××在××市××区××酒店内，与他人共同组织卖淫活动，非法获利人民币50000元。
　　（二）犯罪嫌疑人张××系该卖淫团伙的主要负责人，负责管理卖淫人员、收取嫖资等主要工作。
　　（三）2023年12月10日，犯罪嫌疑人张××被公安机关查获归案。

　　二、上述事实的证据
　　（一）证人证言：卖淫人员赵××、孙××的证言，证实张××对其进行管理并收取嫖资；
　　（二）嫖客证言：钱××、周××的证言，证实向张××支付嫖资；
　　（三）物证：记账本、收款二维码等，证实非法获利情况；
　　（四）犯罪嫌疑人张××的供述与辩解，其对上述犯罪事实供认不讳；
　　（五）银行交易记录，证实资金流转情况。

　　三、我局认为应当起诉的理由
　　（一）犯罪嫌疑人张××组织卖淫人数达10人以上，属情节严重；
　　（二）犯罪嫌疑人张××非法获利数额巨大，达人民币50000元；
　　（三）犯罪嫌疑人张××在犯罪组织中起主要作用，系主犯；
　　（四）犯罪嫌疑人张××的行为严重败坏社会风气，社会危害性极大。

　　四、人民检察院决定错误之处
　　（一）你院认定犯罪嫌疑人张××犯罪情节轻微与事实不符，其行为属情节严重；
　　（二）你院未充分考虑犯罪嫌疑人张××的组织、领导作用；
　　（三）你院对组织卖淫罪"情节严重"的认定标准理解有误。

　　综上所述，犯罪嫌疑人张××的行为已涉嫌组织卖淫罪，情节严重，依法应当追究其刑事责任。根据《中华人民共和国刑事诉讼法》第一百四十四条之规定，特请你院进行复议，依法撤销原决定，对犯罪嫌疑人张××提起公诉。

　　附：案卷　2　册　156　页

　　　　　　　　　　　　　　××市公安局（印）

　　　　　　　　　　　　　　2024年5月8日
```
