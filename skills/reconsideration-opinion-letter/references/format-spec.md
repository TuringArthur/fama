# 要求复议意见书格式规范（2161格式）

## 一、文书定义

要求复议意见书是公安机关认为同级人民检察院的不批准逮捕、不起诉决定有错误时，依法要求人民检察院对其原决定重新进行审议时所制作的文书。

## 二、文书作用

1. 保障公安机关依法行使复议权
2. 对人民检察院的决定提出不同意见的司法凭证
3. 促使人民检察院严格依法办案
4. 保证办案质量，使犯罪分子得到应有的惩罚

## 三、格式结构

### （一）首部

| 要素 | 内容 | 格式要求 |
|------|------|----------|
| 制作机关 | ××××公安局 | 居中 |
| 文书名称 | 要求复议意见书 | 居中 |
| 文书编号 | ×公××字〔××××〕第××号 | - |
| 受文机关 | ××××人民检察院 | 顶格 |

**文书编号说明**：
- 依次写明：制作机关简称 + 案件性质 + 文书名称简称 + 制作年份 + 案件顺序号
- 示例：`×公×刑字〔2024〕第15号`

### （二）正文

#### 1. 案由（引言）

**内容要求**：
- 写明人民检察院作出不批准、不起诉决定的文书编号
- 写明案件名称
- 表明制作机关认为同级人民检察院的决定有误的简要意见

**格式示例**：
```
　　你院于××××年××月××日以×××检×不诉字〔××××〕第×号不起诉决定书（不批准逮捕决定书），对犯罪嫌疑人×××作出不起诉（不批准逮捕）的决定。
我局认为该决定确有错误，现依法提出复议意见。
```

#### 2. 要求复议的理由和意见（重点部分）

**内容要求**：
- 分析人民检察院不批准逮捕、不起诉决定的错误
- 证明公安机关对犯罪嫌疑人提请逮捕或者移送起诉处理意见的正确性
- 结合案件具体事实和证据

**写作要点**：
- 针对性强：针对检察院决定的具体问题
- 论据充分：引用案件事实和证据
- 论证有力：说明为什么应当逮捕/起诉

**格式示例**：
```
　　一、犯罪嫌疑人×××的犯罪事实
　　（一）……
　　（二）……

　　二、上述事实的证据
　　（一）……

　　三、我局认为应当逮捕（起诉）的理由
　　（一）……

　　四、人民检察院决定错误之处
　　（一）……
```

#### 3. 提出复议的法律依据和要求

**法律依据**：
| 复议类型 | 引用条款 |
|----------|----------|
| 不批准逮捕复议 | 《刑事诉讼法》第70条 |
| 不起诉复议 | 《刑事诉讼法》第144条 |

**格式示例**：
```
　　综上所述，根据《中华人民共和国刑事诉讼法》第××条之规定，特请你院进行复议，依法撤销原决定，对犯罪嫌疑人×××作出批准逮捕（提起公诉）的决定。
```

### （三）尾部

#### 1. 附项

**格式示例**：
```
　　附：案卷　×　册　×××　页
```

#### 2. 落款

**格式示例**：
```
　　　　　　　　　　　　××××公安局（印）

　　　　　　　　　　　　××××年××月××日
```

## 四、写作注意事项

1. **针对性强**：针对人民检察院作出的决定的有误之处进行分析
2. **结合案情**：结合案件的具体情况阐明复议理由
3. **证据充分**：引用充分的证据支持复议理由
4. **法律准确**：准确引用相关法律条款
5. **格式规范**：严格遵循2161格式要求

## 五、不批准逮捕复议与不起诉复议的区别

| 区别项 | 不批准逮捕复议 | 不起诉复议 |
|--------|---------------|-----------|
| 法律依据 | 《刑事诉讼法》第70条 | 《刑事诉讼法》第144条 |
| 复议理由 | 证明应当逮捕的理由 | 证明应当起诉的理由 |
| 请求结果 | 批准逮捕 | 提起公诉 |
