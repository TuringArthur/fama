# 行政上诉状撰写技能

## 核心任务

帮助用户撰写符合法律规范的行政上诉状，引起第二审程序的发生。

## 工作流程

### 1. 信息收集

向用户收集以下必要信息：

**案件基本信息**
- 案件名称
- 原审法院名称
- 原审判决/裁定字号（如"×行初字第×号"）
- 原审日期
- 原审判决/裁定结果

**当事人信息**
- 上诉人信息（姓名/单位名称、性别、年龄、民族、籍贯、职业、工作单位及职务、住址；或单位名称、法定代表人、住所地）
- 被上诉人信息（同上格式）

**上诉请求**
- 明确具体的上诉请求（撤销、变更原判决或重新审理）

**上诉理由**
- 原判决在认定事实方面的问题
- 原判决在适用法律方面的问题
- 相关法律、法规依据
- 如有新证据，请提供

### 2. 文档结构

按以下结构撰写行政上诉状：

```
┌─────────────────────────────────────┐
│          行政上诉状（标题居中）        │
├─────────────────────────────────────┤
│ 【首部】                             │
│ 上诉人：...（括号注明一审诉讼地位）    │
│ 被上诉人：...（括号注明一审诉讼地位）  │
├─────────────────────────────────────┤
│ 【正文】                             │
│ 一、上诉事由                          │
│ "上诉人因××一案，不服××人民法院      │
│ ×年×月×日的（××）×行初字第×号       │
│ 判决/裁定，现提出上诉。"              │
│                                     │
│ 二、上诉请求                          │
│ （分点列出具体请求）                  │
│                                     │
│ 三、上诉理由                          │
│ （针对原判决问题进行论述）             │
├─────────────────────────────────────┤
│ 【尾部】                             │
│ 此致                                │
│ ×××人民法院                          │
│                                     │
│ 附：本上诉状副本×份                   │
│ 证据×份                              │
│                                     │
│ 上诉人：×××（签名或盖章）            │
│ ×年×月×日                           │
└─────────────────────────────────────┘
```

### 3. 撰写要点

**首部要求**
- 标题居中书写
- 当事人为自然人：姓名、性别、年龄、民族、籍贯、职业、工作单位及职务、住址
- 当事人为法人：单位名称、法定代表人、住所地、联系方式和邮政编码
- 行政机关作为被上诉人：行政机关名称、法定代表人、住所地

**正文要求**
- 上诉事由：写明不服何法院何字号的判决或裁定
- 上诉请求：明确具体，可分点列出
- 上诉理由：
  - 认真分析第一审判决，找出存在问题
  - 引用相关法律、法规进行论述
  - 抓住关键性问题，避免枝节纠缠
  - 如有新证据应一并提供

**尾部要求**
- 致送机关分两行书写
- 附项写明副本份数和证据
- 上诉人签名或盖章并注明日期

### 4. 质量检查

撰写完成后核对：
- [ ] 项目齐全无遗漏
- [ ] 上诉请求明确具体
- [ ] 上诉理由有理有据
- [ ] 法律依据准确引用
- [ ] 格式符合规范要求

## 输出格式

直接输出行政上诉状全文，供用户使用和修改。
