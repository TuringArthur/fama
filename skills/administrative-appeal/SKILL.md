# 行政申诉状生成器

## 触发条件

当用户要求撰写行政申诉状或相关法律文书时激活本技能。

## 工作流程

### 1. 收集必要信息

收集以下关键信息：
- **申诉人信息**：姓名/单位全称、身份（原告/被告/上诉人/被上诉人）、联系方式
- **原审信息**：原审法院名称、案号、判决/裁定日期
- **案件类型**：行政处罚、行政许可、行政强制等
- **申诉请求**：撤销原判、变更原判、发回重审等
- **申诉理由**：事实错误、法律适用错误、程序违法等
- **证据材料**：新证据清单、证人信息

### 2. 生成文书结构

```
┌─────────────────────────────────────┐
│           申诉状（标题）             │
├─────────────────────────────────────┤
│ 一、首部                              │
│   （一）申诉人基本情况                │
│   （二）原审诉讼地位                  │
├─────────────────────────────────────┤
│ 二、正文                              │
│   （一）申诉事由                      │
│   （二）申诉请求                      │
│   （三）事实与理由                    │
│       1. 原审认定事实概括              │
│       2. 原审错误/不当之处            │
│       3. 正确事实陈述                 │
│       4. 法律依据论证                 │
├─────────────────────────────────────┤
│ 三、尾部                              │
│   （一）致送机关                      │
│   （二）附项（证据清单）               │
│   （三）落款（签名、日期）             │
└─────────────────────────────────────┘
```

### 3. 撰写要点

#### 首部格式
```markdown
申诉人：×××（单位全称）
法定代表人：×××（职务）
委托代理人：×××（职务）
（括号注明原审诉讼地位：如"原审原告"）
```

#### 正文格式
```markdown
申诉人×××因××××一案，不服××××人民法院
××××年××月××日（××××）×字第××号行政判决
（或裁定），现提出申诉。申诉请求和理由如下：
```

#### 尾部格式
```markdown
此致
××××人民法院

附：1. 申诉状副本×份
    2. 证据材料×份
    3. 证人名单

申诉人：×××（签名或盖章）
××××年××月××日
```

### 4. 质量检查清单

- [ ] 申诉人信息完整准确
- [ ] 原审案号、判决日期正确
- [ ] 申诉请求明确具体
- [ ] 事实陈述与证据材料对应
- [ ] 法律依据引用准确
- [ ] 格式规范，层次清晰
- [ ] 附项列明证据材料

## 注意事项

1. 申诉理由必须紧扣案件事实和法律条文
2. 不得使用未经查证的传闻
3. 需注明在原生效裁判中的诉讼地位
4. 可同时申请暂行停止执行
