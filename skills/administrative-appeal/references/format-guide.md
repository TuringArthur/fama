# 行政申诉状格式规范与示例

## 一、文书定义

行政申诉状是行政诉讼当事人不服人民法院作出的已经发生法律效力的行政判决或裁定，申请人民法院再次审理此案时制作的法律文书。

## 二、格式写法

### （一）首部

#### 1. 标题
在文书顶端居中书写文书名称"申诉状"。

#### 2. 申诉人基本情况

**格式要求：**
```
申诉人：×××（姓名或单位全称）
法定代表人：×××（职务）
```

**分类说明：**

| 申诉人类型 | 必填信息 |
|-----------|---------|
| 行政机关 | 全称、法定代表人姓名、职务 |
| 其他组织 | 名称、主要负责人姓名、职务 |
| 公民 | 姓名、性别、出生日期、民族、职业、住址 |

**重要提示：** 必须用括号注明申诉人在原审中的诉讼地位
- 第一审程序：原告/被告
- 第二审程序：上诉人/被上诉人

### （二）正文

#### 1. 申诉事由

**标准格式：**
```markdown
申诉人×××（姓名）因××××（案由）一案，不服××××（原审法院全称）
××××年××月××日（××××）×字第××号行政判决（或裁定），
现提出申诉。申诉请求和理由如下：
```

#### 2. 申诉请求

写明具体要求：
- 撤销原判
- 变更原判
- 发回重审
- 其他请求

#### 3. 事实与理由

**内容结构：**

1. **原审认定事实概括**
   - 简要概括原审法院认定的事实和证据

2. **原审错误/不当之处**（针对性阐述）
   - 认定事实错误
   - 事实不清、证据不足
   - 适用法律错误
   - 严重违反法定诉讼程序

3. **正确事实陈述**
   - 陈述正确的事实
   - 运用证据材料论证真实性

4. **法律论证**
   - 引用相关法律法规
   - 论证申诉请求的合理性、合法性

### （三）尾部

#### 1. 致送机关
```
此致
××××人民法院
```

#### 2. 附项
```
附：1. 申诉状副本×份
    2. 证据材料清单
    3. 证人名单（如有）
```

#### 3. 落款
```
申诉人：×××（签名或盖章）
××××年××月××日
```

# 申诉状

申诉人（（原审原告））：××省××市工商行政管理局
法定代表人：王××，局长
委托代理人：李××，该局法制科科长

申诉人因不服××市中级人民法院（2023）×行初字第45号行政判决，现提出申诉。申诉请求和理由如下：

## 一、申诉请求

1. 请求撤销××市中级人民法院（2023）×行初字第45号行政判决；
2. 请求维持申诉人作出的×工商罚字（2023）第12号行政处罚决定。

## 二、事实与理由

### （一）原审认定事实概述

原审法院认定，申诉人于2023年5月对被申诉人××公司作出行政处罚决定，认定该公司存在虚假宣传行为，处以罚款人民币10万元。

### （二）原审错误之处

1. **认定事实错误**
   原审法院认定申诉人程序违法，但申诉人在作出处罚决定前已依法履行了告知义务，听取了当事人的陈述和申辩，程序合法有效。

2. **适用法律错误**
   原审法院援引《行政处罚法》第三十七条，但该条款适用于案件调查阶段，而非处罚决定阶段。申诉人适用《行政处罚法》第四十六条第一款第（三）项作出处罚，适用法律正确。

3. **证据审查不全面**
   申诉人在原审中提交的现场检查笔录、证人证言、公司宣传资料等证据，足以证明被申诉人存在虚假宣传行为，原审法院对上述证据未予充分审查。

### （三）正确事实

1. 2023年3月15日，申诉人接到消费者举报，对被申诉人进行现场检查；
2. 经查，被申诉人在其官方网站及宣传材料中声称产品具有"国家专利技术"，但经核实，该技术并未取得专利授权；
3. 2023年4月20日，申诉人依法向被申诉人送达了行政处罚告知书；
4. 被申诉人在法定期限内未提出陈述和申辩；
5. 2023年5月10日，申诉人依法作出行政处罚决定。

### （四）法律依据

根据《中华人民共和国行政处罚法》第四十六条第一款第（三）项及《中华人民共和国反不正当竞争法》第八条第一款之规定，被申诉人的行为构成虚假宣传，申诉人有权依法予以处罚。

综上所述，原审判决认定事实不清、适用法律错误，特请求上级人民法院依法撤销原判，维护申诉人的合法权益。

此致

××省高级人民法院

附：1. 本申诉状副本3份
    2. 行政处罚决定书复印件1份
    3. 现场检查笔录1份
    4. 宣传资料复印件5份
    5. 证人证言2份

申诉人：××省××市工商行政管理局（盖章）
法定代表人：王××（签字）
××××年××月××日

## 四、写作注意事项

1. **当事人称谓要规范、准确**，必须注明在原生效判决或裁定中的诉讼地位

2. **充分运用事实论据**进行说理、反驳及论证

3. **紧扣案件事实和法律、法规**阐述理由，绝不能把未经查证的传闻写进申诉状

4. **可同时提出暂行停止执行申请**，经人民法院审查后，如认为申请有理，可裁定暂停对已生效判决或裁定的执行

5. **证据材料需完整列明**，包括新发现的证据
