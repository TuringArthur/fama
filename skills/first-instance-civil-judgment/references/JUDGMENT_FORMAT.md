## 三、撰写注意事项

### 1. 格式要求
- 法院名称居中
- 案号位于标题右下方
- 当事人信息按原告、被告、第三人顺序排列
- 使用"××"作为占位符
- 日期使用阿拉伯数字

### 2. 内容要求
- 事实认定要清楚
- 证据表述要明确
- 法律适用要准确
- 判决结果要具体明确
- 诉讼费用负担要写明

### 3. 常用法律依据
- 《中华人民共和国民法典》第××条（合同、物权、人格权等）
- 《中华人民共和国民法典》婚姻家庭编相关条款
- 《中华人民共和国民事诉讼法》第××条

### 4. 上诉权利告知
- 必须在尾部写明上诉期限（15日）
- 写明上诉法院
- 写明上诉方式（递交上诉状）
