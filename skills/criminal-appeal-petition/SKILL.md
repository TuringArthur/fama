# 刑事申诉状生成器

## 触发识别

当用户请求以下内容时触发本技能：
- 撰写/编写/制作刑事申诉状
- 刑事申诉状格式/模板/范例
- 刑事申诉状怎么写
- 申请刑事再审/申诉

## 信息收集清单

生成刑事申诉状前，必须收集以下信息：

### 申诉人信息
- 申诉人姓名、性别、出生日期、民族、籍贯
- 职业、工作单位及职务、住址
- 与原案被告人的关系（本人/法定代理人/近亲属）
- 如在押或服刑，需注明在押或服刑处所

### 原案信息
- 原审法院名称
- 判决/裁定日期
- 案号（如：（2023）X刑初字第XX号）
- 判决结果
- 原审认定的事实

### 申诉理由
- 原判决存在的错误或不当之处
- 事实真相与原判认定的差异
- 法律依据和申诉请求

### 致送机关
- 人民法院 或 人民检察院

## 输出格式规范

### 1. 首部

```
                    刑事申诉状

申诉人：XXX（身份信息）
    姓名：XXX
    性别：X
    出生日期：XXXX年XX月XX日
    民族：XX族
    籍贯：XX省XX市
    职业：XXX
    工作单位及职务：XXX
    住址：XXX
    与被告人关系：XXX（如适用）
    在押/服刑处所：XXX（如适用）

（律师代理时）
委托诉讼代理人：XXX，XXX律师事务所律师
```

### 2. 正文

**申诉事由：**
申诉人因XXX一案，对XXX人民法院XXXX年XX月XX日（XXXX）X刑X字第XX号刑事判决（裁定），提出申诉。

**请求事项：**
1. 请求XXX人民检察院依法提出抗诉；（向检察院申诉时）
2. 请求XXX人民法院依法再审，撤销（变更）原判决（裁定）。（向法院申诉时）

**事实与理由：**

一、原审认定的事实
（叙述原裁判认定的案件事实）

二、原判决存在的问题
（指出原判决的错误或不当之处）

三、事实真相
（实事求是地说明案件真实情况）

四、申诉理由及法律依据
（阐述申诉的具体理由，援引相关法律条文）

### 3. 尾部

```
此致
XXX人民法院（或XXX人民检察院）

附：原审判决书复印件 X 份

申诉人：XXX（签名）
XXXX年XX月XX日
```

## 写作注意事项

### 请求事项撰写
- 根据具体案情书写，因案而异
- 必须符合法律规定
- 明确请求法院再审还是检察院抗诉

### 申诉理由排序
- **主要错误问题放在前面**阐明
- 次要错误问题放在后面阐明
- 逻辑清晰，层次分明

### 事实叙述要点
- 按最后一次判决认定的事实书写
- 如有多次判决，应予说明
- 实事求是，不夸大不缩小

### 法律依据
- 明确援引相关法律条文
- 可引用《刑事诉讼法》相关规定
- 说明原判决违反的具体法律条款

## 适用场景

| 场景 | 致送机关 | 请求重点 |
|------|----------|----------|
| 对判决不服申诉 | 人民法院/人民检察院 | 请求再审 |
| 请求检察院抗诉 | 人民检察院 | 请求提起抗诉 |
| 对裁定不服 | 人民法院 | 请求重新审查 |

## 质量检查清单

生成完成后，核对以下要点：

- [ ] 申诉人身份信息完整准确
- [ ] 原审法院、案号、判决日期正确
- [ ] 申诉事由表述规范
- [ ] 请求事项明确具体
- [ ] 事实与理由层次清晰
- [ ] 主要错误优先阐述
- [ ] 法律依据充分
- [ ] 尾部格式正确（致送机关、附项、落款）
- [ ] 语言严谨、措辞规范
