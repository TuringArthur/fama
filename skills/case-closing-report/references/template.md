# 呈请破案报告书 - 规范参考

## 一、文书基本信息

- **文书名称**：《呈请破案报告书》
- **文书性质**：刑事侦查终结后的总结性法律文书，是侦查阶段的终点
- **适用范围**：重大或典型案件需写总结性书面材料；一般案件可填写标准格式

## 二、标准格式（五部分结构）

### 第一部分：犯罪嫌疑人的基本情况

```
姓名：___    性别：___    出生日期：___年___月___日
出生地：___省___市（区）___县
身份证号码：___________________________
民族：___    文化程度：___
职业：___    工作单位及职务：___
政治面貌：___（如为人大代表、政协委员需注明）
居住地：___________________________
采取强制措施情况：___________________
简历：_____________________________
```

**特殊情况**：
- 如尚未确定犯罪嫌疑人，写明案件基本情况
- 如涉及多人，需逐一列出基本情况

### 第二部分：呈请事项

```
呈请事项：___________________________
```

### 第三部分：事实依据

**案件事实**：详细叙述有关案件事实

**证据分析**：对现有证据进行详细分析，证明事实的成立

### 第四部分：法律依据

```
依据《中华人民共和国刑事诉讼法》第___条之规定，___________________________
```

### 第五部分：结语和落款

```

                              （盖章）
```

## 三、内容编写要求（五核心板块）

### 1. 案情概况
- 主要内容：案件发生、发现的时间地点，现场勘查情况，调查的情况
- 要求：从"真相大白"角度出发，对立案时的不确定材料进行剔除，概要叙述

### 2. 案情分析
- 内容：在立案分析基础上，结合侦查终结时所得的证据、材料、线索进行深度分析
- 要求：得出更准确、全面、符合实情的判断

### 3. 侦查方案
- 内容：回顾原订的侦查计划
- 要求：根据侦查实践中的修改、补充，反映更全面的了解和更可靠的判断依据

### 4. 破案经过（核心部分）
- **执行方案情况**：方案的变更、修正和补充
- **查明与抓捕**：查明犯罪嫌疑人的过程，执行逮捕（或采取强制措施）的情况
- **预审情况**：审讯的过程和结果
- **编写方式**：可按时间顺序编写，也可按问题、线索逐步查询、疑点逐一查清的过程编写

### 5. 经验总结
- 内容：包括成功的经验和失败的教训
- 要求：实事求是，避免流于形式或虚假夸大

## 四、编号格式

```
×公（ ）破字〔 〕×号
```

## 五、与立案报告的关系

| 阶段 | 文书 | 内容特点 |
|------|------|----------|
| 侦查起点 | 立案报告 | 反映初步认定的犯罪事实 |
| 侦查终点 | 破案报告 | 反映侦查终结后的全部情况，证据确凿充分 |

## 六、常用法律依据条款

- 《刑事诉讼法》第109条：侦查终结条件
- 《刑事诉讼法》第112条：破案条件
- 《刑事诉讼法》第162条：侦查终结后的处理
