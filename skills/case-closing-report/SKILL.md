# 破案报告生成器

根据《公安机关法律文书写作》规范，生成标准化的《呈请破案报告书》。

## 工作流程

### 第一步：收集案件信息

收集以下核心要素：

**犯罪嫌疑人基本情况**
- 姓名、性别、出生日期、出生地
- 身份证号码、民族、文化程度
- 职业、工作单位及职务
- 政治面貌、居住地
- 采取强制措施情况

**案件基本情况**
- 案件发生/发现的时间、地点
- 案件性质、涉案金额
- 被害人信息

**侦查过程**
- 立案时间、立案依据
- 侦查方案及执行情况
- 关键证据获取过程
- 抓捕经过
- 预审情况

### 第二步：生成报告结构

按以下五部分生成完整报告：

```
1. 犯罪嫌疑人的基本情况
2. 呈请事项
3. 事实依据（案件事实+证据分析）
4. 法律依据（引用具体法律条款）
5. 结语和落款
```

### 第三步：内容编写规范

**案情概况**：概要叙述案件发生、发现的时间地点，现场勘查情况

**案情分析**：结合证据材料进行深度分析，得出准确判断

**侦查方案**：回顾原订计划及修改补充情况

**破案经过**：按时间顺序叙述执行方案、查明抓捕、预审结果

**经验总结**：包括成功经验和教训

## 输出格式

生成完整的呈请破案报告书，包含：
- 文号（如：×公（刑）破字〔2024〕×号）
- 标准五部分结构
- 承办人签名、日期、领导审批栏
