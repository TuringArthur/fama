# 刑事上诉状生成技能

## 核心定义

刑事上诉状是刑事诉讼被告人、自诉人及其法定代理人，不服地方各级人民法院第一审未生效的刑事判决或裁定，在法定期限内向上一级人民法院提出上诉时所制作的法律文书。

## 适用场景

- 被告人不服一审有罪判决
- 被告人认为量刑过重
- 被告人认为事实认定有误
- 被告人认为适用法律错误
- 被告人认为程序违法
- 自诉人、自诉被告人上诉

## 文档结构

### 1. 首部

#### 标题
- 文书顶端居中书写"刑事上诉状"

#### 当事人信息（按顺序）

**自诉案件**：
```
上诉人（原审自诉人/原审被告人）：[姓名]，[性别]，[出生日期]出生，[民族]族，[出生地]人，[职业]，[工作单位及职务]，[住所]

被上诉人（原审被告人/原审自诉人）：[同上格式]

[诉讼代理人/辩护人]：[姓名]，[性别]，[职业]，[工作单位及职务]（如是律师）
```

**公诉案件**：
```
上诉人（原审被告人）：[姓名]，[性别]，[出生日期]出生，[民族]族，[出生地]人，[职业]，[工作单位及职务]，[住所]

辩护人：[同上格式]
```

**注意**：
- 公诉案件中没有被上诉人
- 不能把人民检察院列为被上诉人

### 2. 正文

#### （1）上诉事由
```
上诉人因[罪名]一案，于[年]年[月]月[日]日收到[法院名称]人民法院[年]年[月]月[日]日[案号]刑事[判决/裁定]书，现因不服该[判决/裁定]提出上诉。
```

#### （2）上诉请求
明确具体地写出：
- 对原审判决是全部不服还是部分不服
- 请求撤销原判、全部改判还是部分改判
- 请求从轻、减轻、免除处罚或宣告无罪

#### （3）上诉理由
从以下四个方面展开说理：

**事实认定方面**：
- 原判决认定的事实是否清楚
- 证据是否确实、充分
- 是否有遗漏重要事实

**适用法律方面**：
- 适用法律是否正确
- 罪名认定是否准确
- 量刑情节是否遗漏或错误认定

**量刑方面**：
- 量刑是否适当
- 是否存在法定/酌定从轻、减轻情节
- 是否存在法定/酌定从重情节未被考虑

**诉讼程序方面**：
- 审判程序是否合法
- 是否存在应当回避的情形
- 是否保障了被告人的诉讼权利

**总结格式**：
```
综上所述，原判决[概括错误之处]……充分说明原判决是错误的。为此，特向你院上诉，请求撤销原判，[具体改判请求]。
```

### 3. 尾部

#### 致送机关
```
此致
[第二审人民法院名称]人民法院
```

#### 附项
```
附：1. 上诉状副本[X]份
    2. 证据材料[X]份
```

#### 落款
```
上诉人：[签名或盖章]
[年]年[月]月[日]日
```

## 案件类型判断

| 案件类型 | 上诉人 | 被上诉人 | 格式要求 |
|---------|--------|---------|---------|
| 公诉案件（被告人上诉） | 原审被告人 | 无 | 辩护人信息 |
| 自诉案件（自诉人上诉） | 原审自诉人 | 原审被告人 | 诉讼代理人 |
| 自诉案件（被告人上诉） | 原审被告人 | 原审自诉人 | 辩护人 |

## 写作要求

1. **上诉请求**：明确具体，具有针对性
2. **上诉理由**：简明扼要、全面完整
3. **论证逻辑**：事实根据与法律根据相结合
4. **语言风格**：专业、规范、逻辑严密

## 输出格式

生成完整的刑事上诉状文档，包含：
- 规范的文书格式
- 完整的三部分结构
- 专业的法律术语
- 逻辑清晰的说理部分
