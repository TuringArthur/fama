# 刑事上诉状详细撰写指南

## 一、文书性质与功能

### 法律定位
刑事上诉状是刑事诉讼被告人、自诉人及其法定代理人，不服地方各级人民法院第一审未生效的刑事判决或裁定，在法定期限内依照法定程序向上一级人民法院提出上诉，请求变更或撤销原判决、裁定或者发回重审时所制作的法律文书。

### 核心功能
1. **权利救济**：上诉人维护自己合法权益的重要手段
2. **程序启动**：启动第二审程序的法定根据
3. **司法纠错**：保障人民法院及时发现和纠正错误的判决和裁定
4. **人权保障**：避免冤、假、错案的发生，确保司法公正

### 适用范围限制
- 只能针对地方各级人民法院的第一审刑事判决或裁定
- 不得对最高人民法院的判决或裁定提出上诉（立即生效）
- 不得对第二审终审判决提出上诉（已生效）

## 二、格式规范详解

### 首部格式

#### 标题书写规范
- 位于文书顶端
- 使用"刑事上诉状"四字
- 居中排列
- 不加"关于"等修饰词
- 不加书名号

#### 当事人基本情况

**自然人信息项**：
| 信息项 | 说明 | 示例 |
|-------|------|------|
| 姓名 | 户籍登记姓名 | 张三 |
| 性别 | 男/女 | 男 |
| 出生日期 | 具体到年月日 | 1980年1月1日 |
| 民族 | 具体民族 | 汉族 |
| 出生地 | 省/市/县 | 北京市朝阳区 |
| 职业 | 职业类别 | 农民/工人/教师/无业 |
| 工作单位及职务 | 单位名称及担任职务 | 北京市某某公司职员 |
| 住所 | 户籍所在地或经常居住地 | 北京市海淀区某某路某某号 |

**诉讼代理人/辩护人**：

若是律师，应写明：
- 姓名
- 工作单位（律师事务所名称）
- 职务（律师）

若是普通公民，应写明：
- 姓名
- 性别
- 职业
- 工作单位及职务
- 住所

### 正文章节格式

#### 上诉事由表达模板
```
上诉人因[罪名]一案，于[年]年[月]月[日]日收到[法院级别+法院名称]人民法院[年]年[月]月[日]日[案号]刑事[判决/裁定]书，现因不服该[判决/裁定]提出上诉。
```

**示例**：
```
上诉人因盗窃罪一案，于2024年3月15日收到北京市第一中级人民法院2024年3月10日（2024）京01刑初字第58号刑事判决书，现因不服该判决提出上诉。
```

#### 上诉请求表达规范

**请求类型**：
1. **撤销原判**：请求撤销一审全部判决
2. **改判无罪**：请求宣告被告人无罪
3. **从轻处罚**：请求在法定刑范围内从轻判处
4. **减轻处罚**：请求在法定刑以下判处
5. **免除处罚**：请求免除刑事处罚
6. **发回重审**：请求撤销原判、发回原审法院重新审理
7. **部分改判**：对部分判项不服，请求变更

**表述要求**：
- 明确是对全部还是部分判决不服
- 写明具体的改判请求
- 使用规范性法律术语

#### 上诉理由论证框架

**事实认定维度**：
- 原判决认定的犯罪事实是否清楚
- 认定事实的证据是否确实、充分
- 是否存在遗漏或错误认定的事实
- 证据之间是否存在矛盾
- 证据的收集程序是否合法

**法律适用维度**：
- 罪名认定是否正确
- 适用法律条文是否准确
- 量刑情节的认定是否正确
- 是否遗漏从轻、减轻、免除处罚情节
- 是否错误认定从重、加重情节

**量刑适当维度**：
- 量刑是否在法定刑幅度内
- 量刑是否符合罪责刑相适应原则
- 是否充分考虑从轻、从重情节
- 是否存在量刑畸重或畸轻情况

**程序合法维度**：
- 管辖是否符合规定
- 审判组织是否合法
- 回避制度是否执行
- 公开审判是否落实
- 辩护权是否得到保障
- 质证程序是否合法
- 诉讼权利告知是否完整

#### 总结段落格式
```
综上所述，原判决[具体错误]……[具体错误]……充分说明原判决[认定事实不清/适用法律错误/量刑不当/程序违法]。为此，特向你院上诉，请求撤销原判，[具体改判请求，如：宣告上诉人无罪/对上诉人从轻处罚/发回原审法院重新审理]。
```

### 尾部格式

#### 致送机关
```
此致
[上诉法院全称]人民法院
```

**注意**：应写第二审人民法院全称，如"浙江省高级人民法院"

#### 附项格式
```
附：1. 上诉状副本[X]份
    2. 证据材料清单：
       （1）[证据名称]，[证据来源]，[证明内容]
       （2）……
```

#### 落款格式
```
上诉人：[签名]
[签名]
[年]年[月]月[日]日
```

**注意**：
- 上诉人是法人或其他组织的，加盖公章
- 上诉人是多人的，分别签名或盖章
- 日期使用汉字数字

## 三、特殊案件类型处理

### 公诉案件格式
```
刑事上诉状

上诉人（原审被告人）：[基本信息]

辩护人：[律师或公民信息]

上诉人因[罪名]一案，……
```

**关键点**：
- 没有被上诉人
- 可以委托辩护人
- 检察院不出庭应诉（由一审法院移送案卷）

### 自诉案件格式
```
刑事上诉状

上诉人（原审自诉人）：[基本信息]

被上诉人（原审被告人）：[基本信息]

诉讼代理人：[律师或公民信息]

上诉人因[罪名]一案，……
```

**或（被告人上诉）**：
```
刑事上诉状

上诉人（原审被告人）：[基本信息]

被上诉人（原审自诉人）：[基本信息]

辩护人：[律师或公民信息]

上诉人因[罪名]一案，……
```

### 单位犯罪案件格式
```
刑事上诉状

上诉单位：[单位名称]
法定代表人：[姓名]，[职务]

辩护人：[律师信息]

上诉单位因[罪名]一案，……
```

## 五、写作注意事项

### 形式要求
1. 使用标准A4纸
2. 字迹工整清晰
3. 使用黑色钢笔或签字笔书写
4. 打印稿双面打印

### 内容要求
1. **针对性**：针对一审裁判的具体错误
2. **明确性**：上诉请求具体明确
3. **充分性**：理由论述充分有力
4. **逻辑性**：论证层次清晰
5. **规范性**：使用法律专业术语

### 提交要求
1. 上诉期限：判决送达后10日内（裁定5日内）
2. 提交途径：通过原审法院或直接向上一级法院
3. 副本份数：根据被上诉人人数确定

## 六、常见错误与纠正

### 常见格式错误
| 错误类型 | 正确做法 |
|---------|---------|
| 标题写"刑事上诉书" | 应写"刑事上诉状" |
| 公诉案件写了被上诉人 | 公诉案件没有被上诉人 |
| 把检察院列为被上诉人 | 检察院不能列为被上诉人 |
| 致送机关写原审法院 | 应写上一级法院 |

### 常见内容错误
| 错误类型 | 正确做法 |
|---------|---------|
| 上诉请求模糊不清 | 明确具体写出改判请求 |
| 理由泛泛而谈 | 针对性指出具体错误 |
| 事实理由混淆 | 分清事实根据与法律根据 |
| 遗漏重要上诉理由 | 全面审查事实、法律、量刑、程序 |

## 七、文书实例

### 完整实例（公诉案件）

```
刑事上诉状

上诉人（原审被告人）：李四，男，1975年6月15日出生，汉族，河南省郑州市人，原系郑州市某某公司经理，住郑州市金水区某某路某某号。

辩护人：王律师，河南某某律师事务所律师。

上诉人因故意伤害罪一案，于2024年5月20日收到郑州市中级人民法院2024年5月15日（2024）郑01刑初字第89号刑事判决书，现因不服该判决提出上诉。

上诉请求：
一、撤销郑州市中级人民法院（2024）郑01刑初字第89号刑事判决；
二、改判上诉人缓刑。

上诉理由：

一、原判决认定事实不清
原判决认定上诉人"持刀将被害人砍伤"与事实不符。实际情况是，上诉人在遭受被害人持砖块追打的情况下，为制止不法侵害而采取的防卫行为，当时现场有多名证人可以证实。原判决未充分考虑案发时的客观环境和双方的力量对比，错误认定上诉人"持刀砍伤"被害人。

二、原判决量刑过重
上诉人具有以下从轻情节：
1. 防卫过当：上诉人的行为属于防卫过当，依法应当减轻或免除处罚；
2. 认罪悔罪：上诉人归案后如实供述犯罪事实，认罪态度良好；
3. 积极赔偿：上诉人已与被害人达成和解协议，赔偿被害人经济损失8万元；
4. 被害人过错：被害人对案件发生存在重大过错；
5. 初犯偶犯：上诉人此前无任何违法犯罪记录。

三、原判决量刑不符合罪责刑相适应原则
综合本案事实、情节和社会危害程度，对上诉人判处有期徒刑三年实刑，明显过重，与上诉人的犯罪行为和主观恶性不相适应，不利于上诉人的教育改造。

综上所述，原判决认定事实不清，未充分考虑防卫过当情节和被害人过错，对上诉人量刑过重，不符合罪责刑相适应原则，充分说明原判决是错误的。为此，特向你院上诉，请求撤销原判，对上诉人改判缓刑。

此致
河南省高级人民法院

附：1. 上诉状副本3份
    2. 证据材料：
       （1）现场目击证人证言5份
       （2）法医鉴定意见书1份
       （3）和解协议及赔偿凭证各1份
       （4）上诉人无犯罪记录证明1份

上诉人：李四
2024年5月25日
```
