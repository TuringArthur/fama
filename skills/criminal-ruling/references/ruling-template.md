## 七、不同裁定类型的适用情形

| 裁定类型 | 适用情形 | 法律依据 |
|---------|---------|---------|
| 维持原判 | 原判事实清楚、证据充分、定罪准确、量刑适当 | 刑诉法第236条第1款第(一)项 |
| 撤销原判发回重审 | 原判事实不清、证据不足或程序违法 | 刑诉法第236条第1款第(三)项 |
| 撤销原判改判 | 原判认定事实错误或适用法律错误 | 刑诉法第236条第1款第(二)项 |
| 准许撤诉 | 检察院申请撤诉且符合条件 | 刑诉法第171条 |
| 终止审理 | 被告人死亡、刑诉法第16条规定情形 | 刑诉法第16条 |
| 管辖权异议 | 当事人对管辖权提出异议 | 刑诉法第25条 |

## 八、裁定书与判决书的区别

| 区别 | 裁定书 | 判决书 |
|-----|-------|-------|
| 解决问题 | 程序性问题 | 实体性问题 |
| 适用阶段 | 贯穿整个诉讼过程 | 只在审判结束时适用 |
| 形式 | 书面或口头 | 必须书面 |
| 救济途径 | 可复议或上诉 | 上诉 |
| 数量 | 可多个 | 一个案件一份 |
| 内容 | 侧重程序事项 | 侧重定罪量刑 |

## 九、裁定书的效力

### 生效时间
- 一审裁定书：送达后超过上诉期限未上诉即生效
- 二审裁定书：送达后即生效（终审裁定）

### 效力特征
- 程序性效力：具有约束力
- 执行效力：部分裁定具有执行力
- 既判力：对同一事项不得再行起诉
