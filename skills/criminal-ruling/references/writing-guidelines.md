# 刑事裁定书撰写规范指南

## 一、裁定书的法律性质与特点

### 1. 裁定书的定义

刑事裁定书是人民法院在刑事诉讼过程中，就程序性问题或部分实体性问题所作出的书面决定。它与刑事判决书一起构成人民法院刑事裁判文书的主要形式。

### 2. 裁定书的特点

| 特点 | 说明 |
|-----|------|
| 程序性 | 主要解决程序问题 |
| 多样性 | 适用情形多种多样 |
| 即时性 | 可在诉讼过程中随时作出 |
| 灵活性 | 形式相对灵活 |
| 可救济性 | 部分裁定可上诉或复议 |

---

## 二、裁定书与判决书的区别

### 1. 解决的问题不同

| 区别 | 裁定书 | 判决书 |
|-----|-------|-------|
| 解决问题 | 程序性问题 | 实体性问题 |
| 典型内容 | 驳回起诉、维持原判 | 定罪量刑 |
| 适用范围 | 广泛 | 特定 |

### 2. 制作时间不同

| 区别 | 裁定书 | 判决书 |
|-----|-------|-------|
| 制作时间 | 诉讼过程中随时 | 审判结束时 |
| 数量 | 一个案件可多个 | 一个案件一份 |
| 阶段性 | 各阶段均可 | 仅审判阶段 |

### 3. 表现形式不同

| 区别 | 裁定书 | 判决书 |
|-----|-------|-------|
| 形式 | 书面或口头 | 必须书面 |
| 内容 | 侧重程序事项 | 侧重定罪量刑 |
| 结构 | 相对简化 | 结构完整 |

## 三、二审刑事裁定书撰写要点

### 1. 首部撰写

**法院名称**
- 使用规范的法院全称
- 二审法院为省级或市级中级人民法院

**案号**
- 格式：(年份)+刑终字+编号+号
- 与一审案号区分

**诉讼参与人**
- 原公诉机关
- 上诉人（原审被告人）
- 辩护人
- 其他诉讼参与人

**表述示例**：
```
原公诉机关：×××人民检察院

上诉人（原审被告人）：王某某，男，1980年5月15日出生，汉族，初中文化程度，农民，住××省××县××镇××村。因涉嫌犯××罪，于××××年××月××日被刑事拘留，同年××月××日被逮捕。现羁押于×××看守所。

辩护人：李某某，×××律师事务所律师。
```

### 2. 审理经过撰写

**原审情况**
- 原审法院名称
- 原审判决时间
- 原审案号
- 原判认定的罪名和刑罚

**上诉情况**
- 上诉人提出上诉的时间
- 上诉的主要理由
- 辩护人的意见

**审理方式**
- 是否公开审理
- 合议庭组成
- 是否开庭审理

**表述示例**：
```
×××人民法院审理被告人王某某犯盗窃罪一案，于2024年5月20日作出(2024)×刑初字第128号刑事判决，认定被告人王某某犯盗窃罪，判处有期徒刑四年。被告人王某某不服，以原判量刑过重为由提出上诉。本院依法组成合议庭进行了审理。现已审理终结。
```

### 3. 事实认定撰写

**二审审理查明的事实**
- 与原审认定一致的事实
- 或二审查明的新事实
- 或纠正原审认定的事实

**证据列举**
- 原审证据的评价
- 新证据（如有）
- 证据分析

**表述示例**：
```
经审理查明，2023年8月15日14时许，上诉人王某某在本市××区××商场内，趁被害人李某某不备，将其放在柜台上的手提包盗走，内有现金人民币5000元及手机一部（价值人民币3000元）。案发后，上诉人王某某被当场抓获。

上述事实，有原审法院经庭审质证的下列证据予以证实：
1. 被害人李某某的陈述，证实其手提包被盗的经过；
2. 证人张某某（商场保安）的证言，证实抓获上诉人的经过；
3. 上诉人王某某的供述和辩解，供认了盗窃事实；
4. 价格鉴定意见，证实被盗物品价值；
5. 现场监控视频，证实盗窃经过。
```

### 4. 本院认为撰写

**对上诉理由的分析**
- 逐一分析上诉理由是否成立
- 对合理的上诉理由应予采纳
- 对不合理的上诉理由应予驳回

**对原判的评价**
- 事实认定是否清楚
- 证据是否充分
- 法律适用是否正确
- 量刑是否适当
- 程序是否合法

**裁定理由**
- 维持原判的理由
- 或撤销原判的理由
- 或改判的理由

**表述示例**：
```
本院认为，上诉人王某某以非法占有为目的，秘密窃取他人财物，数额较大，其行为已构成盗窃罪，依法应予惩处。原审判决认定事实清楚，证据确实充分，定罪准确。关于上诉人王某某所提原判量刑过重的上诉理由，经查，上诉人王某某盗窃数额共计人民币8000元，属于数额较大，原审法院综合考虑其犯罪事实、情节及认罪态度，判处有期徒刑四年，量刑适当。上诉理由不能成立，本院不予采纳。
```

### 5. 裁定结果撰写

**维持原判格式**：
```
依照《中华人民共和国刑事诉讼法》第二百三十六条第一款第（一）项之规定，裁定如下：

一、驳回上诉，维持×××人民法院(××××)×刑初字第××号刑事判决；
二、本裁定为终审裁定。
```

**撤销原判发回重审格式**：
```
依照《中华人民共和国刑事诉讼法》第二百三十六条第一款第（三）项之规定，裁定如下：

一、撤销×××人民法院(××××)×刑初字第××号刑事判决；
二、发回×××人民法院重新审判。
```

**撤销原判改判格式**：
```
依照《中华人民共和国刑事诉讼法》第二百三十六条第一款第（二）项、《中华人民共和国刑法》第×××条之规定，裁定如下：

一、撤销×××人民法院(××××)×刑初字第××号刑事判决；
二、上诉人（原审被告人）×××犯××罪，判处……（新的刑罚）。
```

### 6. 尾部撰写

**合议庭成员署名**
```
审判长  ×××
审判员  ×××
审判员  ×××
```

**日期**
- 使用汉字数字
- 格式：二〇××年××月××日

**书记员署名**
```
书记员  ×××
```

## 四、一审刑事裁定书撰写要点

### 1. 管辖权异议裁定

**首部**
```
×××人民法院
刑事裁定书

(××××)×刑初字第××号

申请人：×××，……
```

**正文**
```
本院收到申请人×××提出的管辖权异议申请后，依法进行了审查。现审查终结。

申请人×××认为：……（申请理由）。

本院经审查认为：……（审查意见）。
```

**裁定结果**
```
依照《中华人民共和国刑事诉讼法》第二十五条之规定，裁定如下：

驳回申请人×××对本案管辖权提出的异议。
```

### 2. 准许撤诉裁定

**正文**
```
本院在审理×××人民检察院提起公诉的被告人×××犯××罪一案过程中，×××人民检察院于××××年××月××日以×××号撤诉决定书，要求撤回起诉。

本院经审查认为：……（审查意见）。
```

**裁定结果**
```
依照《中华人民共和国刑事诉讼法》第十六条第（二）项、第一百七十一条之规定，裁定如下：

准许×××人民检察院撤回对被告人×××的起诉。
```

### 3. 终止审理裁定

**正文**
```
本院在审理被告人×××犯××罪一案过程中，……（需要终止审理的情形）。

本院经审查认为：……（审查意见）。
```

**裁定结果**
```
依照《中华人民共和国刑事诉讼法》第十六条第（五）项之规定，裁定如下：

本案终止审理。
```

## 五、不同类型裁定书的撰写要点

### 1. 维持原判裁定

**重点内容**：
- 确认原审认定的事实
- 评价原审证据是否充分
- 评价原审法律适用是否正确
- 评价原判量刑是否适当
- 逐一分析上诉理由是否成立

**常见表述**：
- "原审判决认定事实清楚，证据确实充分"
- "定罪准确，量刑适当"
- "审判程序合法"
- "上诉理由不能成立"

### 2. 撤销原判发回重审裁定

**重点内容**：
- 指出原判存在的问题
- 详细说明撤销原判的理由
- 如系程序违法，应明确指出违反哪条规定

**常见表述**：
- "原审判决认定事实不清，证据不足"
- "原审法院的审理违反法定程序"
- "可能影响公正审判"

### 3. 撤销原判改判裁定

**重点内容**：
- 纠正原审认定的事实错误
- 纠正原审适用法律的错误
- 纠正量刑不当
- 作出新的判决

**常见表述**：
- "原审判决认定事实错误"
- "原审判决适用法律不当"
- "应予纠正"

### 4. 准许撤诉裁定

**重点内容**：
- 列明撤诉申请
- 说明审查意见
- 裁定准许撤诉

**常见表述**：
- "申请撤回起诉的理由符合法律规定"
- "准许撤回起诉"

## 六、裁定书的格式规范

### 1. 标题格式

```
                              ××××人民法院
                                刑事裁定书

                              (××××)×刑终/初字第××号
```

### 2. 当事人称谓

| 情形 | 称谓 |
|-----|------|
| 二审上诉人 | 上诉人（原审被告人） |
| 一审被告人 | 被告人 |
| 申请人 | 申请人 |
| 公诉机关 | 原公诉机关/公诉机关 |

### 3. 日期格式

- 正确：二〇二四年五月十五日
- 错误：2024年5月15日

### 4. 签章格式

```
                                 审判长  ×××
                                 审判员  ×××
                                 审判员  ×××

                              ××××年××月××日

                                 书记员  ×××
```

## 七、常见错误及纠正

### 1. 裁定理由不充分

**问题表现**：
- 裁定理由过于简单
- 未逐一回应上诉理由
- 说理不透彻

**纠正方法**：
- 详细阐述裁定理由
- 逐一回应上诉理由
- 充分论证

### 2. 法律引用错误

**问题表现**：
- 引用条款不正确
- 条款编号错误
- 款项引用错误

**纠正方法**：
- 核实法律条款
- 确保引用准确
- 核对条款编号

### 3. 事实认定错误

**问题表现**：
- 事实认定与证据不符
- 遗漏重要事实
- 认定事实不清

**纠正方法**：
- 严格按照证据认定事实
- 确保事实清楚
- 证据与事实对应

### 4. 格式不规范

**问题表现**：
- 标题格式不规范
- 日期格式错误
- 署名不规范

**纠正方法**：
- 严格按照格式要求
- 使用规范用语
- 注意细节

### 5. 遗漏必要要素

**问题表现**：
- 缺少必要的文书要素
- 内容不完整

**纠正方法**：
- 确保文书完整
- 包含所有必要要素
- 内容充实

## 九、特殊情况的处理

### 1. 多人犯罪案件

- 分别列明各上诉人的上诉理由
- 逐一分析各上诉理由是否成立
- 分别作出裁定

### 2. 数罪并罚案件

- 对各罪分别评价
- 对数罪并罚原则的评价
- 对量刑综合评价

### 3. 附带民事诉讼案件

- 刑事部分与民事部分分别处理
- 裁定结果应明确
- 注意相互关联

### 4. 程序违法案件

- 明确指出违法情形
- 说明违反哪条规定
- 说明为何影响公正审判
