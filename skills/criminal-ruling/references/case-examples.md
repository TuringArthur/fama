# 刑事裁定书实例分析

## 实例一：二审维持原判裁定（寻衅滋事罪）

---

```
                              ×××省××市中级人民法院
                                刑事裁定书

                            (2024)×刑终字第86号

原公诉机关：××县人民检察院

上诉人（原审被告人）：王丹阳，男，1976年2月10日出生，汉族，农民，住××省××县××村。因涉嫌犯寻衅滋事罪，于2024年4月12日被刑事拘留，同年4月20日被逮捕。现羁押于××县看守所。

辩护人：杨佳明，××省××县××律师事务所律师。

××县人民法院审理被告人王丹阳犯寻衅滋事罪一案，于2024年8月10日作出(2024)×刑初字第387号刑事判决，认定被告人王丹阳犯寻衅滋事罪，判处有期徒刑二年。被告人王丹阳不服，以没有殴打被害人为由提出上诉。本院依法组成合议庭进行了审理。现已审理终结。

经审理查明，2020年3月13日，上诉人王丹阳与王晓英协议离婚。2020年冬至2021年3月间，上诉人王丹阳因对离婚不满，多次深夜到××县××村王明、王静、王心谛（均系王晓英家属）家中滋扰，砸坏门窗，并打伤王俊（经鉴定为轻微伤）。

上述事实，有原审法院经庭审质证的下列证据予以证实：
1. 被告人王丹阳的供述和辩解，供称未与被害人发生冲突；
2. 被害人王明的陈述，证实多次被砸玻璃及对骂经过；
3. 被害人王心谛的陈述，证实多次深夜玻璃被砸及受惊吓情况；
4. 被害人王静的陈述，证实滋扰次数及弟弟被打；
5. 被害人张保、王俊的陈述，证实内容与上述基本一致；
6. 证人程英（邻居）证言，证实听到砸玻璃声及吵闹声；
7. 证人张荣（邻居）证言，证实曾见房顶有人并捡到木棍；
8. 证人龙英（邻居）证言，证实长期存在深夜滋扰情况；
9. 证人路文、张保证言，证实内容基本一致；
10. 物证照片、物品清单、扣押清单；
11. 被害人王俊伤情鉴定结论及伤情照片；
12. 王丹阳与王晓英离婚手续证明；
13. 公安机关抓获证明；
14. 公安机关接处警记录；
15. 被告人王丹阳户籍证明。

原审判决：依照《中华人民共和国刑法》第二百九十三条之规定，认定被告人王丹阳犯寻衅滋事罪，判处有期徒刑二年。

上诉理由：上诉人王丹阳上诉称，其没有殴打被害人，原判事实不清。

本院认为，上诉人王丹阳因生活琐事，多次深夜到他人住宅滋扰，辱骂、恐吓他人，任意毁损公私财物，情节恶劣，其行为已构成寻衅滋事罪，依法应予惩处。原审判决认定事实清楚，证据确实充分，定罪准确，量刑适当，审判程序合法。上诉人王丹阳所提其没有殴打被害人的上诉理由，与查明的事实不符，不能成立。原审法院鉴于本案系因民间纠纷引发，已对其酌情从轻处罚，量刑并无不当。辩护人的辩护意见本院不予采纳。

综上所述，依照《中华人民共和国刑事诉讼法》第二百三十六条第一款第（一）项之规定，裁定如下：

                                裁定结果

一、驳回上诉，维持××县人民法院(2024)×刑初字第387号刑事判决；
二、本裁定为终审裁定。

                                 审判长  侯隶书
                                 审判员  杨秀丽
                                 审判员  李彩霞

                              二〇二四年十月十二日

                                 书记员  石中兴
```

## 实例二：二审撤销原判发回重审裁定

---

```
                              山东省高级人民法院
                                刑事裁定书

                            (2024)鲁刑终字第215号

原公诉机关：山东省人民检察院

上诉人（原审被告人）：张某某，男，1985年6月15日出生，汉族，大学文化程度，原系××公司总经理，住济南市××区××路××号。因涉嫌犯合同诈骗罪，于2023年9月5日被刑事拘留，同年9月18日被逮捕。现羁押于山东省看守所。

辩护人：李某某，山东××律师事务所律师。

济南市中级人民法院审理被告人张某某犯合同诈骗罪一案，于2024年5月20日作出(2024)济刑初字第128号刑事判决，认定被告人张某某犯合同诈骗罪，判处有期徒刑十五年。被告人张某某不服，提出上诉。本院依法组成合议庭进行了审理。现已审理终结。

经审理查明，2019年3月至2021年8月期间，上诉人张某某在担任××公司总经理期间，以公司名义与多家被害单位签订购销合同，收取预付款后未按合同约定供货，涉及金额特别巨大。

本院在审理过程中发现，原审判决认定上诉人张某某诈骗金额的事实不清，证据不足。具体表现在：
1. 原审判决认定的诈骗金额中，部分项目存在真实的交易背景，是否属于诈骗事实存疑；
2. 部分证人证言之间存在矛盾，未能排除合理怀疑；
3. 对上诉人张某某主观故意的认定依据不足。

本院认为，原审判决认定事实不清，证据不足。依照《中华人民共和国刑事诉讼法》第二百三十六条第一款第（三）项之规定，裁定如下：

                                裁定结果

一、撤销济南市中级人民法院(2024)济刑初字第128号刑事判决；
二、发回济南市中级人民法院重新审判。

                                 审判长  王某某
                                 审判员  陈某某
                                 审判员  刘某某

                              二〇二四年八月二十五日

                                 书记员  赵某某
```

## 实例三：二审撤销原判改判裁定

---

```
                              广东省高级人民法院
                                刑事裁定书

                            (2024)粤刑终字第342号

原公诉机关：广东省人民检察院

上诉人（原审被告人）：李某某，男，1978年3月22日出生，汉族，初中文化程度，农民，住广东省××县××镇××村。因涉嫌犯故意伤害罪，于2023年11月10日被刑事拘留，同年11月20日被逮捕。现羁押于广东省看守所。

辩护人：周某某，广东××律师事务所律师。

广州市中级人民法院审理被告人李某某犯故意伤害罪一案，于2024年3月15日作出(2024)穗刑初字第201号刑事判决，认定被告人李某某犯故意伤害罪，判处有期徒刑八年。被告人李某某不服，提出上诉。本院依法组成合议庭进行了审理。现已审理终结。

经审理查明，2023年8月5日20时许，上诉人李某某在广东省××县因邻里纠纷与被害人王某某发生争执并相互打斗。期间，李某某用拳头击打王某某面部，致王某某鼻骨骨折、左眼眶壁骨折（经鉴定为轻伤一级）。

另查明，案发后，上诉人李某某明知他人报警而在现场等待，抓捕时无抗拒行为，到案后如实供述了犯罪事实。案发后，上诉人李某某与被害人王某某达成和解协议，赔偿被害人经济损失人民币8万元，被害人对上诉人的行为表示谅解。

原审判决：依照《中华人民共和国刑法》第二百三十四条第一款之规定，认定被告人李某某犯故意伤害罪，判处有期徒刑八年。

上诉理由：上诉人李某某上诉称，原判量刑过重，请求从轻处罚。辩护人提出，上诉人具有自首情节，已赔偿被害人并取得谅解，原判量刑不当。

本院认为，上诉人李某某故意伤害他人身体，致人轻伤一级，其行为已构成故意伤害罪，依法应予惩处。原审判决认定事实清楚，证据确实充分，定罪准确。但考虑到：1. 上诉人李某某明知他人报警而在现场等待，抓捕时无抗拒行为，到案后如实供述犯罪事实，依法构成自首，可以从轻或减轻处罚；2. 上诉人已赔偿被害人全部经济损失并取得谅解，可酌情从轻处罚；3. 本案系民间纠纷引发，被害人对矛盾的激化负有一定责任，可酌情对上诉人从轻处罚。原审判决未认定上诉人构成自首，量刑不当，应予纠正。

综上所述，依照《中华人民共和国刑事诉讼法》第二百三十六条第一款第（二）项、《中华人民共和国刑法》第二百三十四条第一款、第六十七条第一款、第七十二条第一款、第七十三条第二款、第三款之规定，裁定如下：

                                裁定结果

一、撤销广州市中级人民法院(2024)穗刑初字第201号刑事判决；
二、上诉人李某某犯故意伤害罪，判处有期徒刑三年，缓刑四年。

缓刑考验期限，从判决确定之日起计算，即自2024年9月10日起至2028年9月9日止。

                                 审判长  何某某
                                 审判员  吴某某
                                 审判员  郑某某

                              二〇二四年九月十日

                                 书记员  孙某某
```

## 实例四：一审管辖权异议裁定

---

```
                              北京市朝阳区人民法院
                                刑事裁定书

                            (2024)朝刑初字第456号

申请人：陈某某，男，1980年7月8日出生，汉族，大专文化程度，公司职员，住北京市海淀区××路××号。

本院收到申请人陈某某提出的管辖权异议申请后，依法进行了审查。现审查终结。

申请人陈某某认为：本案涉及的犯罪行为地和结果地均在北京市海淀区，且申请人住所地也在北京市海淀区，故本案应由北京市海淀区人民法院管辖，请求将本案移送北京市海淀区人民法院审理。

本院经审查认为：根据《中华人民共和国刑事诉讼法》第二十五条之规定，刑事案件由犯罪地的人民法院管辖。如果由被告人居住地的人民法院审判更为适宜的，可以由被告人居住地的人民法院管辖。本案中，虽然犯罪行为地涉及北京市海淀区，但根据本院调查，本案的主要犯罪事实发生地及部分被害人均在本院管辖范围内，本院依法对本案具有管辖权。申请人陈某某提出的管辖权异议不能成立。

依照《中华人民共和国刑事诉讼法》第二十五条、《最高人民法院关于适用〈中华人民共和国刑事诉讼法〉的解释》第三条之规定，裁定如下：

                                裁定结果

驳回申请人陈某某对本案管辖权提出的异议。

如不服本裁定，可以在收到裁定书后五日以内向北京市第三中级人民法院提出上诉。

                                 审判长  张某某
                                 审判员  李某某
                                 审判员  王某某

                              二〇二四年四月二十八日

                                 书记员  刘某某
```

准许上海市浦东新区人民检察院撤回对被告人赵某某的起诉。

## 实例分析要点

### 1. 维持原判裁定要点

| 要点 | 内容 | 说明 |
|-----|------|------|
| 事实认定 | 确认原审认定 | 一般不再重复 |
| 证据评价 | 评价是否充分 | 确认证据确实充分 |
| 法律适用 | 评价是否正确 | 确认定罪准确 |
| 量刑评价 | 评价是否适当 | 确认量刑恰当 |
| 上诉理由 | 逐一分析 | 说明采纳与否 |

### 2. 发回重审裁定要点

| 要点 | 内容 | 说明 |
|-----|------|------|
| 问题指出 | 明确指出问题 | 事实不清/程序违法 |
| 理由说明 | 详细说明理由 | 为何撤销原判 |
| 程序违法 | 如实指出 | 违反哪条规定 |

### 3. 改判裁定要点

| 要点 | 内容 | 说明 |
|-----|------|------|
| 纠正内容 | 明确纠正内容 | 事实/法律/量刑 |
| 新证据 | 如有新证据 | 应予列明 |
| 新判决 | 作出新判决 | 完整表述 |

### 4. 一审裁定要点

| 要点 | 内容 | 说明 |
|-----|------|------|
| 申请内容 | 明确申请内容 | 管辖异议等 |
| 审查意见 | 说明审查过程 | 程序性审查 |
| 裁定结果 | 明确裁定结果 | 驳回/准许 |
