# 刑事裁定书撰写技能

## 触发条件

当用户请求以下操作时激活本技能：
- 撰写刑事裁定书
- 生成法院裁定文书
- 创建二审刑事裁定书
- 制作裁定书模板
- 处理维持原判裁定
- 处理撤销原判裁定
- 撰写发回重审裁定

## 工作流程

### 1. 确定裁定书类型

根据案件情况确定裁定书的具体类型：

**二审裁定书类型：**
- 维持原判裁定
- 撤销原判发回重审裁定
- 撤销原判改判裁定
- 准许撤诉裁定
- 终止审理裁定

**一审裁定书类型：**
- 管辖权异议裁定
- 财产保全裁定
- 强制措施裁定
- 终止审理裁定

### 2. 收集案件基本信息

**案件信息：**
- 法院名称
- 案号（年份+刑终/刑初字+编号+号）
- 原公诉机关
- 上诉人（原审被告人）信息
- 辩护人信息

**上诉人信息：**
- 姓名
- 性别
- 出生年月日
- 民族
- 住址
- 职业

### 3. 撰写首部

包含以下要素：
- 法院名称
- 文书种类（刑事裁定书）
- 案号
- 诉讼参与人信息

### 4. 撰写案由与审理经过

**原审情况：**
- 原审法院名称
- 原审判决日期和案号
- 原判认定的罪名和刑罚

**上诉情况：**
- 上诉人提出上诉的时间和理由
- 辩护人的意见

### 5. 撰写二审审理查明

**事实认定：**
- 原审认定的事实
- 二审查明的事实（与原审一致或不同）
- 对上诉理由的分析

**证据列举：**
- 原审证据
- 新证据（如有）
- 证据分析

### 6. 撰写本院认为

**法律分析：**
- 对上诉理由逐一回应
- 对辩护意见逐一分析
- 论证裁定依据

**裁定理由：**
- 维持原判的理由
- 或撤销原判的理由
- 或其他裁定的理由

### 7. 撰写裁定结果

根据案件类型选择：

**维持原判：**
```
一、驳回上诉，维持×××人民法院(××××)×刑初字第××号刑事判决；
二、本裁定为终审裁定。
```

**撤销原判发回重审：**
```
一、撤销×××人民法院(××××)×刑初字第××号刑事判决；
二、发回×××人民法院重新审判。
```

**撤销原判改判：**
```
一、撤销×××人民法院(××××)×刑初字第××号刑事判决；
二、被告人XXX犯XX罪，判处……
（写明改判后的定罪量刑）
```

**准许撤诉：**
```
准许人民检察院撤回起诉。
```

### 8. 撰写尾部

包含以下要素：
- 合议庭成员署名（审判长、审判员）
- 日期（年 月 日）
- 书记员署名

## 法律依据

### 刑事诉讼法常用条款

**二审程序（第227-230条）：**
- 第227条：二审法院对上诉、抗诉案件的处理
- 第228条：发回重审的情形
- 第229条：二审裁定的期限
- 第230条：二审程序

**第一审程序（第185-197条）：**
- 第185条：裁定的适用范围
- 第186条：裁定的生效

**其他程序：**
- 审判监督程序
- 执行程序中的裁定
- 减刑假释裁定

## 注意事项

1. **裁定与判决的区别**
   - 裁定解决程序性问题
   - 判决解决实体性问题
   - 裁定可以针对诉讼中的各种程序问题

2. **二审裁定书特点**
   - 重点审查上诉理由
   - 审查原判事实认定和法律适用
   - 裁定结果应明确

3. **上诉理由审查**
   - 逐一分析上诉理由
   - 对合理的上诉理由应予采纳
   - 对不合理的上诉理由应予驳回

4. **证据审查**
   - 原审证据是否充分
   - 是否有新证据
   - 证据是否确实充分

5. **法律适用**
   - 引用的法律条款必须准确
   - 裁定依据必须充分

## 输出格式

生成的裁定书应包含完整的所有要素，格式规范，内容完整，可直接用于司法实践。
