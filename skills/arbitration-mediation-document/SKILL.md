# 仲裁调解书

## 一、仲裁调解书的概念

仲裁调解书是指仲裁机构通过调解方式，促使双方当事人自愿达成协议，将协议结果制作而成的法律文书。

### 核心要点

- **调解原则**：应在查明事实、分清是非的基础上，坚持自愿、合法原则
- **法律依据**：《仲裁法》第51条规定，仲裁庭在作出裁决前可以先行调解，当事人自愿调解的，仲裁庭应当调解
- **法律效力**：调解书与仲裁裁决书具有同等法律效力
- **生效条件**：必须双方自愿达成一致，且经过双方当事人签收后才发生法律效力。若有一方不签收，则调解书无效

---

## 二、仲裁调解书的结构

仲裁调解书由**首部**、**正文**、**尾部**和**附项**四部分组成。

### （一）首部

应写明以下几项：

1. **标题**：写明仲裁委员会的名称和文书种类，如"**××仲裁委员会仲裁调解书**"
2. **编号**：在标题下方注明"**（年度）×字第×号**"
3. **当事人的基本情况**：
   - 按仲裁申请人和被申请人的顺序书写
   - 一方或双方有法定代理人、法定代表人、代表人或委托代理人的，要分别写明
   - 委托代理人为律师的，写明其姓名和所在律师事务所及职务
   - 委托代理人为一般公民的，应写事项同当事人

### （二）正文

正文应包括**案件事实**、**协议内容**：

1. **事实**：应简要叙述争议发生的原因、经过、争执焦点、请求事项等内容。对于其中的仲裁请求，应当根据申请人的仲裁申请书以及仲裁请求书写清楚和完整

2. **协议内容**：写明双方当事人达成的协议的具体内容。调解书应当写明申请人与被申请人协议的结果。该部分是仲裁调解书的核心部分，其内容决定双方的权利、义务，既包括实体权利争议所达成的协议内容，也包括有关仲裁费用分担的内容

3. **仲裁庭审查意见**：仲裁庭应当对双方当事人达成的协议内容进行审查，确认其与事实相符、不违反法律规定、不损害他人合法权益

### （三）尾部

依次写明下述内容：

1. **生效时间**：根据《仲裁法》的规定，可表述为："本调解书与仲裁裁决书具有同等法律效力，自双方申请人签收之日起生效。"
2. **仲裁庭成员署名**
3. **制作调解书的日期和地点**
4. **书记员署名，加盖仲裁委员会印章**

### （四）附项

1. 写明调解书发生法律效力的时间。一般写成："本调解书经双方签收后，即发生法律效力。"
2. 注明该仲裁调解书的份数及双方当事人所持的份数

```
    住所：×××
    法定代表人：×××（职务）
    委托代理人：×××（职务）

    住所：×××
    法定代表人：×××（职务）
    委托代理人：×××（职务）

```

## 四、制作仲裁调解书应注意的问题

### 1. 内容要求
- 仲裁调解书的内容，应当是申请人双方达成协议的内容
- 写入调解书的内容必须是符合法律规定的

### 2. 表述要求
- 对于争议的事实可以简要概括
- 但对于仲裁请求和协议内容应当**具体、明确写明**
- 对于放弃仲裁请求的内容也应一并写明
- 协议的内容不止一项的，应分项列出

### 3. 可操作性要求
- 应当明确履行期限和方式，确保协议的"可操作性"
- 调解依据必须实事求是，既可以依法达成，也可以依照社会情理达成，但需结合实际情况说明依据

## 五、案例范例（工伤待遇争议）

```
××劳动人事争议仲裁委员会
                    仲裁调解书

××劳人仲案字（2017）第028号

申请人：×××，男，23岁，汉族，住×××
    委托代理人：×××，××律师事务所律师

被申请人：××工程有限公司
    法定代表人：×××，该公司经理
    委托代理人：×××，××律师事务所律师

（案由：工伤待遇争议）

本会受理的申请人×××与被申请人××工程有限公司工伤待遇争议一案，
经本会依法组成独任庭审理，现已审理终结。

经本会主持调解，双方当事人本着互谅互让、协商解决的精神，达成如下协议：

一、被申请人一次性支付申请人各项工伤待遇共计人民币柒万捌仟元整（￥78,000.00元）。

二、上述款项于2018年1月19日前支付完毕。

三、如被申请人逾期支付上述款项，应按中国人民银行同期贷款利率加倍支付利息。

四、本协议履行完毕之日，双方劳动关系及工伤保险关系即行终止。

五、申请人放弃其他仲裁请求，双方当事人就本案再无其他争议。

本调解书与仲裁裁决书具有同等法律效力，自双方当事人签收之日起生效。

                    仲裁员：×××
                    二〇一八年一月十九日
                    （××劳动人事争议仲裁委员会印）

书记员：×××
附：本调解书一式三份，双方当事人各执一份，本会存档一份。
```

## 六、常用法律依据

- 《中华人民共和国仲裁法》第51条
- 《中华人民共和国劳动争议调解仲裁法》
- 《劳动人事争议仲裁办案规则》
- 各地《工伤保险条例》实施办法
