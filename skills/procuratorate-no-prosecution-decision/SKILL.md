# 人民检察院不起诉决定书生成器

不起诉决定书是人民检察院对侦查机关移送起诉或自行侦查的案件进行审查后，依法作出不起诉决定的诉讼文书。

## 一、文书结构

不起诉决定书由**首部**、**正文**和**尾部**三部分组成，遵循"人—案—事—理—法—果"的逻辑顺序。

### 1.1 首部

| 要素 | 格式要求 |
|------|----------|
| 检察院名称 | 位于文书顶端居中，如"XX省XX市人民检察院" |
| 文书名称 | "不起诉决定书"，紧跟院名 |
| 案号 | 右下方，格式：`XX检XX刑不诉〔年份〕X号` |

**案号示例：**
```
京检刑不诉〔2023〕1号
X检刑不诉〔2024〕15号
```

### 1.2 正文（五要素）

#### 要素一：被不起诉人基本情况

| 信息项 | 填写说明 |
|--------|----------|
| 姓名 | 与身份证一致 |
| 性别 | 男/女 |
| 出生年月日 | 格式：XXXX年XX月XX日 |
| 身份证号码 | 18位身份证号 |
| 民族 | 如：汉族 |
| 文化程度 | 如：大学本科 |
| 职业/工作单位及职务 | 具体填写 |
| 户籍地 | 省市区县 |
| 住址 | 详细地址 |

**特殊情况：**
- 国家工作人员职务犯罪：需注明犯罪期间的职务
- 曾受刑事处罚或被采取强制措施：需注明
- 单位犯罪：写明单位名称和住所地

#### 要素二：辩护人基本情况

| 信息项 | 填写说明 |
|--------|----------|
| 辩护人姓名 | |
| 所属单位 | 律师事务所名称 |

#### 要素三：案由和案件来源

**标准表述：**
```
本案由XXX（侦查机关）侦查终结，以被不起诉人XXX涉嫌XXX罪，于XXXX年XX月XX日移送审查起诉。
```

**特殊情况表述：**
- **自侦案件：** 本院于XXXX年XX月XX日立案侦查
- **退回补充侦查：** 本案经退回补充侦查一次
- **其他检察院移送：** 本案由XXX检察院移送

**注意：** 不起诉决定书中**不得使用"犯罪事实"等提法**，应使用"经审查查明"。

#### 要素四：案件事实情况

根据不起诉类型，有不同的叙述要求：

| 不起诉类型 | 叙述重点 |
|------------|----------|
| 法定不起诉（第16条） | 重点叙述符合法定不追究情形的事实和证据 |
| 情节显著轻微 | 先概括移送起诉认定的事实，再叙述审查认定的事实，突出"情节显著轻微、危害不大"的特征 |
| 没有犯罪事实 | 重点叙述不存在犯罪事实，或犯罪事实并非被不起诉人所为的依据 |
| 证据不足不起诉 | 叙述补充侦查后证据仍不足的情况 |

#### 要素五：不起诉理由、法律依据和决定事项

**叙述式（逻辑推导型）：**
```
综上所述/根据以上事实，本院认为，[被不起诉人姓名]的行为……（阐述理由）。

根据《中华人民共和国刑事诉讼法》第XX条第X款之规定，决定对[被不起诉人姓名]不起诉。
```

**条文式（直接宣告型）：**
```
根据《中华人民共和国刑事诉讼法》第XX条第X款之规定，决定如下：
1. 对XXX不起诉；
2. 本决定自送达之日起生效。
```

### 1.3 尾部

| 要素 | 格式要求 |
|------|----------|
| 检察官署名 | 格式：检察官 XXX |
| 日期 | XXXX年XX月XX日（签发日期） |
| 院印 | 加盖人民检察院公章 |

## 二、法律依据速查

| 不起诉类型 | 法律依据 | 适用情形 |
|------------|----------|----------|
| 法定不起诉 | 《刑诉法》第16条 | 情节显著轻微、危害不大；已过追诉时效；经特赦令免除刑罚；犯罪嫌疑人死亡；其他法律规定免予追究 |
| 证据不足不起诉 | 《刑诉法》第175条第4款 | 经过二次补充侦查，证据仍不足 |
| 酌定不起诉 | 《刑诉法》第177条第2款 | 犯罪情节轻微，依照刑法规定不需要判处刑罚或免除刑罚 |

## 三、涉案财物处理

根据《刑诉法》第177条第3款：

```
对侦查机关冻结的XXX的存款/汇款的处理意见：[退还/没收/移送]。

如被不起诉人需给予行政处罚：应提出检察意见，移送有关主管机关处理。
```

## 四、权利告知

### 被害人权利
```
被害人如果不服本决定，可在收到决定书后七日以内向上一级人民检察院申诉，也可以不经申诉直接向人民法院起诉。
```

### 被不起诉人权利
```
被不起诉人如果不服本决定，可在收到决定书后七日以内向本院申诉。
```

## 五、送达规定

| 送达对象 | 送达要求 |
|----------|----------|
| 被不起诉人 | 公开宣布，交本人及所在单位 |
| 公安机关 | 送达决定书公安机关，可要求复议 |
| 被害人 | 送达决定书，告知申诉和起诉权利 |
| 在押人员 | 宣布后立即释放 |

## 六、不起诉类型判定

请根据以下条件确定不起诉类型：

**法定不起诉（第16条）—— 符合任一情形：**
- 情节显著轻微，危害不大，不认为是犯罪
- 犯罪已过追诉时效期限
- 经特赦令免除刑罚
- 依照刑法告诉才处理的犯罪，没有告诉或撤回告诉
- 犯罪嫌疑人、被告人死亡
- 其他法律规定免予追究刑事责任

**证据不足不起诉（第175条第4款）：**
- 经过二次补充侦查
- 证据仍然不足
- 不能认定构成犯罪

**酌定不起诉（第177条第2款）：**
- 犯罪情节轻微
- 依照刑法规定不需要判处刑罚
- 或免除刑罚
