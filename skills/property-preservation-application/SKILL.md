# 财产保全申请书

财产保全申请书是在民事案件审理过程中，公民、法人或其他组织为维护自己的合法权益和保证将来裁判的执行，请求人民法院对诉讼标的物或与本案有关的财物采取某种强制措施时使用的文书。

## 文书类型

### 1. 诉讼中的财产保全申请书
在民事案件、经济案件中，对于可能因当事人的行为或其他原因，使将来的判决不能执行或难以执行，另一方当事人申请人民法院对对方当事人的财产或者有争议的标的物采取保全措施时制作的文书。

### 2. 诉讼前的财产保全申请书
在情况紧急下，利害关系人如果不立即申请财产保全将会使自己的合法权益受到难以弥补的损害，在起诉前向人民法院申请采取财产保全时所制作的文书。

## 结构规范

### 首部

1. **标题**：在文书顶端居中书写"财产保全申请书"

2. **当事人基本情况**：
   - 当事人是自然人的：写明姓名、性别、年龄、出生日期、籍贯、工作单位或职业、现家庭住址等
   - 当事人是法人或其他组织的：写明名称、住址、法定代表人姓名及职务

### 正文

#### 1. 申请事项
写明需要采取保全措施的标的物名称、数量、价值。必须写明要求保全财产的总金额，如无法计算准确，也可以写作"或以相应价值的财产"。

#### 2. 事实与理由
- 写明申请人与被申请人因何发生纠纷
- 说明需要采取保全措施的标的物名称、数量、价值
- 写明申请财产保全措施的原因（如被申请人有损毁、转移、挥霍、隐匿、出卖诉争标的物的迹象、行为与可能等）
- 举出相关证据
- 论证财产保全措施的必要性和迫切性

#### 3. 请求目的
- 请求人民法院对被申请人的何项财产实施财产保全
- 采取何种方法（查封、扣押、冻结等）
- 表明是否提供担保以及提供何种担保

### 尾部

1. **致送机关**：
   - 第一行：此致
   - 第二行：××××人民法院（顶格）

2. **附项**：
   - 附上有关证据
   - 附上担保材料

3. **落款**：
   - 申请人签名或盖章
   - 申请日期

## 写作要点

1. **申请条件说明**：申请书中应向受诉人民法院说明申请诉讼财产保全的原因

2. **保全内容明确**：请求保全的标的物或者有关财产的种类、数量、价额及所在地必须明确

3. **法定程序**：申请财产保全时，必须依照法定程序进行

4. **双方均可申请**：双方当事人的任何一方都可以向受诉人民法院申请财产保全

5. **提供担保**：申请人应说明是否提供担保以及担保方式

## 保全措施类型

- **查封**：对财产进行封存，禁止转移处分
- **扣押**：将财产置于法院控制之下
- **冻结**：冻结银行账户或资金
- **其他**：法律允许的其他保全措施

## 示例结构框架

```
财产保全申请书

申请人：×××，男/女，××年××月××日出生，...
（或：申请人：××公司，住所地：...）

被申请人：×××，...

申请事项：
请求对被申请人所有的××财产（数量：××，价值：××元）采取保全措施。

事实与理由：
一、申请人与被申请人因××纠纷，已由××人民法院受理...
二、被申请人有转移、隐匿财产的行为/迹象...
三、如不采取保全措施，将导致判决难以执行...

请求目的：
1. 请求对被申请人名下的××财产采取查封/扣押/冻结措施
2. 申请人愿提供××担保

此致
××××人民法院

申请人：×××
××年××月××日

附：
1. 证据清单
2. 担保材料
```
