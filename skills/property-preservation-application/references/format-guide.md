# 财产保全申请书格式指南

## 8171 格式写法

财产保全申请书由首部、正文和尾部三部分组成。

### 1 首部

首部包括标题和当事人的基本情况。

#### （1）标题

即在文书顶端居中书写文书名称"财产保全申请书"。

#### （2）当事人的基本情况

分两种情况：
- **当事人是自然人的**：分别写明其姓名、性别、年龄、出生日期、籍贯、工作单位或职业、现家庭住址等
- **当事人是法人或其他组织的**：分别写明其名称、住址、法定代表人姓名及职务

### 2 正文

正文包括申请事项、事实与理由、请求目的。

#### （1）申请事项

即写明需要采取保全措施的标的物名称、数量、价值。必须写明要求保全财产的总金额，如无法计算准确，也可以写作"或以相应价值的财产"。

#### （2）事实与理由

首先，写明申请人与被申请人因何发生纠纷，需要采取保全措施的标的物名称、数量、价值；其次，写明申请财产保全措施的原因，如被申请人有损毁、转移、挥霍、隐匿、出卖诉争标的物的迹象、行为与可能等，并举出证据；最后，论证理由，应着重根据有关事实和判断，说明实施财产保全措施的必要性和迫切性。

#### （3）请求目的

请求人民法院对被申请人的何项财产实施财产保全，以及采取何种方法（如查封、扣押、冻结等）实施财产保全；同时，还要表明自己是否提供担保以及提供何种担保。

### 3 尾部

尾部包括致送机关、附项和落款。

#### （1）致送机关

即在正文左下角第一行书写"此致"，第二行顶格写"××××人民法院"。

#### （2）附项

除附上有关证据外，还应附上担保材料。

#### （3）落款

即由申请人签名或盖章，下一行注明申请日期。

---

## 8172 写作注意事项

1. **书面申请要求**：当事人书面申请诉讼中财产保全时，申请书中应向受诉人民法院说明申请诉讼财产保全的原因，请求保全的标的物或者有关财产的种类、数量、价额及所在地。

2. **法定程序**：申请财产保全时，必须依照法定程序进行，不能任意提出。

3. **双方均可申请**：双方当事人的任何一方都可以向受诉人民法院申请财产保全，递交财产保全申请书。

## 完整范文

### 诉讼中财产保全申请书范文

```
财产保全申请书

申请人：李某某，男，1975年3月15日出生，汉族，籍贯山东济南，现住北京市朝阳区XX路XX号，工作单位：北京市某某贸易公司，职务：经理。身份证号：110XXXXXXXXXXXXXXX。

被申请人：王某某，男，1972年8月20日出生，汉族，籍贯河北石家庄，现住北京市海淀区XX路XX号，工作单位：河北省某某建材公司，职务：总经理。身份证号：130XXXXXXXXXXXXXXX。

申请事项：
请求对被申请人王某某名下位于北京市海淀区XX路XX号房产（房产证号：X京房权证海字第XXXX号）及银行账户（开户行：中国工商银行北京分行，账号：XXXXXXXXXXXXXXXXXXX）采取保全措施，保全金额以人民币300万元为限。

事实与理由：

一、申请人与被申请人因买卖合同纠纷，已由北京市第一中级人民法院受理，案号为（2024）京01民初XXX号，目前正在审理中。

二、涉案标的金额为人民币300万元，被申请人作为合同违约方，至今未履行付款义务。申请人有充分证据证明被申请人存在转移、隐匿财产的行为：（1）被申请人于2024年5月将其持有的某某公司股权转让给第三方；（2）被申请人近期有多笔大额资金转出记录；（3）被申请人名下有多处房产正在挂牌出售。

三、鉴于上述情况，如不立即采取财产保全措施，将可能导致被申请人转移、隐匿财产，使申请人将来胜诉后难以执行，严重损害申请人的合法权益。

请求目的：
1. 请求对被申请人王某某名下价值约300万元的财产采取查封、冻结措施；
2. 申请人愿意提供某某担保公司出具的保函作为担保。

此致

北京市第一中级人民法院

申请人：李某某（签名）
2024年6月15日

附：
1. 买卖合同复印件1份
2. 被申请人转移财产的证据材料3份
3. 担保函原件1份
4. 申请人身份证复印件1份
```

```
财产保全申请书

被申请人：乙公司，住所地上海市浦东新区XX路XX号，法定代表人：赵某，职务：总经理。

申请事项：
请求对被申请人乙公司名下银行账户（开户行：招商银行上海分行，账号：XXXXXXXXXXXXXXXX）及应收债权采取保全措施，保全金额以人民币500万元为限。

事实与理由：

一、申请人与被申请人存在多年的供货合作关系，被申请人拖欠申请人货款累计人民币500万元。

二、申请人近日获悉，被申请人因经营不善，已有多起债务纠纷案件被起诉，且被申请人正在秘密转移资产，准备注销公司。

三、情况紧急，如不立即申请财产保全，申请人将遭受难以弥补的损失。

请求目的：
1. 请求对被申请人名下银行账户采取冻结措施；
2. 请求对被申请人的应收债权采取保全措施；
3. 申请人已提供等额保证金作为担保。

此致

上海市浦东新区人民法院

申请人：甲公司（盖章）
法定代表人：张某（签名）
2024年7月1日

附：
1. 供货合同复印件5份
2. 欠款对账单2份
3. 被申请人债务纠纷案件材料3份
4. 银行转账记录5份
5. 保证金缴款凭证1份
```

## 法律依据

### 《中华人民共和国民事诉讼法》相关条款

- **第一百条**：人民法院对于可能因当事人一方的行为或者其他原因，使判决难以执行或者造成当事人其他损害的案件，根据对方当事人的申请，可以裁定对其财产进行保全、责令其作出一定行为或者禁止其作出一定行为。

- **第一百零一条**：利害关系人因情况紧急，不立即申请保全将会使其合法权益受到难以弥补的损害的，可以在提起诉讼或者申请仲裁前向被保全财产所在地、被申请人住所地或者对案件有管辖权的人民法院申请采取保全措施。

### 《最高人民法院关于人民法院办理财产保全案件若干问题的规定》

- 第一条：当事人、利害关系人申请财产保全，应当向人民法院递交申请书
- 第九条：在诉讼中，可以用其依法占有或者担保的财产申请保全
- 第十条：申请保全应当提供担保
