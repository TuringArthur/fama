# 通知撤销案件书生成器

## 功能概述

本技能用于生成符合中国检察机关规范格式的《通知撤销案件书》。该文书是人民检察院在审查侦查机关（公安机关）回复的《立案理由说明书》后，认为立案理由不能成立时，向侦查机关发出的撤销案件通知书。

## 适用场景

- 检察官起草撤销案件通知
- 法务人员撰写案件撤销文书
- 法律专业学生练习法律文书写作
- 司法机关工作人员处理案件撤销流程

## 文书格式规范

### 标准结构

```
[机关名称]人民检察院
通知撤销案件书

[文号]：×检×通撤〔×××〕×号

（主送机关名称）：
本院于____年____月____日收到你局回复的《"立案理由说明书"》，经审查认为：（写明侦查机关立案理由不能成立的原因和应当撤销案件的事实、法律依据）。

根据《中华人民共和国刑事诉讼法》及《人民检察院刑事诉讼规则》第561、563、564条之规定，现通知你局撤销____案。望接到本通知后，按照相关规定办理撤销案件手续，并将撤销案件决定书复印件及时送达本院。

                                                           20××年××月××日
                                                           （院印）
```

### 格式要求

| 要素 | 格式规范 |
|------|----------|
| 机关名称 | 居中，黑体或加粗宋体 |
| 文种名称 | 居中，字号稍大 |
| 文号 | 右对齐，使用六角括号〔〕 |
| 主送机关 | 左对齐，顶格书写 |
| 正文 | 每段开头缩进两个字符 |
| 落款日期 | 右对齐 |

## 交互流程

### 第一步：收集基本信息

请提供以下信息：

- **检察院名称**：例如"北京市人民检察院"
- **案件类型**：刑事案件/经济案件/职务犯罪案件等
- **主送机关**：侦查机关名称，如"北京市公安局"
- **案件接收日期**：收到《立案理由说明书》的日期
- **案件编号**：原立案案件编号

### 第二步：收集案件详情

- **案件名称**：案件具体名称
- **立案机关**：最初立案的侦查机关
- **立案时间**：原立案日期
- **立案理由摘要**：侦查机关提出的立案理由

### 第三步：撤销理由分析

请详细说明：

- **事实依据**：说明立案理由不成立的具体事实
- **法律依据**：引用的具体法律条款
- **审查结论**：综合分析结论

### 第四步：确认生成

- 确认文书内容无误
- 确认格式符合规范
- 确认需要生成正式文档

## 输出示例

### 输入信息

- 检察院：上海市人民检察院
- 主送机关：上海市公安局
- 案件接收日期：2024年3月15日
- 案件编号：（2024）沪公经立字第023号
- 案件名称：某某公司涉嫌合同诈骗案

### 生成结果

```
上海市人民检察院
通知撤销案件书

沪检通撤〔2024〕015号

上海市公安局：
本院于2024年3月15日收到你局回复的《"立案理由说明书"》，经审查认为：根据现有证据材料，你局所立的"某某公司涉嫌合同诈骗案"存在以下问题：一是涉案合同经鉴定系双方真实意思表示，不存在欺诈情形；二是涉案款项往来均有银行转账记录及财务凭证，款项性质明确；三是现有证据不能证明当事人具有非法占有目的。综上，你局的立案理由不能成立。

根据《中华人民共和国刑事诉讼法》第一百六十三条及《人民检察院刑事诉讼规则》第561、563、564条之规定，现通知你局撤销（2024）沪公经立字第023号案件。望接到本通知后，按照相关规定办理撤销案件手续，并将撤销案件决定书复印件及时送达本院。

                                                           2024年4月20日
                                                           （院印）
```

## 法律依据

- 《中华人民共和国刑事诉讼法》第一百六十三条
- 《人民检察院刑事诉讼规则》第五百六十一条、第五百六十三条、第五百六十四条
- 相关司法解释和规范性文件

## 注意事项

1. 文书中涉及的具体日期、案号、案件名称等内容需根据实际情况填写
2. 撤销理由部分需详实充分，事实与法律依据相结合
3. 法律条款引用需准确，与案件情况相匹配
4. 文书格式需严格遵循检察机关公文规范
5. 正式使用前需经相关领导审批并加盖院印
