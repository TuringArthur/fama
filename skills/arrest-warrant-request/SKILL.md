# 提请批准逮捕书生成器

本工具用于生成标准格式的《提请批准逮捕书》，适用于公安机关办理刑事案件时依法向同级人民检察院提请批准逮捕犯罪嫌疑人。

## 适用场景

- 公安机关侦查终结后，认为需要逮捕犯罪嫌疑人
- 案件事实清楚，证据确实充分
- 犯罪嫌疑人符合逮捕条件
- 需要向检察院呈请批准逮捕

## 文书结构

### 1. 文书标题

```
[公安局名称]
提请批准逮捕书
```

### 2. 文书编号

格式：`[公安局简称]公提捕字〔年份〕××号`

示例：`×公提捕字〔2020〕××号`

### 3. 犯罪嫌疑人基本信息

| 字段 | 说明 | 示例 |
|------|------|------|
| 姓名 | 犯罪嫌疑人的正式姓名 | 王×× |
| 性别 | 男/女 | 男 |
| 出生日期 | 精确到年月日 | 19××年×月×日 |
| 籍贯 | 籍贯所在地 | 江苏省××区人 |
| 身份证号 | 18位身份证号码 | 321023××××××26×××× |
| 民族 | 民族名称 | 汉族 |
| 文化程度 | 学历层次 | 初中文化 |
| 职业 | 工作职务 | ××市标准厂司机 |
| 现住址 | 实际居住地址 | ××路南街4号 |
| 前科情况 | 是否有犯罪前科 | 2007年因偷窃被劳教2年，于2009年7月被解除教养 |
| 强制措施 | 当前强制措施状态 | 2020年11月19日因涉嫌盗窃罪、抢劫罪、放火罪被刑事拘留 |

### 4. 案由和案件来源

包含以下要素：
- **案件名称**：犯罪嫌疑人涉嫌罪名
- **案件来源**：发现人报案/公安机关自行发现等
- **立案时间**：公安机关立案侦查的时间
- **归案情况**：犯罪嫌疑人归案时间及方式

### 5. 违法犯罪事实

详细叙述犯罪嫌疑人的违法犯罪行为，包括：

**叙事要素**：
- 作案动机
- 作案时间
- 作案地点
- 作案手段
- 犯罪对象
- 犯罪后果
- 涉案金额
- 归案情况

### 6. 证据清单

列举认定犯罪事实的证据材料：

| 证据类型 | 证据名称 | 数量/份数 |
|----------|----------|------------|
| 物证 | 克丝钳子、菜刀、剪刀等 | 若干 |
| 书证 | 报案记录、调查报告等 | 若干份 |
| 勘验笔录 | 现场勘查笔录 | 1份 |
| 鉴定意见 | 尸检鉴定书、脚印对比鉴定书等 | 若干份 |
| 供述材料 | 讯问记录、犯罪嫌疑人口供 | 若干份 |

### 7. 法律依据与提请理由

**法律依据**：

- **实体法依据**：《中华人民共和国刑法》相关条款
- **程序法依据**：《中华人民共和国刑事诉讼法》第八十一条、第八十七条

**提请理由**：

1. 犯罪事实清楚，证据确实充分
2. 涉嫌罪名及触犯的法律条款
3. 可能判处的刑罚
4. 采取逮捕措施的必要性和紧迫性

### 8. 呈送机关

```
此致
[同级人民检察院全称]
```

### 9. 落款

```
[公安局名称]（印）
年  月  日
```

### 10. 附件

列明随文书移送的案件材料：

- 卷宗册数及页数
- 证据清单
- 其他相关材料

## 填写说明

### 基本信息填写规范

1. **姓名**：使用正式姓名，不得使用绰号或别名
2. **出生日期**：按"年×月×日"格式填写
3. **身份证号**：填写完整的18位身份证号码
4. **籍贯**：填写至区县级行政区划
5. **现住址**：填写实际居住地址，精确到门牌号

### 犯罪事实叙述要点

1. **时间明确**：具体到年、月、日、时
2. **地点精确**：具体到省、市、区、路、号
3. **手段具体**：详细描述犯罪方法和使用工具
4. **后果清晰**：说明犯罪造成的具体损害结果
5. **金额准确**：涉及财产犯罪的需注明具体数额

### 证据列举要求

1. **分类列举**：按物证、书证、证人证言等分类
2. **数量准确**：注明每类证据的具体数量
3. **来源合法**：说明证据的收集方式和合法依据

### 法律依据引用

1. **刑法条款**：引用涉嫌罪名对应的刑法条文
2. **刑诉法条款**：引用程序法依据条款
3. **条款准确**：确保引用的法律条款现行有效

## 注意事项

1. 文书内容应当客观、真实、全面
2. 犯罪事实的叙述应当因果关系清晰、逻辑严密
3. 证据与犯罪事实之间应当相互印证
4. 法律依据的引用应当准确、完整
5. 文书格式应当符合公安部规定的标准格式
6. 涉及国家秘密、个人隐私的内容应当保密处理
