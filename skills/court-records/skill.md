# 法院笔录制作技能

## 概述

法院笔录是人民法院在审判活动中形成的重要法律文书，用于客观记录庭审过程、合议庭评议和宣判活动等司法程序。本技能指导生成符合规范的各类法院笔录文书。

## 笔录类型分类

本技能涵盖以下四类核心笔录：

| 笔录类型 | 适用范围 | 核心功能 |
|---------|---------|---------|
| 法庭审理笔录 | 开庭审判活动 | 记录庭审全过程 |
| 合议庭评议笔录 | 案件评议讨论 | 记录合议庭意见 |
| 宣判笔录 | 判决宣告活动 | 记录宣判过程 |
| 询问/讯问笔录 | 调查取证活动 | 记录讯问内容 |

## 第一部分：法庭审理笔录

### 1.1 基本格式结构

```
[法院名称]
法庭审理笔录（第 次）

时间：____年____月____日____时____分至____时____分
地点：本院____法庭
是否公开审理：是（ ）否（ ）
旁听人数：____人

审判人员：审判长____、审判员____、人民陪审员____
书记员：____

公诉人：____
辩护人：____
鉴定人/翻译人：____

记录如下：
```

### 1.2 首部必填信息

| 序号 | 字段名称 | 填写要求 | 示例 |
|-----|---------|---------|------|
| 1 | 法院名称 | 全称，冠以省、市、县名称 | 北京市高级人民法院 |
| 2 | 笔录次数 | 第几次开庭 | 第1次 |
| 3 | 庭审时间 | 精确到年月日时分 | 2024年3月15日9时0分至11时30分 |
| 4 | 庭审地点 | 具体法庭编号 | 本院第二法庭 |
| 5 | 公开情况 | 是或否，不公开需说明理由 | 是 |
| 6 | 旁听人数 | 实际旁听人数 | 15人 |
| 7 | 审判长 | 合议庭组长姓名 | 张三 |
| 8 | 审判员 | 其他法官姓名 | 李四、王五 |
| 9 | 人民陪审员 | 参与审判的陪审员 | 赵六 |
| 10 | 书记员 | 记录人员姓名 | 钱七 |
| 11 | 公诉人 | 检察院名称及检察员 | 北京市人民检察院检察员孙八 |
| 12 | 辩护人 | 律师姓名及所属律所 | 周九，北京市XX律师事务所 |
| 13 | 案由 | 案件性质 | 故意伤害罪 |

### 1.3 庭审过程记录要点

#### 1.3.1 预备程序

- 宣布开庭的具体时间
- 宣布案由和案件名称
- 核对当事人基本情况

#### 1.3.2 身份核实（被告人信息）

| 信息项 | 记录内容 |
|-------|---------|
| 姓名 | 真实姓名 |
| 别名/曾用名 | 如有需注明 |
| 性别 | 男/女 |
| 出生年月日 | 精确到年月日 |
| 年龄 | 以出生日期计算 |
| 民族 | 如汉族、满族等 |
| 籍贯/户籍所在地 | 身份证登记地址 |
| 家庭住址 | 实际居住地址 |
| 文化程度 | 最高学历 |
| 捕前单位及职务 | 原工作单位和职位 |
| 党派团体 | 政治面貌 |
| 前科/法律处分 | 劳动教养、刑事处罚记录 |
| 本次被拘留原因 | 涉嫌罪名 |
| 强制措施时间 | 拘留、逮捕日期 |

#### 1.3.3 权利告知程序

- 告知回避权：询问是否申请合议庭成员、公诉人、书记员回避
- 告知辩护权：宣读《刑事诉讼法》第29、30、31、32条相关条款
- 告知最后陈述权

#### 1.3.4 法庭调查阶段

记录以下庭审活动：

1. **审判人员讯问**：审判长、审判员的具体提问内容
2. **被告人陈述**：被告人对指控的回应
3. **公诉人发言**：公诉人的讯问和指控要点
4. **辩护人发问**：辩护人的质证和辩护意见
5. **证人证言**：证人的陈述和质证过程
6. **鉴定人意见**：鉴定人宣读的鉴定结论
7. **物证书证**：出示证据的名称和内容

#### 1.3.5 法庭辩论阶段

| 阶段 | 记录内容 |
|-----|---------|
| 公诉词 | 公诉人的控告意见 |
| 辩护词 | 辩护人的辩护意见 |
| 互相辩论 | 控辩双方的辩论要点 |
| 辩论终结 | 宣布辩论结束 |
| 最后陈述 | 被告人的最后陈述 |

#### 1.3.6 特殊表情动作记录

需准确记录当事人在法庭上的特殊表情和动作，包括但不限于：

- 哭泣、笑声
- 低头不语
- 踌躇思考
- 支吾搪塞
- 态度坚决
- 情绪激动

### 1.4 问答式记录符号

| 符号 | 含义 | 使用场景 |
|-----|------|---------|
| 审？ | 审判长/审判员提问 | 正式记录格式 |
| 原： | 原告回答 | 民事案件 |
| 被： | 被告回答 | 各类案件 |
| 陪？ | 人民陪审员提问 | 陪审员发问 |
| 公： | 公诉人发言 | 刑事公诉案件 |
| 辩： | 辩护人发言 | 有辩护人时 |
| 证： | 证人陈述 | 证人出庭 |

### 1.5 尾部签署要求

```
笔录经审判长阅看后，由审判长和书记员签名。

证人证言部分应当交证人阅读或向其宣读，经确认无误后由证人签名或盖章。

当事人有权阅读笔录或请求宣读，认为有遗漏或差错的有权请求补充或改正。

审判长：________（签名）
书记员：________（签名）

[页码]
```

## 第二部分：合议庭评议笔录

### 2.1 基本格式

```
[法院名称]
合议庭评议笔录

会议次数：第____次
时间：____年____月____日____时____分至____时____分
地点：本院____办公室

审判长：____
审判员（人民陪审员）：____

书记员：____

评议案件：____一案

评议记录如下：
```

### 2.2 首部必填信息

| 字段 | 要求 |
|-----|------|
| 会议次数 | 第几次评议 |
| 时间 | 精确到年月日时分 |
| 地点 | 具体评议场所 |
| 审判长 | 姓名 |
| 审判员/人民陪审员 | 所有成员姓名 |
| 书记员 | 记录人员 |
| 评议案件 | 案由和当事人 |

### 2.3 评议内容要点

评议笔录必须详细记录合议庭对以下八个方面的讨论：

#### 2.3.1 案情认定

- 案件基本事实的确认
- 起诉书指控事实的核实

#### 2.3.2 证据核实

- 各证据的真实性认定
- 证据之间的印证关系
- 证据链的完整性
- 存疑证据的处理意见

#### 2.3.3 定罪意见

- 被告人的行为是否构成犯罪
- 构成何种罪名
- 犯罪性质认定（故意/过失）

#### 2.3.4 量刑情节

| 情节类型 | 记录内容 |
|---------|---------|
| 从重情节 | 具体情节和认定依据 |
| 从轻情节 | 具体情节和认定依据 |
| 减轻情节 | 法定/酌定减轻理由 |
| 免除情节 | 免除处罚的法定条件 |

#### 2.3.5 刑罚确定

- 主刑种类选择
- 刑期确定（有期徒刑年数、拘役月数等）
- 附加刑（罚金、剥夺政治权利等）
- 数罪并罚的处理
- 缓刑适用意见

#### 2.3.6 附带民事诉讼

- 民事赔偿范围
- 赔偿金额认定
- 赔偿责任分配

#### 2.3.7 法律依据

- 引用的具体法律条文
- 适用理由说明

#### 2.3.8 财物处理

- 赃款追缴意见
- 赃物处理方式
- 违禁品处理

### 2.4 记录要求

1. **民主集中制**：反映评议的民主讨论过程
2. **全面性**：完整记录各成员的发言内容
3. **分歧记录**：少数人的不同意见必须详细记载
4. **准确性**：精确记录每个人的原话

### 2.5 评议结果格式

```
评议结果：

定罪：____罪
法律依据：____
主刑：____
附加刑：____
附带民事：____
赃款赃物处理：____

[全体合议庭成员签名]
审判长：________
审判员：________
人民陪审员：________
书记员：________
```

## 第三部分：宣判笔录

### 3.1 基本格式

```
[法院名称]
宣判笔录

时间：____年____月____日____时____分
地点：本院____法庭

审判长：____
审判员（人民陪审员）：____
书记员：____

到庭公诉人：____
到庭当事人及其他诉讼参与人：____

记录如下：
```

### 3.2 首部必填信息

| 字段 | 填写内容 |
|-----|---------|
| 宣判时间 | 精确到年月日时分 |
| 宣判地点 | 具体法庭 |
| 审判人员 | 审判长、审判员姓名 |
| 书记员 | 记录人员 |
| 公诉人 | 检察院及检察员 |
| 当事人 | 到庭人员基本信息 |

### 3.3 当事人信息详细项

| 信息项 | 要求 |
|-------|------|
| 姓名 | 准确记录 |
| 性别 | 男/女 |
| 年龄 | 周岁 |
| 民族 | 如汉族 |
| 籍贯 | 出生地 |
| 住址 | 现住址 |
| 职业/工作单位 | 捕前职业 |
| 被逮捕日期 | 羁押日期 |

### 3.4 宣判记录内容

#### 3.4.1 宣判形式

```
审判长（员）宣读____人民法院____年____月____日____号____判决书：
```

#### 3.4.2 必须记录的内容

1. **判决书文号**：如(2024)XX刑初XX号
2. **法律依据**：引用的具体法条
3. **判决结果**：
   - 罪名认定
   - 主刑内容（刑种和刑期）
   - 附加刑内容
   - 是否数罪并罚
4. **上诉权利告知**：
   - 上诉期限（刑事10日，民事15日）
   - 上诉状提交份数
   - 上诉法院名称
5. **判决书送达情况**

#### 3.4.3 问答记录

```
审：被告人，你听清了吗？
被：听清了。
审：你是否上诉？
被：不上诉。（或：我上诉。）
```

### 3.5 尾部签署

```
当事人：________（签名或盖章）
宣判人：________（签名）
书记员：________（签名）

____年____月____日
```

```

### 4.2 讯问内容记录要点

#### 4.2.1 犯罪事实记录

| 要素 | 记录要求 |
|-----|---------|
| 时间 | 案发具体时间点 |
| 地点 | 案发具体场所 |
| 动机 | 犯罪主观原因 |

问：你以上笔录和你说的相符吗？

```

## 第五部分：通用记录规范

### 5.1 语言文字要求

| 要求 | 说明 |
|-----|------|
| 字体工整 | 使用规范汉字 |
| 文句通顺 | 语法正确，表意清晰 |
| 标点准确 | 正确使用标点符号 |
| 禁止错别字 | 不得出现文字错误 |
| 口语化保留 | 当事人原话尽量保留 |

### 5.2 记录原则

#### 5.2.1 忠于原意

- 不要求字不漏地记录
- 但必须准确反映当事人原意
- 对归纳性记录需经当事人确认

#### 5.2.2 详略得当

| 应当详记 | 可以略记 |
|---------|---------|
| 案件关键环节 | 与案件无关的闲话 |
| 时间地点 | 一般性程序性话语 |
| 当事人姓名 | 重复确认性话语 |
| 涉及财物 | 过渡性陈述 |
| 证据要点 | 情绪性表达 |
| 核心辩解意见 | 一般性认罪态度 |

#### 5.2.3 准确迅速

- 提前熟悉案情
- 掌握案件重点
- 准确记录专业术语
- 遇有遗漏可提示审判长

### 5.3 格式统一要求

- 每页需标注页码
- 续页应保持连续编号
- 笔录不得涂改
- 如有修改需签字确认

## 第六部分：输入信息清单

制作各类笔录时，需收集以下信息：

### 6.1 案件基本信息

- 法院名称
- 案件编号
- 案由/罪名
- 案件性质（刑事/民事/行政）

### 6.2 时间地点信息

- 开庭/评议/宣判时间
- 具体地点
- 第几次开庭/评议

### 6.3 人员信息

| 人员类别 | 所需信息 |
|---------|---------|
| 审判长 | 姓名 |
| 审判员 | 姓名（多人需全部） |
| 人民陪审员 | 姓名 |
| 书记员 | 姓名 |
| 公诉人/原告 | 姓名、机关 |
| 辩护人/被告代理人 | 姓名、律所 |
| 被告/当事人 | 详细身份信息 |
| 证人 | 姓名、基本信息 |

### 6.4 案件内容信息

- 起诉/指控内容
- 被告人陈述
- 证据情况
- 辩论要点
- 评议意见
- 判决结果

## 第七部分：输出示例

### 7.1 法庭审理笔录片段示例

```
北京市中级人民法院
法庭审理笔录（第1次）

时间：2024年3月15日9时0分至11时30分
地点：本院第三法庭
是否公开审理：是
旁听人数：12人

审判人员：审判长张明、审判员李华、审判员王强

书记员：赵敏

公诉人：北京市人民检察院检察员陈伟
辩护人：孙磊，北京市华正律师事务所律师

案由：故意伤害罪

记录如下：

审：现在开庭。传被告人李某某到庭。
（被告人李某某到庭）

审：被告人李某某，你的姓名？
被：我叫李某某。
审：有无其他名字？
被：没有。
审：性别？
被：男。
审：出生日期？
被：1985年6月15日。
审：多大年龄？
被：38周岁。
审：民族？
被：汉族。
审：籍贯？
被：北京市朝阳区。
审：住址？
被：北京市海淀区XX小区3号楼201室。
审：文化程度？
被：大学本科。
审：捕前工作单位及职务？
被：北京市XX公司，工程师。
...

审判长：张明（签名）
书记员：赵敏（签名）
```

## 注意事项

1. **真实性优先**：所有记录必须客观真实，不得篡改
2. **完整性保证**：关键信息不得遗漏
3. **合法性审查**：确保程序符合法律规定
4. **隐私保护**：涉及隐私内容应按规定处理
5. **及时整理**：闭庭后应及时校正、整理、补漏
