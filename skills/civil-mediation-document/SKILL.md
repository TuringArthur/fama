# 民事调解书生成技能

## 触发条件

当用户请求以下内容时触发本技能：
- "生成调解书"、"生成民事调解书"
- "一审调解书"、"二审调解书"
- "法院调解"、"民事调解"
- 需要制作人民法院调解书

## 工作流程

### 步骤 1：收集案件信息

使用 GenUI 表单向用户收集以下信息：

**基本信息（必填）**
- 案件类型：一审 / 二审
- 法院名称
- 案号（如：(2024)X民初XXX号 或 (2024)X民终XXX号）

**当事人信息**
- 原告/上诉人：姓名或名称、基本情况
- 被告/被上诉人：姓名或名称、基本情况
- 第三人（如有）：姓名或名称、基本情况

**案件内容**
- 案由
- 案件事实
- 诉讼请求
- 协议内容
- 诉讼费用负担

**签署信息**
- 审判长姓名
- 审判员姓名（如有两名）
- 书记员姓名
- 判决日期

### 步骤 2：生成调解书

根据案件类型选择对应格式：

**一审民事调解书格式：**

```
××××人民法院
民事调解书
(××)××民初××号

原告：……（写明姓名或名称等基本情况）
被告：……（写明姓名或名称等基本情况）
第三人：……（写明姓名或名称等基本情况）

案由：……

……（案件事实）

……（诉讼请求）

经本院主持调解，双方当事人自愿达成如下协议：
……（协议内容）

上述协议，符合有关法律规定，本院予以确认。本调解书经双方当事人签收后，即具有法律效力。

……（诉讼费用的负担）

审判长    ××
审判员    ××
审判员    ××
××年××月××日
（院印）

本件与原本核对无异        书记员    ××
××年××月××日
```

**二审民事调解书格式：**

```
××××人民法院
民事调解书
(××)××民终××号

上诉人（原审××）：……（写明姓名或名称等基本情况）
被上诉人（原审××）：……（写明姓名或名称等基本情况）
第三人：……（写明姓名或名称等基本情况）

案由：……

上诉人××不服××人民法院(××)××民初××号民事判决，向本院提起上诉，请求本院……（写明上诉请求）。

……（案件事实）

经本院主持调解，双方当事人自愿达成如下协议：
……（协议内容）

上述协议，符合有关法律规定，本院予以确认。本调解书经双方当事人签收后，即具有法律效力。

……（诉讼费用的负担）

原判决被撤销。

审判长    ××
审判员    ××
审判员    ××
××年××月××日
（院印）

本件与原本核对无异        书记员    ××
××年××月××日
```

### 步骤 3：输出文档

将生成的调解书保存为以下格式之一：
- Word 文档（.docx）
- PDF 文档（.pdf）
- 纯文本（.txt）

## 法律依据

- 《中华人民共和国民事诉讼法》
- 《最高人民法院关于人民法院民事调解工作若干问题的规定》
- 《法律文书写作（第五版）》

## 注意事项

1. 调解书应当体现**自愿、合法**原则
2. 调解书由**首部、事实、理由、调解协议、尾部**五部分组成
3. 二审调解书需写明**原判决被撤销**
4. 当事人信息写法与一审/二审民事判决书相同
5. 协议内容应明确、具体，具有可执行性
