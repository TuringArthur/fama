# 行政起诉状撰写技能

## 适用场景

- 公民、法人认为行政机关具体行政行为侵犯合法权益
- 对行政处罚决定不服需要提起行政诉讼
- 对行政机关不履行法定职责提起诉讼
- 申请行政赔偿

## 起诉条件

| 条件 | 说明 |
|------|------|
| 原告资格 | 认为具体行政行为侵犯合法权益的公民、法人或其他组织 |
| 明确被告 | 做出具体行政行为的行政机关或法律法规授权的组织 |
| 具体请求 | 有具体的诉讼请求和事实根据 |
| 受案范围 | 属于人民法院受案范围和受诉法院管辖 |

## 核心要素

### 1. 首部

- **标题**：居中书写"行政起诉状"
- **当事人**：
  - 原告：公民、法人或其他组织
  - 被告：只能是行政机关或法律法规授权的组织（不能以工作人员为被告）

### 2. 正文

#### 诉讼请求类型

| 类型 | 适用情形 | 请求内容 |
|------|----------|----------|
| 撤销判决 | 证据不足、适用法律错误、违反程序、滥用职权 | 请求撤销或部分撤销 |
| 履行判决 | 被告不履行或拖延履行法定职责 | 请求限期履行 |
| 变更判决 | 行政处罚显失公正 | 请求变更处罚 |
| 赔偿判决 | 具体行政行为造成损失 | 请求行政赔偿 |

#### 事实部分
- 行政机关做出具体行政行为的原因事实
- 具体行政行为的实施经过、行为内容、行为后果
- 违法之处的条理展现

#### 理由部分
- "综上所述，原告认为……"作为引语
- 对具体行政行为的不服之处
- 被告行为的错误或不当之处分析及论证
- 相关法律、法规的引用

#### 证据部分
- 行政侵权行为致害的证据
- 逐项列举各类证据

### 3. 尾部

- 致送机关："此致" + "××××人民法院"
- 附项：副本份数
- 落款：起诉人签名/盖章 + 日期

## 重要注意事项

| 可起诉 | 不可起诉 |
|--------|----------|
| 具体行政行为 | 抽象行政行为 |
| 行政处罚决定 | 准立法行为 |
| 行政许可决定 | 行政指导行为 |
| 行政强制措施 | 国家行为 |

## 附件要求

- 行政处罚决定书
- 行政复议决定书（如有）
- 其他相关证据材料

## 输出格式

生成完整的行政起诉状文档，包含：
- 标准法律文书格式
- 完整的四部分结构
- 针对性的诉讼请求和事实理由表述
