# 行政起诉状格式与示例参考

## 一、概念界定

### 1.1 文书定义

行政起诉状是公民、法人或者其他组织认为行政机关及其工作人员的具体行政行为侵犯其合法权益，以行政机关为被告向人民法院提起行政诉讼，请求人民法院作出裁判而制作的法律文书。

### 1.2 "民告官"性质

行政起诉状是一种"民告官"形式的法律文书，体现了公民对行政权力的司法监督权。

### 1.3 起诉条件

| 序号 | 条件 | 说明 |
|------|------|------|
| 1 | 原告资格 | 认为具体行政行为侵犯其合法权益的公民、法人或其他组织 |
| 2 | 明确被告 | 有明确的被告（行政机关或法律法规授权的组织） |
| 3 | 具体请求 | 有具体的诉讼请求和事实根据 |
| 4 | 受案范围 | 属于人民法院受案范围和受诉人民法院管辖 |

## 二、格式规范（8.4.1）

### 2.1 首部

#### （1）标题

在文书顶端居中书写"行政起诉状"。

#### （2）当事人基本情况

**原告为自然人**：
```
原告：XXX，男/女，XXXX年XX月XX日出生，X族，籍贯XX省XX市，
职业：XX，工作单位及职务：XXXX，住址：XXXXXX

被告：XXXX（行政机关名称）
住所地：XXXXXX
法定代表人（主要负责人）：XXX，职务：XXXXXX
```

**原告为法人或其他组织**：
```
原告：XXXX（单位名称）
地址：XXXXXX
联系方式：XXXXXX

法定代表人（主要负责人）：XXX，职务：XXXXXX

被告：XXXX（行政机关名称）
住所地：XXXXXX
法定代表人（主要负责人）：XXX，职务：XXXXXX
```

**委托诉讼代理人**：
在委托人项下另起一行写明：
```
委托诉讼代理人：XXX，男/女，工作单位：XXXXXX
```

**被告的特殊要求**：
- 只能是做出具体行政行为的行政机关
- 或者是法律法规授权的组织
- **不能以**行政机关工作人员为被告
- **不能以**未经法律法规授予行政权的其他组织为被告

### 2.2 正文

#### （1）诉讼请求

根据案件具体情况，可提出以下诉讼请求：

**请求类型一：撤销或部分撤销**
```
一、请求人民法院依法撤销被告XXXX作出的XXXX号《行政处罚决定书》；
二、请求人民法院依法撤销被告XXXX作出的XXXX号《行政强制决定书》（部分撤销）。
```

适用情形：
- 行政行为所依据的主要证据不足
- 适用法律法规错误
- 违反法定程序
- 滥用职权

**请求类型二：履行法定职责**
```
一、请求人民法院依法判令被告XXXX在一定期限内履行XXXX法定职责；
二、请求人民法院依法判令被告XXXX依法作出XXXX决定。
```

适用情形：
- 被告不履行法定职责
- 被告拖延履行法定职责

**请求类型三：变更行政处罚**
```
一、请求人民法院依法变更被告XXXX作出的XXXX号《行政处罚决定书》，
    将XXXX（处罚内容）变更为XXXX。
```

适用情形：
- 行政处罚显失公正

**请求类型四：行政赔偿**
```
一、请求人民法院依法判令被告XXXX赔偿因其错误的具体行政行为
    给原告造成的经济损失共计人民币XXXXX元（￥XXXXX.XX元）；
二、请求人民法院依法判令被告XXXX就其错误的具体行政行为
    向原告赔礼道歉。
```

适用情形：
- 具体行政行为侵犯合法权益造成损失

**综合请求示例**：
```
诉讼请求

一、请求人民法院依法撤销被告XX市公安局交通管理局作出的
    XX公交决字[XXXX]第XXXX号《行政处罚决定书》；
二、请求人民法院依法判令被告XX市公安局交通管理局赔偿因其
    错误行政处罚给原告造成的经济损失共计人民币XXXX元（￥XXXX.XX元）。
```

#### （2）事实

```
事实与理由

一、具体行政行为的原因事实

XXXX年XX月XX日，原告因XXXX（原因）在XXXX（地点）实施XXXX（行为）。
被告认为原告的行为违反了《中华人民共和国XXXX法》第XX条之规定，
于XXXX年XX月XX日对原告作出了XXXX（具体行政行为）。

二、具体行政行为的实施经过

XXXX年XX月XX日，被告以原告XXXX（违法行为）为由，
向原告送达了XXXX号《行政处罚告知书》。原告于XXXX年XX月XX日
提出了陈述申辩意见。被告于XXXX年XX月XX日经研究后，
作出了XXXX号《行政处罚决定书》，决定对原告处以：
1. 警告；
2. 罚款人民币XXXX元；
3. 没收违法所得XXXX元（如涉及）；
4. 责令停产停业（如涉及）；
5. 吊销许可证件（如涉及）；
6. 其他处罚：XXXX。

三、具体行政行为的内容

被告作出的XXXX号《行政处罚决定书》主要内容为：
XXXXXX（具体写明处罚内容）。

四、具体行政行为的后果

由于被告的错误具体行政行为，给原告造成了以下后果：
1. 经济损失：人民币XXXX元；
2. 经营损失：人民币XXXX元；
3. 名誉损失：XXXXXX；
4. 其他损失：XXXXXX。

五、违法之处

原告认为被告的具体行政行为存在以下违法之处：
1. 主要证据不足：XXXXXX；
2. 适用法律错误：XXXXXX；
3. 违反法定程序：XXXXXX；
4. 滥用职权：XXXXXX；
5. 显失公正：XXXXXX。
```

#### （3）理由

```
理由

综上所述，原告认为：

一、被告做出的XXXX号《行政处罚决定书》认定事实不清。

（详细论证被告认定事实方面的错误）

二、被告做出的XXXX号《行政处罚决定书》适用法律错误。

（详细论证被告适用法律方面的错误）

三、被告做出的XXXX号《行政处罚决定书》违反法定程序。

（详细论证被告程序方面的违法之处）

四、被告做出的XXXX号《行政处罚决定书》滥用职权/显失公正。

（详细论证被告滥用职权或处罚显失公正之处）

综上所述，被告的具体行政行为违反了以下法律规定：

1. 《中华人民共和国行政处罚法》第XX条第X款……；
2. 《中华人民共和国行政诉讼法》第XX条……；
3. 其他相关法律法规：XXXXXX。

根据《中华人民共和国行政诉讼法》第七十条/第七十二条/第七十六条之规定，
原告的诉讼请求合法有据，请求人民法院依法支持。
```

#### （4）证据

```

1. 书证：
   （1）XXXX号《行政处罚决定书》复印件X份，证明被告作出具体行政行为的事实；
   （2）XXXX号《行政处罚告知书》复印件X份，证明被告程序违法的事实；
   （3）罚款缴纳凭证原件X份，证明原告已缴纳罚款的事实；
   （4）原告身份证明文件复印件X份，证明原告身份；
   （5）营业执照复印件X份（如原告为单位）；
   （6）其他书证XXXXXX。

2. 物证：
   （1）XXXXXX；
   （2）XXXXXX。

3. 证人证言：
   （1）证人XXX（身份证号：XXXXXX）出具的书面证言原件X份；
   （2）证人XXX出庭作证申请书。

4. 鉴定意见：
   （1）XXXX鉴定意见书原件X份；
   （2）XXXX评估报告原件X份。

5. 视听资料/电子数据：
   （1）监控录像光盘X张；
   （2）照片X张；
   （3）录音光盘X张；
   （4）其他电子数据XXXXXX。

6. 当事人陈述：
   原告陈述原件X份，陈述被告具体行政行为违法的事实及给原告造成的损失。

   （1）医疗费票据原件X份，证明原告支出医疗费XXXX元；
   （2）误工证明原件X份，证明原告误工损失XXXX元；
   （3）财产损失清单原件X份，证明原告财产损失XXXX元；

8. 行政复议材料：
   （1）行政复议申请书复印件X份；
   （2）行政复议决定书复印件X份。
```

## 三、写作注意事项（8.4.2）

### 3.1 时效要求

行政起诉状应在诉讼时效期限内向有管辖权的人民法院提交。

### 3.2 被告确定

| 可以作为被告 | 不可以作为被告 |
|--------------|----------------|
| 做出具体行政行为的行政机关 | 行政机关工作人员 |
| 法律法规授权的组织 | 未经授权的其他组织 |
| 法律法规授权以自己名义做出行政行为的组织 | 行政机关的内设机构（如未经授权） |

### 3.3 受案范围

| 可以起诉 | 不可起诉 |
|----------|----------|
| 具体行政行为 | 抽象行政行为 |
| 行政处罚决定 | 准立法行为 |
| 行政许可决定 | 行政指导行为 |
| 行政强制措施 | 国家行为 |
| 行政不作为 | 内部行政行为 |
| 行政征收征用 | 仲裁裁决 |

### 3.4 附件要求

行政起诉状应附上：
- 行政处罚决定书
- 行政复议决定书（如经过复议）
- 证据材料清单

### 3.5 特殊情形

**经过行政复议的**：
- 原机关维持的，以原机关和复议机关为共同被告
- 复议机关改变原具体行政行为的，以复议机关为被告

**共同被告情形**：
- 两个以上行政机关做出同一具体行政行为的，共同做出具体行政行为的行政机关为共同被告

## 四、常见问题解答

### Q1：哪些人可以作为原告？

- 公民（自然人）
- 法人（企业法人、机关法人、事业单位法人、社会团体法人）
- 其他组织（依法登记领取营业执照的组织和有关机关核准登记的组织）

### Q2：行政诉讼的管辖法院如何确定？

- 一般由最初做出具体行政行为的行政机关所在地人民法院管辖
- 经过行政复议的，可以由原机关所在地或复议机关所在地法院管辖
- 对限制人身自由的行政强制措施不服，可由原告所在地法院管辖

### Q3：诉讼时效是多少？

- 直接起诉：知道做出具体行政行为之日起6个月内
- 经过复议：收到复议决定书之日起15日内

### Q4：原告需要承担举证责任吗？

- 原告对证明起诉符合法定条件承担举证责任
- 原告对被诉具体行政行为的违法性承担初步证明责任
- 被告对具体行政行为的合法性承担举证责任

### Q5：行政赔偿的程序？

1. 先向赔偿义务机关提出赔偿申请
2. 赔偿义务机关不予赔偿或赔偿金额有异议
3. 在提起行政诉讼时可以一并提出赔偿请求
4. 也可以单独提起行政赔偿诉讼
