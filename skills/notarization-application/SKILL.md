# 公证申请书

## 一、公证申请书的概念

公证申请书是当事人请求公证机构对法律行为、法律事实或有法律意义的文书予以公证的申请文书。

**主要分类：**
- 国内民事公证申请书
- 国内经济公证申请书
- 涉外民事公证申请书
- 涉外经济公证申请书

## 二、内容和写作方法

公证申请书格式有两种：**文字形式**和**表格形式**。

### （一）文字形式

文字形式由四个部分组成：

**1. 首部**
- 标题：`××公证申请书`（居中）
- 申请人基本情况：姓名、性别、出生日期、身份证号码、现住址

**2. 正文**
- 申请公证的内容
- 申请理由
- 相关事实
- 拟解决的问题

**3. 尾部**
- 申请人签名或盖章
- 申请日期

**4. 附项**
- 附件列表（相关证明材料）

#### 范例：学位公证申请书

```
学位公证申请书

申请人：王××，女，××××年7月18日出生，身份证号码××××，现住××省××市××区××路××号。

本人于20××年9月至20××年7月在××大学经济系学习，完成学业并获得经济学学士学位，请予公证。

附项：本人所获××大学毕业证书及学士学位证书。

                                    申请人：王××
                                    ××××年××月××日
```

### （二）表格形式

表格形式由公证机构提供，特点：规范、简明、便于填写。

申请人必须据实填写，最后签名或盖章（不能签名的由本人捺指印），并附上相关证明材料。

## 四、填写说明

1. **据实填写**：所有信息必须真实准确
2. **签名盖章**：完成后申请人必须签名或盖章，不能签名者由本人捺指印
3. **附证明材料**：随申请表提交相关法律文件
4. **选择表格类型**：
   - 个人事务 → 国内民事/涉外民事
   - 商业/经济事务 → 国内经济/涉外经济
   - 涉及境外使用 → 涉外类型
