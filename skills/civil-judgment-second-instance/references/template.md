# 第二审民事判决书模板

## 完整模板结构

```
[法院名称]人民法院
民事判决书
（二审维持原判或改判用）

（[年份]）[省简称][市简称]民终[序号]号

上诉人（写明原审诉讼地位）：[姓名]，[性别]，[民族]，[出生日期]日出生，住[住址]，身份证号：[身份证号]
[如是法人：上诉人（写明原审诉讼地位）：[公司名称]，住所地：[住所地]，法定代表人：[姓名]，职务：[职务]]

被上诉人（写明原审诉讼地位）：[姓名]，[性别]，[民族]，[出生日期]日出生，住[住址]，身份证号：[身份证号]

第三人（如有）：[姓名]，[性别]，[民族]，[出生日期]日出生，住[住址]

上诉人[姓名]因不服[原审法院名称]人民法院（[年份]）[案号类型][序号]号民事判决，向本院提起上诉。本院依法组成合议庭，公开开庭（或不开庭）审理了本案。[写明到庭参加诉讼的当事人及诉讼代理人]。本案现已审理终结。

## 案情概况

### 原审情况
[概括一审认定的事实和判决结果]

### 上诉主张
[上诉人的上诉请求及主要理由]

### 答辩意见
[被上诉人的主要辩解意见]

## 二审认定事实
经审理查明：
[详细写明二审法院根据证据认定的案件真实情况]

## 法院理由
本院认为：
[针对上诉请求和理由进行法律分析，评价原审判决，阐明维持/改判/发回重审的法理依据]

## 判决结果
依照《[法律名称]》第[条款]条[的规定]，判决如下：

[根据不同情形选择以下格式]

### 情形一：维持原判
驳回上诉，维持原判。

### 情形二：全部改判
一、撤销[原审法院名称]人民法院（[年份]）[案号类型][序号]号民事判决；
二、[改判后的具体内容]。

### 情形三：部分改判
一、维持[原审法院名称]人民法院（[年份]）[案号类型][序号]号民事判决第[项]项；
二、撤销[原审法院名称]人民法院（[年份]）[案号类型][序号]号民事判决第[项]项；
三、[改判后的具体内容]。

### 情形四：维持原判但加判
一、维持[原审法院名称]人民法院（[年份]）[案号类型][序号]号民事判决；
二、[增加的判处内容]。

二审案件受理费[金额]元，由[承担方]负担。

本判决为终审判决。

审判长    [姓名]
审判员    [姓名]
审判员    [姓名]

[年份]年[月份]月[日期]日
（院印）

书记员    [姓名]
本件与原本核对无异
```

## 案号类型说明

- 一审：民初
- 二审：民终
