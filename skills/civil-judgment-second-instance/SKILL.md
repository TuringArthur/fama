# 第二审人民法院民事判决书技能

本技能用于撰写符合中国法律规范要求的第二审人民法院民事判决书，适用于上诉案件。

## 核心结构

### 一、标题与案号

```
[法院全称]人民法院
民事判决书
（二审维持原判或改判用）

（××××）×民终××号
```

**格式要求：**
- 法院名称居中，占两行
- 案号位于标题右下方
- 案号格式：(年份)省简称+市简称+民终+序号

### 二、当事人信息

按以下顺序排列：
1. **上诉人**（原审诉讼地位）
   - 姓名、性别、民族、出生日期、住址、身份证号
   - 法人：名称、住所地、法定代表人及职务
2. **被上诉人**（原审诉讼地位）
   - 同上格式
3. **第三人**（如有）

### 三、审理经过

```
上诉人×××因不服××人民法院（××××）×民初××号民事判决，向本院提起上诉。本院依法组成合议庭，公开开庭（或不开庭）审理了本案。现已审理终结。
```

**必含要素：**
- 原审法院及案号
- 合议庭组成
- 审理方式
- 结案语

### 四、案情概况

**原审情况：** 概括一审认定的事实和判决结果

**上诉主张：** 简述上诉请求和主要理由

**答辩意见：** 被上诉人的主要辩解

### 五、二审认定事实

**格式：**
```
经审理查明：
[二审查明的事实]
```

**写作要点：**
- 重点针对上诉人提出的问题阐述
- 双方无分歧事实可概述或不写
- 引用证据支撑

### 六、法院理由

**格式：**
```
本院认为：
[法理分析]
```

**内容要求：**
- 回应上诉理由
- 分析上诉理由是否成立
- 评价原审判决的正确性
- 阐明维持/改判/发回重审的依据

### 七、判决结果

**四种情形及标准表述：**

#### 情形一：维持原判
```
驳回上诉，维持原判。
```

#### 情形二：全部改判
```
一、撤销××人民法院（××××）×民初××号民事判决；
二、[改判后的具体内容]。
```

#### 情形三：部分改判
```
一、维持××人民法院（××××）×民初××号民事判决第×项；
二、撤销××人民法院（××××）×民初××号民事判决第×项；
三、[改判后的具体内容]。
```

#### 情形四：维持原判但加判
```
一、维持××人民法院（××××）×民初××号民事判决；
二、[增加的判处内容]。
```

**诉讼费用：** 判决后写明二审诉讼费用负担

### 八、尾部

```
本判决为终审判决。

审判长    ××
审判员    ××
审判员    ××

××××年××月××日
（院印）

书记员    ××
```

**必含要素：**
- "本判决为终审判决"声明
- 合议庭成员署名
- 判决日期
- 院印
- 书记员署名

## 写作规范

### 语言要求
- 使用法律专业术语
- 表述严谨、中立
- 避免主观情感色彩

### 逻辑要求
- 程序交代→事实认定→法理分析→结论
- 针对上诉理由重点阐述
- 以理服人，分清是非

### 格式要求
- 层级分明，使用序号
- 占位符格式：××
- 法律条款引用完整

## 快速生成模板

当用户提供基本信息时，按以下结构生成：

1. **输入信息：**
   - 案号
   - 当事人信息
   - 原审案号及判决
   - 上诉请求
   - 上诉理由
   - 判决类型（维持/改判）

2. **输出结构：**
   - 完整判决书全文
   - 符合规范格式
   - 法条引用准确

## 常见错误避免

1. ❌ 遗漏"本判决为终审判决"
2. ❌ 案号格式错误
3. ❌ 改判情形未撤销原判
4. ❌ 遗漏诉讼费用分担
5. ❌ 合议庭成员署名不全
