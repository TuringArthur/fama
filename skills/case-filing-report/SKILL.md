# 立案报告生成技能

## 触发条件

当用户请求以下内容时触发本技能：
- 起草立案报告
- 编写刑事案件报告
- 生成公安机关公文
- 制作案件呈报文书

## 工作流程

### 1. 收集案件基本信息

在生成立案报告前，收集以下核心信息：

| 信息类别 | 具体内容 |
|---------|---------|
| **案件要素** | 案发时间、地点、案件类型、涉案人员 |
| **发现经过** | 报案人身份、报案时间、案发情形 |
| **现场情况** | 勘查结果、痕迹物证、损失情况 |
| **调查进展** | 已获取的线索、证人证言、排查情况 |
| **案情判断** | 案件性质分析、作案手段推断、嫌疑人特征 |
| **侦查计划** | 下一步工作方向、分工安排 |

### 2. 执行层 - 立案报告结构

生成的立案报告应包含以下五个核心部分：

#### (1) 报告标题

公文式标题，格式：
```
关于『[日期/案件特征]』[案由]的立案报告
```

示例：`关于『8·24』抢劫案的立案报告`

#### (2) 开头部分（简述）

简要交代：
- 报案时间
- 报案人基本情况
- 案发地点
- 初步确定的案件性质

#### (3) 正文五项内容

**一、发现经过**
- 案件如何被发现（报案、举报、主动发现）
- 报案人身份信息
- 发现时的现场情形

**二、现场勘查情况**
- 现场环境描述（地理位置、周围环境、天气）
- 现场状态（凌乱程度、进出口情况）
- 痕迹物证（指纹、足迹、工具痕迹、遗留物）
- 损失情况（财物名称、数量、价值）

**三、调查情况**
- 询问报案人/受害人获取的信息
- 走访目击者获取的线索
- 初步摸底排查情况

**四、案情分析**
- 案件性质认定
- 作案特点分析（手段、方式）
- 作案人特征刻画（年龄、性别、体态、职业背景）
- 动机分析（图财、报复、临时起意）
- 作案人数及分工推断

**五、侦查计划**
- 任务分工安排
- 重点侦查范围
- 具体侦查措施
- 时间安排

#### (4) 落款格式

```
呈报单位：[公安局/分局全称]
日  期：[XXXX年XX月XX日]
```

### 3. 格式规范

| 规范项 | 要求 |
|-------|------|
| **语言风格** | 严谨、客观、简练、准确 |
| **人称** | 采用第三人称叙述 |
| **时间表述** | 使用具体日期，避免模糊表述 |
| **证据描述** | 客观陈述，不添加主观判断 |
| **编号格式** | 使用"一、二、三"或"（一）（二）（三）" |

### 4. 输出格式

生成完整的立案报告文档，包含：
1. 规范的文件头（标题、主送机关）
2. 完整的五部分正文
3. 正式的落款

## 注意事项

1. **保密性**：立案报告属于内部机密文书，生成后提醒用户注意保密
2. **事实准确性**：所有描述必须基于用户提供的信息，如信息不全应主动询问
3. **法律规范**：符合《公安机关办理刑事案件程序规定》的要求
4. **文书编号**：可根据需要添加文书编号
