# 立案报告写作指南

## 一、立案报告概述

立案报告是公安机关在接受刑事案件后，经初步审查认为有犯罪事实需要追究刑事责任时，向有关领导或部门提交的请求批准立案的正式文书。

### 1.1 法律依据

根据《公安机关办理刑事案件程序规定》，公安机关接受案件后，经审查，认为有犯罪事实需要追究刑事责任的，应当制作立案报告书，报县级以上公安机关负责人批准。

### 1.2 文书性质

- **法律效力**：经批准后，正式启动刑事侦查程序
- **保密等级**：内部机密文书
- **存档要求**：永久保存于案件卷宗

## 二、立案报告结构详解

### 2.1 标题格式

```
关于『[案件特征/日期]』[案由]的立案报告
```

**命名规则**：
- 案件特征：可使用案件代号、案发时间、特殊情节等
- 案由：应使用刑法分则规定的具体罪名或案件类型

**示例**：
- 关于『8·24』抢劫案的立案报告
- 关于『12·5』杀人案的立案报告
- 关于某仓库被盗案的立案报告

### 2.2 主送机关

- 格式：居左顶格书写
- 内容：所属公安局/分局名称
- 示例：`呈报：XX省公安厅`

### 2.3 正文内容

#### （1）开头部分（简述）

简要概述案件基本情况，字数控制在100-200字：

```
XXXX年XX月XX日XX时，接到[报案人]报告（或发现），
[地点]发生[案件类型]。经初步审查，
认为[案件性质]。现将情况报告如下：
```

#### （2）发现经过

| 要素 | 内容要求 |
|-----|---------|
| 发现主体 | 报案人、举报人、单位或个人 |
| 发现时间 | 具体到年月日时分 |
| 发现地点 | 精确到具体位置 |
| 发现情形 | 原始状态描述，区分亲眼所见与转述 |

**写作要点**：
- 必须客观真实
- 区分第一手信息和转述信息
- 注明信息获取方式

#### （3）现场勘查情况

**环境描述**：
- 地理位置（行政区划、具体地址）
- 周围环境（交通、建筑布局）
- 当日天气情况
- 现场保护状态

**现场状态**：
- 进出口位置及状态
- 室内/室外凌乱程度
- 异常发现位置

**痕迹物证**：
- 指纹（位置、数量、清晰度）
- 足迹（数量、鞋型、大小）
- 工具痕迹（类型、位置）
- 遗留物（烟头、纸张、纤维等）

**损失情况**：
| 项目 | 格式 |
|-----|-----|
| 财物名称 | 具体物品名称 |
| 数量 | 数字+单位 |
| 价值 | 估算金额或实际价值 |

#### （4）调查情况

**报案人/受害人陈述**：
- 基本信息（姓名、性别、年龄、职业）
- 与案件关系
- 陈述内容（按时间顺序）
- 可信度评估

**证人证言**：
- 证人基本信息
- 证言内容
- 与案件关联性

**摸底排查**：
- 排查范围
- 排查方法
- 发现的可疑情况

#### （5）案情分析

**案件性质认定**：
- 是否构成犯罪
- 具体罪名
- 罪名依据

**作案特点**：
- 作案时间选择
- 作案手段分析
- 是否熟练（初犯/惯犯）
- 作案工具

**作案人特征刻画**：

| 特征维度 | 分析内容 |
|---------|---------|
| 性别 | 男性/女性/不确定 |
| 年龄 | 年龄段估计 |
| 身高 | 范围估计 |
| 体态 | 身材特征 |
| 人数 | 单独/团伙 |
| 职业背景 | 可能的职业 |
| 反侦查能力 | 是否有经验 |

**动机分析**：
- 图财型
- 报复型
- 情感纠纷型
- 临时起意型
- 其他

#### （6）侦查计划

**任务分工**：
```
总负责人：[职务/姓名]
现场组：[职责和成员]
走访组：[职责和成员]
情报组：[职责和成员]
```

**侦查范围**：
- 地域范围
- 人员范围
- 时间范围

**侦查措施**：
- 现场复勘
- 痕迹比对
- 摸底排查
- 设卡堵截
- 情报研判
- 技术侦查

**时间安排**：
- 各阶段完成时限
- 阶段性汇报时间

### 2.4 落款

```
呈报单位：[公安局/分局全称，带印章]
日  期：[XXXX年XX月XX日]
```

## 三、写作质量标准

### 3.1 语言规范

| 要求 | 说明 |
|-----|-----|
| **准确** | 用词精确，避免歧义 |
| **简练** | 语言简洁，删除冗余 |
| **客观** | 陈述事实，不加评判 |
| **规范** | 使用法律术语和公文用语 |

### 3.2 常见问题

**问题一：要素不全**
- 缺少案件发现时间
- 现场描述过于简略
- 损失情况不明确

**问题二：逻辑混乱**
- 时间顺序颠倒
- 内容重复
- 分析与事实混淆

**问题三：用词不当**
- 使用主观色彩词汇
- 法律术语错误
- 口语化表达

### 3.3 修改要点

1. 检查所有时间节点是否准确
2. 核实数字和人名是否一致
3. 确认分析结论有事实支撑
4. 验证侦查计划的可操作性
