# 第二审人民法院刑事判决书生成技能

## 触发条件

当用户请求以下内容时激活本技能：
- 撰写二审刑事判决书
- 生成刑事二审裁判文书
- 制作终审刑事判决
- 按照第二审程序制作刑事判决书
- 刑事案件二审改判
- 二审维持原判判决书

## 文书基本信息

### 文书定义
第二审刑事判决书是二审法院根据当事人的上诉或检察院的抗诉，依照二审程序，对一审法院尚未发生法律效力的判决或裁定进行审查后作出的书面决定。

### 法律依据
- **《刑事诉讼法》第227条**：规定上诉权
- **《刑事诉讼法》第228条**：规定抗诉权
- 二审法院应当进行全面审查，不受上诉或抗诉范围限制

### 效力说明
本判决为终审判决，判决一经宣告即发生法律效力。

## 文书结构

### 一、首部

#### 1. 文书标题
- 居中写明法院名称
- 文书种类"刑事判决书"居中

#### 2. 文书编号
- 格式：`(××)×刑终××号`
- 位于标题右下方
- "刑终"二字表明适用二审程序，为终审判决

#### 3. 当事人信息
- **原公诉机关**：原一审的检察院名称
- **上诉人（原审被告人）**：
  - 姓名
  - 性别
  - 出生年月日
  - 民族
  - 户籍所在地
  - 职业
  - 住址
  - 强制措施情况
  - 羁押地点
- **辩护人**：姓名、工作单位及职务

### 二、正文

#### 1. 案由、审理经过
- 原审法院名称及案号
- 原审判决结果
- 上诉或抗诉情况
- 二审法院组成合议庭情况
- 是否公开审理
- 检察院出庭情况
- 当事人到庭情况

#### 2. 事实
- 原判概述：事实、证据、理由和结果
- 上诉意见概述
- 检察院意见概述
- 二审经审理查明的事实
- 定案证据
- 针对上诉异议进行分析认证

#### 3. 理由（本院认为）
- 根据二审查明的事实和法律
- 论证原判是否正确
- 分析是否采纳上诉人、辩护人、检察员意见
- 阐明具体理由

### 三、判决结果

#### 1. 全部改判
```
依照……的规定，判决如下：
一、撤销……人民法院(××)×刑初××号刑事判决；
二、上诉人××犯……罪，判处……（写明具体刑罚）。
```

#### 2. 部分改判
```
依照……的规定，判决如下：
一、维持……人民法院(××)×刑初××号刑事判决第×项……；
二、撤销……人民法院(××)×刑初××号刑事判决第×项……；
三、上诉人××……（写明改判内容）。
```

#### 3. 维持原判
```
依照……的规定，判决如下：
驳回上诉，维持原判。
```

### 四、尾部

- 审判长：××
- 审判员：××
- 审判员：××
- ××××年××月××日
- （院印）
- 书记员：××

### 五、核对章
- 左下角标注：本件与原本核对无异

## 制作要点

### 格式要求
1. 使用标准法律文书格式
2. 占位符使用"××"表示
3. 括号内为撰写说明
4. 日期使用阿拉伯数字

### 内容要求
1. 先写撤销部分，再写维持部分
2. 注意区分"判决"与"判决书"的用语
3. 改判时应明确具体内容
4. 二审应全面审查，不受上诉范围限制

### 常用法律依据
- 《中华人民共和国刑法》第×条
- 《中华人民共和国刑事诉讼法》第×条

## 输出格式

请用户提供以下信息以生成完整的判决书：
1. 法院名称
2. 案号
3. 原审信息（原审法院、案号、判决结果）
4. 当事人信息（上诉人、辩护人）
5. 上诉理由
6. 二审查明的事实
7. 判决类型（全部改判/部分改判/维持原判）
8. 具体判决内容
