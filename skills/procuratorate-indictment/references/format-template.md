# 格式一：自然人犯罪案件普通程序适用起诉书

```
××人民检察院
起诉书
××检××刑诉〔 〕××号

被告人[姓名]，[性别]，[出生年月日]日出生，身份证号码：[号码]，
[民族]族，[文化程度]，职业或工作单位：[单位]，职务：[职务]。
户籍地：[地址]，住址：[地址]。

[曾用名/别名/化名/绰号]（如有）。

[前科情况：如曾受过刑事处罚或行政处罚，应写明时间、种类、决定机关及执行情况]。

[强制措施情况：因本案于××年××月××日被[强制措施种类]，现羁押于[处所]/被[取保候审/监视居住]于[处所]]。

[是否系人大代表/政协委员]。

本案由[侦查机关名称]侦查终结，以被告人[姓名]涉嫌[罪名]，于××年××月××日移送起诉。
本院于××年××月××日告知被告人有权委托辩护人，已告知被害人及其法定代理人有权委托诉讼代理人。
本院已听取辩护人/被害人及其诉讼代理人的意见。
[如系改变管辖/退回补充侦查/延长审查起诉期限等，应写明相应情况]。

经依法审查查明：
[详细叙述犯罪事实，包括时间、地点、动机、手段、后果等要素]

认定上述事实的证据如下：
[证人证言/物证/书证/鉴定意见/勘验笔录等]

本院认为，被告人[姓名]的行为触犯了《中华人民共和国刑法》第×条之规定，构成[罪名]，
[具有/不具有]从[重/轻]处罚情节，[认罪认罚情况]。
[如造成物质损失，应说明附带民事诉讼情况]。
根据《中华人民共和国刑事诉讼法》第一百七十六条的规定，本院提起公诉，请依法判处。

此致
××人民法院

检察官：×××
检察官助理：×××
××年××月××日
（院印）

附：
1. 被告人现羁押于[处所]/取保候审于[处所]
2. 案卷材料和证据共×册
3. 证人名单及主要证据复印件
4. 有关涉案款物情况
5. 被害人（单位）附带民事诉讼情况
[6. 认罪认罚具结书（认罪认罚案件）]
[7. 量刑建议书（认罪认罚案件）]
```

# 格式二：认罪认罚案件适用起诉书

```
××人民检察院
起诉书
××检××刑诉〔 〕××号

被告人[姓名]，[性别]，[出生年月日]日出生，身份证号码：[号码]，
[民族]族，[文化程度]，职业或工作单位：[单位]，职务：[职务]。
户籍地：[地址]，住址：[地址]。

[前科情况]

[强制措施情况]

[是否系人大代表/政协委员]

本案由[侦查机关]侦查终结，以被告人[姓名]涉嫌[罪名]，于××年××月××日移送起诉。
本院于××年××月××日受理。
本院已告知被告人认罪认罚的法律后果，已告知被害人及其法定代理人有权委托诉讼代理人。
本院已听取辩护人/值班律师/被害人及其诉讼代理人的意见。
被告人[姓名]于××年××月××日签署《认罪认罚具结书》。
[如适用简易程序/速裁程序，应写明]。

经依法审查查明：
[叙述犯罪事实，被告人到案后如实供述犯罪事实，自愿认罪认罚]

[如赔偿被害人损失并取得谅解，应写明]

认定上述事实的证据如下：
[证据清单]

本院认为，被告人[姓名]的行为触犯了《中华人民共和国刑法》第×条，构成[罪名]。
[具有从轻/减轻处罚情节]。被告人[认罪认罚]。
根据《中华人民共和国刑事诉讼法》第十五条、第一百七十六条的规定，
提起公诉，请依法判处。
[如提出量刑建议，应写明具体内容]。

此致
××人民法院

检察官：×××
检察官助理：×××
××年××月××日
（院印）

附：
1. 被告人现羁押于[处所]/取保候审于[处所]
2. 案卷材料和证据共×册
3. 证人名单及主要证据复印件
4. 《认罪认罚具结书》副本×份
5. 《量刑建议书》副本×份
6. 有关涉案款物情况
7. 被害人（单位）附带民事诉讼情况
```

# 格式三：单位犯罪案件普通程序适用起诉书

```
××人民检察院
起诉书
××检××刑诉〔 〕××号

被告单位：[单位名称]，组织机构代码：[代码]，住所地：[地址]。
法定代表人：[姓名]，职务：[职务]。
诉讼代表人：[姓名]，[性别]，[出生年月日]日出生，工作单位：[单位]，职务：[职务]。

直接负责的主管人员[姓名]，[性别]，[出生年月日]日出生，身份证号码：[号码]，
[民族]族，[文化程度]，职业或工作单位：[单位]，职务：[职务]。
户籍地：[地址]，住址：[地址]。

[前科情况]

[强制措施情况]

[其他直接责任人员信息，如有多人应分别写明]

本案由[侦查机关]侦查终结，以被告单位[名称]及被告人[姓名]涉嫌[罪名]，于××年××月××日移送起诉。
[后续程序同格式一/二]
```

# 格式五：刑事附带民事起诉书

```
××人民检察院
刑事附带民事起诉书
××检××刑附民诉〔 〕××号

被告人（刑事案件被告人）：[姓名]，[性别]，[出生年月日]日出生，[民族]族，
[文化程度]，职业或工作单位：[单位]，职务：[职务]。
户籍地：[地址]，住址：[地址]。

[被告单位信息，如有]

被害单位：[单位名称]，[所有制性质]，住所地：[地址]。
法定代表人：[姓名]，职务：[职务]。

诉讼请求：
[写明具体的诉讼请求，要求赔偿因犯罪行为造成的经济损失]

事实、证据和理由：
[概括犯罪事实及证据]
[说明被告人应承担民事责任的理由]
[引用法律依据：《中华人民共和国刑事诉讼法》第一百零一条第二款]

本院已于××年××月××日以××号起诉书提起公诉，现依法提起附带民事诉讼。

此致
××人民法院

检察官：×××
检察官助理：×××
××年××月××日
（院印）

附：
1. 刑事附带民事起诉书副本×份
2. 其他需要附注的事项
```
