# 人民检察院起诉书示例

## 示例一：盗窃罪起诉书

```
北京市朝阳区人民检察院
起诉书
京朝检刑诉〔2024〕0156号

被告人张伟，男，1990年5月12日出生，身份证号码：110105199005123456，
汉族，大学本科文化，无业人员。户籍地：北京市朝阳区建国路1号，
住址：北京市朝阳区双桥东路5号。

前科情况：无。

强制措施情况：因本案于2024年3月15日被北京市公安局朝阳分局刑事拘留，
于2024年3月22日被北京市朝阳区人民检察院批准逮捕，现羁押于朝阳区看守所。

本案由北京市公安局朝阳分局侦查终结，以被告人张伟涉嫌盗窃罪，
于2024年4月10日移送起诉。本院于2024年4月12日受理。
本院于2024年4月12日告知被告人有权委托辩护人，已告知被害人有权委托诉讼代理人。
本院已听取辩护人的意见。

经依法审查查明：
2024年3月5日零时许，被告人张伟在北京市朝阳区三里屯酒吧街附近，
趁被害人李明不备，将其放在外套口袋内的苹果手机（型号iPhone 15 Pro）
窃走，后被当场抓获。经鉴定，被盗手机价值人民币8500元。
被告人张伟到案后如实供述了上述犯罪事实。

认定上述事实的证据如下：
1. 被害人李明的陈述
2. 证人王红的证言
3. 被告人张伟的供述和辩解
4. 被盗手机物证照片
5. 价格鉴定意见书
6. 案发现场监控录像

本院认为，被告人张伟以非法占有为目的，秘密窃取他人财物，价值人民币8500元，
数额较大，其行为触犯了《中华人民共和国刑法》第二百六十四条之规定，
构成盗窃罪。被告人张伟到案后如实供述犯罪事实，具有坦白情节，
根据《中华人民共和国刑法》第六十七条第三款的规定，可以从轻处罚。
被告人认罪认罚。根据《中华人民共和国刑事诉讼法》第十五条、第一百七十六条的规定，
本院提起公诉，请依法判处。

此致
北京市朝阳区人民法院

检察官：王建军
检察官助理：刘芳
2024年4月25日
（院印）

附：
1. 被告人现羁押于朝阳区看守所
2. 案卷材料和证据共2册
3. 证人名单及主要证据复印件
4. 认罪认罚具结书副本1份
5. 量刑建议书副本1份
```

## 示例二：故意伤害罪起诉书（认罪认罚）

```
上海市浦东新区人民检察院
起诉书
沪浦检刑诉〔2024〕0234号

被告人李强，男，1985年8月20日出生，身份证号码：310115198508203456，
汉族，初中文化，个体工商户。户籍地：上海市浦东新区潍坊新村1号，
住址：上海市浦东新区潍坊新村1号。

前科情况：2015年3月因赌博被上海市公安局浦东分局行政拘留5日。

强制措施情况：因本案于2024年2月10日被上海市公安局浦东分局刑事拘留，
于2024年2月17日被本院批准逮捕，现羁押于浦东新区看守所。

本案由上海市公安局浦东分局侦查终结，以被告人李强涉嫌故意伤害罪，
于2024年3月5日移送起诉。本院于2024年3月6日受理。
本院已告知被告人认罪认罚的法律后果，已告知被害人及其法定代理人有权委托诉讼代理人。
本院已听取辩护人的意见。
被告人李强于2024年3月8日签署《认罪认罚具结书》。

经依法审查查明：
2024年2月8日21时许，被告人李强在上海市浦东新区陆家嘴环路一餐厅内，
因琐事与被害人张华发生口角，后李强用啤酒瓶击打张华头部，致张华头部外伤、
颅内出血。经鉴定，被害人张华的损伤程度为轻伤一级。
被告人李强到案后如实供述犯罪事实。案发后，李强的家属代为赔偿被害人张华
经济损失人民币15万元，已取得被害人谅解。

认定上述事实的证据如下：
1. 被害人张华的陈述
2. 证人赵某、刘某的证言
3. 被告人李强的供述和辩解
4. 法医鉴定意见书
5. 现场勘验笔录
6. 赔偿协议及谅解书

本院认为，被告人李强故意伤害他人身体，致一人轻伤一级，
其行为触犯了《中华人民共和国刑法》第二百三十四条第一款之规定，
构成故意伤害罪。被告人李强到案后如实供述犯罪事实，认罪认罚，
且已赔偿被害人经济损失并取得谅解，具有从轻处罚情节。
根据《中华人民共和国刑事诉讼法》第十五条、第一百七十六条的规定，
本院提起公诉，请依法判处。

此致
上海市浦东新区人民法院

检察官：陈明
检察官助理：周琳
2024年3月20日
（院印）

附：
1. 被告人现羁押于浦东新区看守所
2. 案卷材料和证据共3册
3. 证人名单及主要证据复印件
4. 认罪认罚具结书副本1份
5. 量刑建议书副本1份
```

## 示例三：受贿罪起诉书（职务犯罪）

```
北京市人民检察院第一分院
起诉书
京检一分刑诉〔2024〕0089号

被告人王建国，男，1970年3月15日出生，身份证号码：110101197003156789，
汉族，硕士研究生文化，原系北京市规划和自然资源委员会副主任（正处级）。
户籍地：北京市西城区复兴门内大街1号，住址：北京市朝阳区团结湖小区5号楼。

被告人王建国于2024年1月20日被北京市监察委员会留置，
于2024年3月20日被本院决定逮捕，现羁押于北京市第一看守所。

本案由北京市监察委员会调查终结，以被告人王建国涉嫌受贿罪，
于2024年3月25日移送起诉。本院于2024年3月26日受理。
本院已告知被告人有权委托辩护人。

经依法审查查明：
一、2015年至2020年，被告人王建国在担任北京市规划和自然资源委员会用地处处长期间，
利用职务便利，为北京某房地产公司开发的项目在土地规划审批方面提供帮助，
先后收受该公司董事长赵某给予的现金人民币共计200万元。

二、2018年至2023年，被告人王建国利用担任北京市规划和自然资源委员会副主任的职务便利，
为个体商人钱某在办理土地使用权转让手续方面提供帮助，先后收受钱某给予的现金
人民币共计150万元，以及价值人民币50万元的房产一套。

三、2020年至2023年，被告人王建国利用职务便利，为某规划设计院在承接规划设计项目方面提供帮助，
先后收受该院院长孙某给予的现金人民币共计80万元。

综上，被告人王建国利用职务便利，收受他人财物共计人民币480万元，
以及价值人民币50万元的房产一套。

认定上述事实的证据如下：
1. 证人赵某、钱某、孙某的证言
2. 被告人王建国的供述和辩解
3. 银行转账记录、房产交易文件
4. 土地规划审批文件
5. 会计师事务所出具的审计报告
6. 搜查笔录、扣押清单

本院认为，被告人王建国身为国家工作人员，利用职务上的便利，
非法收受他人财物共计人民币530万元，数额特别巨大，为他人谋取利益，
其行为触犯了《中华人民共和国刑法》第三百八十五条第一款、第三百八十六条之规定，
构成受贿罪。被告人王建国的受贿行为严重侵害了国家工作人员职务的廉洁性，
造成了恶劣的社会影响，应当从重处罚。被告人王建国到案后如实供述犯罪事实，
认罪认罚，具有从轻处罚情节。根据《中华人民共和国刑事诉讼法》第十五条、
第一百七十六条的规定，本院提起公诉，请依法判处。

此致
北京市第一中级人民法院

检察官：刘志刚
检察官助理：李晓燕
2024年4月15日
（院印）

附：
1. 被告人现羁押于北京市第一看守所
2. 案卷材料和证据共12册
3. 证人名单及主要证据复印件
4. 冻结款物清单
5. 认罪认罚具结书副本1份
6. 量刑建议书副本1份
```
