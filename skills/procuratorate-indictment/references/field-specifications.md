# 人民检察院起诉书字段说明

## 一、首部字段

### 1. 检察院名称
- **要求**：地方各级检察院需冠以省（自治区、直辖市）名称
- **涉外案件**：需注明"中华人民共和国"字样
- **最高检**：直接使用"最高人民检察院"

### 2. 文号格式
- **格式**：`院简称 + 刑诉 + [起诉年度] + 案件顺序号`
- **示例**：`京检刑诉〔2024〕015号`
- **要求**：
  - 年度必须使用4位数字
  - 文号位于该行最右端
  - 上下各空一行

## 二、被告人基本情况字段

### 自然人被告人

| 字段名称 | 说明 | 填写要求 |
|---------|------|---------|
| 姓名 | 包括曾用名、别名、化名、绰号 | 需注在括号内 |
| 性别 | 男/女 | 必填 |
| 出生年月日 | 公历为准 | 必填 |
| 身份证号码 | 18位身份证号 | 必填 |
| 民族 | 如"汉族" | 必填 |
| 文化程度 | 如"大学本科"、"高中"等 | 必填 |
| 职业 | 具体的职业类别 | 必填 |
| 工作单位 | 所在工作单位全称 | 无工作单位可填"无" |
| 职务 | 在单位担任的具体职务 | 无职务可填"无" |
| 户籍地 | 户籍登记地址 | 必填 |
| 经常居住地 | 实际居住地址 | 与户籍地不同时填写 |
| 前科情况 | 曾受行政处罚或刑事处罚 | 重点记录与本案相关的 |
| 强制措施 | 种类、执行机关、时间 | 必填 |
| 人大代表/政协委员 | 是否担任相关职务 | 如是需注明 |

### 单位被告人

| 字段名称 | 说明 | 填写要求 |
|---------|------|---------|
| 单位名称 | 企业/组织的全称 | 必填 |
| 组织机构代码 | 统一社会信用代码 | 必填 |
| 住所地 | 注册地址 | 必填 |
| 法定代表人 | 姓名、职务 | 必填 |
| 诉讼代表人 | 姓名、性别、出生日期、工作单位、职务 | 必填 |
| 直接负责的主管人员 | 姓名及基本情况 | 必填 |
| 其他直接责任人员 | 姓名及基本情况 | 如有需填写 |

### 特殊情形处理

| 情形 | 处理方式 |
|-----|---------|
| 外国籍被告人 | 注明国籍、护照号码、居留证件 |
| 未成年人 | 核实出生日期（公历为准） |
| 多名被告人 | 按主从关系顺序叙写 |
| 身份不明 | 注明现有情况 |

## 三、案由和案件来源字段

| 字段名称 | 说明 | 填写要求 |
|---------|------|---------|
| 案由 | 涉嫌的罪名 | 必填 |
| 案件来源 | 本院自侦/公安机关移送 | 必填 |
| 侦查终结机关 | 负责侦查的机关名称 | 必填 |
| 移送起诉日期 | 案件移送检察院的日期 | 必填 |
| 检察院受理日期 | 检察院正式受理的日期 | 必填 |
| 告知权利日期 | 告知辩护权等的日期 | 必填 |
| 听取意见情况 | 听取辩护人/被害人意见 | 如有需填写 |
| 认罪认罚情况 | 签署具结书日期等 | 认罪认罚案件必填 |

## 四、案件事实字段

### 五要素

| 要素 | 说明 |
|-----|------|
| 时间 | 犯罪发生的具体时间 |
| 地点 | 犯罪发生的具体地点 |
| 动机 | 犯罪的动机和目的 |
| 手段 | 犯罪的具体方式和经过 |
| 后果 | 犯罪造成的损害结果 |

### 叙写方法

1. **按时间顺序**：适用于单一犯罪事实
2. **突出主罪法**：先述主罪、重罪，后述次罪
3. **综合归纳法**：适用于被告人作案次数多且性质相同
4. **罪证分述**：使罪行、证据分明

### 共同犯罪叙写

- 分清各被告人的地位和作用
- 明确主从犯关系
- 分别叙述各自罪责

## 五、证据字段

| 证据类型 | 说明 |
|---------|------|
| 物证 | 赃物、作案工具等 |
| 书证 | 合同、票据、文件等 |
| 证人证言 | 证人就案件情况所作陈述 |
| 被害人陈述 | 被害人就案件情况所作陈述 |
| 被告人供述和辩解 | 被告人的供述和辩解 |
| 鉴定意见 | 专门机构出具的鉴定结论 |
| 勘验笔录 | 现场勘验、检查笔录 |
| 视听资料 | 录音、录像、电子数据 |

### 叙写方式
- **一事一证**：在每一宗犯罪事实后写明主要证据
- **概括列举**：所有事实后概括写明主要证据种类

## 六、起诉理由和根据字段

| 字段名称 | 说明 |
|---------|------|
| 行为定性 | 被告人的行为构成什么性质的罪行 |
| 社会危害性 | 行为的社会危害性大小 |
| 法律依据 | 根据何项法律规定 |
| 从重情节 | 如实坦白、累犯等 |
| 从轻情节 | 自首、立功、赔偿等 |
| 减轻情节 | 从犯、未成年等 |
| 认罪认罚 | 是否认罪认罚 |
| 附带民事 | 如造成物质损失应说明 |

## 七、尾部字段

| 字段名称 | 说明 |
|---------|------|
| 致送机关 | 此致 ××人民法院 |
| 检察官 | 承办案件的检察官姓名 |
| 检察官助理 | 检察官助理姓名 |
| 日期 | 签发日期（年月日） |
| 院印 | 加盖检察院公章 |

| 3 | 证人、鉴定人名单及主要证据复印件 |
| 4 | 有关涉案款物情况 |
| 5 | 被害人（单位）附带民事诉讼情况 |
| 6 | 认罪认罚具结书（认罪认罚案件） |
| 7 | 量刑建议书（认罪认罚案件） |
