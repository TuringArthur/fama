# 人民检察院起诉书生成技能

本技能用于生成符合中国检察机关规范的人民检察院起诉书。

## 一、起诉书概述

起诉书是人民检察院代表国家向人民法院提起公诉的正式法律文书，适用于：
- 普通程序（自然人犯罪）
- 认罪认罚从宽程序
- 单位犯罪案件
- 刑事附带民事诉讼

## 二、文书结构（七大部分）

### （一）首部
- 检察院名称（冠以省/自治区/直辖市名称）
- 文号格式：`院简称 + 刑诉 + [起诉年度] + 案件顺序号`
- 年度使用4位数字，文号右对齐

### （二）被告人基本情况

**自然人被告人字段：**
- 姓名（曾用名、别名、化名、绰号）
- 性别、出生年月日、身份证号码
- 民族、文化程度
- 职业或工作单位及职务
- 是否系人大代表/政协委员
- 户籍地、经常居住地
- 前科情况（行政处罚/刑事处罚）
- 强制措施情况（种类、执行机关、时间）

**单位被告人字段：**
- 单位名称、组织机构代码、住所地
- 法定代表人姓名、职务
- 诉讼代表人情况
- 直接负责的主管人员/其他直接责任人员

### （三）案由和案件来源
- 案由（涉嫌罪名）
- 案件来源（本院自侦/公安机关移送）
- 审查时间轴：侦查终结→移送起诉→受理日期
- 程序告知：告知辩护权、诉讼代理人权
- 听取意见情况

### （四）案件事实
**事实要素（五何）：**
- 时间
- 地点
- 动机
- 手段（具体经过）
- 后果

**叙写方法：**
- 按犯罪时间顺序
- 或突出主罪
- 或综合归纳法（适用于次数多且性质相同）
- 罪证分述

### （五）证据
- 主要证据名称、种类
- 采用"一事一证"方法
- 或概括列举证据种类

### （六）起诉的要求和根据
**法律评价内容：**
- 行为定性（罪名）
- 社会危害性
- 法律依据（具体条款）
- 情节考量（从重/从轻/加重/减轻）
- 附带民事诉讼（如有）

### （七）尾部
- 致送机关：`此致 ××人民法院`
- 承办检察官、检察官助理署名
- 日期（年月日）
- 院印

### 附项
1. 被告人现羁押/处置场所
2. 案卷材料和证据册数
3. 证人、鉴定人名单
4. 有关涉案款物情况
5. 被害人附带民事诉讼情况
6. 认罪认罚具结书（认罪认罚案件）
7. 量刑建议书（认罪认罚案件）

## 三、制作要求

### 四大基本要求
1. **细致**：交代清楚时间、地点、人物、事件、原因、后果
2. **准确**：事实认定准确，定性准确，避免模糊用语
3. **全面**：从重从轻情节兼顾，客观公正
4. **一致**：观点统率材料，材料说明观点

### 禁止事项
- 避免主观色彩浓重的词汇（如"令人发指"、"血债累累"）
- 避免时间表达不协调
- 避免定性模糊

## 四、法律依据

- 《中华人民共和国刑事诉讼法》第一百七十六条
- 《中华人民共和国刑法》（相关条款）
- 《人民检察院工作文书格式样本（2020年版）》
- 《人民检察院刑事诉讼法律文书格式样本（2020年版）》

## 五、输出格式

生成的人民检察院起诉书应包含：
1. 完整的七大部分结构
2. 所有必需字段信息
3. 符合格式规范的法律用语
4. 准确的法律条款引用
