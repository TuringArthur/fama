# 提请减刑建议书格式与示例参考

## 一、概念界定

### 1.1 文书定义

提请减刑建议书是监狱依照法定程序，对符合法定减刑条件的罪犯向人民法院提出减刑建议时所制作的文书。

### 1.2 主要作用

| 作用 | 说明 |
|------|------|
| 请求裁定 | 要求人民法院依法对罪犯裁定减刑 |
| 体现政策 | 体现党和国家对罪犯采取的惩办与宽大相结合政策 |
| 促进改造 | 促使罪犯接受改造，改恶从善，重新做人 |

---

## 二、格式规范（5.1.1）

### 2.1 首部

#### （1）标题

```
××××监狱
提请减刑建议书
```

分上下行居中书写。

#### （2）文书编号

在标题右下方书写：
```
（××××）×监减字×号
```

格式说明：
- 立案年度
- 机关代字
- 文书序号

### 2.2 正文

#### （1）提请的事由

```

该犯近期确有××××表现，具体事实如下：
```

表述方式：
- 悔改表现
- 立功表现
- 或两者兼有

#### （2）悔改或立功表现的具体事实（重点内容）

```

（三）学习教育情况

考试成绩优良/合格。……

二、劳动改造方面

（一）劳动态度
该犯积极参加生产劳动，劳动态度端正，不怕脏、不怕累，
能够保质保量完成劳动任务。……

（二）劳动成果
该犯在××车间从事××劳动，能够熟练掌握生产技能，
月均完成劳动定额×××%，年累计完成劳动产值×××元。……

三、立功表现（如有）

（一）检举揭发
该犯在服刑期间，积极检举揭发他犯违规违纪行为，
经查证属实×起，有效维护了监管秩序。……

（二）防止事故
该犯在××年××月××日，及时报告/制止了××事故/事件，
避免了事故扩大/人员伤亡。……

（三）重大发明创造/技术革新
该犯在劳动改造中，积极钻研业务，攻克技术难题，
发明了×××，提高了生产效率。……

（四）抢险救灾
该犯在××情况下，不顾个人安危，积极参与抢险救灾，
为保护国家财产/他人安全作出贡献。……

（五）其他立功表现
该犯在××方面表现突出，具体事实为：……
```

#### （3）减刑理由

```
减刑理由

根据上述事实，该犯在服刑期间表现良好，具体表现为：

一、该犯能够认罪悔罪，深刻反省自己的违法犯罪行为，
    主观恶性得到有效矫正；

二、该犯能够严格遵守监规纪律和各项管理制度，
    无任何违规违纪行为，改造态度端正；

三、该犯积极参加政治学习和文化学习，
    政治觉悟和文化水平得到提高；

四、该犯积极参加生产劳动，劳动技能得到提升，
    能够完成或超额完成劳动任务；

五、该犯在服刑期间有重大立功/悔改表现，符合减刑条件。

综上所述，该犯在服刑期间认罪悔罪，遵规守纪，积极改造，
符合《中华人民共和国监狱法》规定的减刑条件，
依法应当予以减刑。
```

#### （4）法律依据及监狱意见

```

根据《中华人民共和国监狱法》第二十九条关于"被判处无期徒刑、

考核的结果，可以减刑"的规定，以及《中华人民共和国刑事诉讼法》
第二百七十三条等相关规定，该犯符合减刑条件。

为此，根据《中华人民共和国监狱法》第XX条、《中华人民共和国
刑事诉讼法》第XXX条第X款的规定，建议对罪犯XXX予以减刑。
特提请裁定。
```

## 三、写作注意事项（5.1.2）

### 3.1 叙述事实的基本要求

| 要求 | 说明 |
|------|------|
| 实事求是 | 所写的事实材料必须属实，准确可靠，没有差错 |
| 符合法定条件 | 突出重点，抓住重要的、典型的事例叙述 |
| 突出关键情节 | 突出关键情节或主要立功事实 |
| 避免程式化 | 避免用套话或程式化的语言去代替具体事实的叙述 |

### 3.2 事实要素

叙述事实时要注意将以下要素写清楚：

| 要素 | 说明 |
|------|------|
| 时间 | 事实发生的具体时间 |
| 地点 | 事实发生的具体地点 |
| 人物 | 涉及的相关人员 |
| 主要情节 | 事件的具体经过 |
| 原因 | 事件发生的原因 |
| 结果 | 事件产生的结果/影响 |

### 3.3 悔改表现与立功表现的区别

| 类型 | 内容 |
|------|------|
| 悔改表现 | 认罪悔罪、遵规守纪、学习教育、劳动改造等方面的日常表现 |
| 立功表现 | 检举揭发、防止事故、发明创造、抢险救灾等特殊贡献 |

### 3.4 减刑的法定条件

根据《监狱法》第二十九条规定：

```
减刑条件：

一、确有悔改表现
   （一）认罪悔罪
   （二）遵守监规纪律
   （三）接受教育改造
   （四）参加劳动生产

二、立功表现
   （一）阻止他人犯罪活动
   （二）检举监狱内外犯罪活动
   （三）有发明创造或重大技术革新
   （四）舍己救人
   （五）在抗御自然灾害中起作用
   （六）其他有利于国家和社会的突出贡献

三、重大立功表现（应当减刑）
   （一）阻止他人重大犯罪活动
   （二）检举重大犯罪活动
   （三）有发明创造或重大技术革新
   （四）舍己救人
   （五）在抗御自然灾害中起作用
   （六）其他有利于国家和社会的突出贡献
```

## 五、常见问题解答

### Q1：减刑建议书的法律依据有哪些？

- 《中华人民共和国监狱法》第二十九条、第三十条
- 《中华人民共和国刑事诉讼法》第二百七十三条
- 最高人民法院关于办理减刑、假释案件具体应用法律的规定

### Q2：悔改表现包括哪些方面？

- 认罪悔罪
- 遵守监规纪律
- 接受教育改造
- 参加劳动生产
- 政治学习和文化学习

### Q3：立功表现包括哪些情形？

- 阻止他人犯罪活动
- 检举监狱内外犯罪活动
- 有发明创造或重大技术革新
- 舍己救人
- 在抗御自然灾害中起作用
- 其他有利于国家和社会的突出贡献

### Q4：减刑的幅度有何限制？

根据法律规定：
- 减刑以后实际执行的刑期，判处管制、拘役、有期徒刑的，
  不能少于原判刑期的二分之一
- 判处无期徒刑的，不能少于十三年

### Q5：提请减刑的间隔时间？

- 被判处有期徒刑的罪犯，两次减刑之间一般应当间隔一年以上
- 被判处无期徒刑的罪犯，一般应当二年以上方可减刑
- 特殊情况下，可以不受上述期限限制
