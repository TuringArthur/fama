# 监狱提请减刑建议书撰写技能

## 适用场景

- 监狱对符合法定减刑条件的罪犯提请减刑
- 为罪犯准备减刑申请材料
- 制作罪犯服刑改造表现材料

## 文书性质

提请减刑建议书是监狱依照法定程序，对符合法定减刑条件的罪犯向人民法院提出减刑建议时所制作的文书。

## 核心要素

### 1. 首部

- **标题**：制作机关名称 + "提请减刑建议书"（分行居中）
- **文书编号**：（年度）×监减字×号
- **罪犯基本情况**：
  - 姓名、性别、出生日期、民族、籍贯
  - 原判罪名、原判法院、判决日期、判决案号
  - 附加刑情况、刑期、收监日期
  - 刑种/刑期/服刑场所变动情况（如有）

### 2. 正文

#### 提请事由
概括说明罪犯因何故提请减刑

#### 悔改/立功表现（重点内容）
- 具体、详细叙述
- 符合法定条件
- 突出重点和典型事例
- 关键情节或主要立功事实

#### 减刑理由
- 根据事实进行分析评论
- 写出结论性意见
- 引用法律法规进行分析说理

#### 法律依据及监狱意见
- 《监狱法》相关条款
- 《刑事诉讼法》相关条款
- 明确提请减刑建议

### 3. 尾部

- **主送机关**："此致" + "××××人民法院"
- **落款**：监狱名称、盖公章、签发日期
- **附件**：罪犯服刑改造档案卷数、册数、页数

## 事实叙述要求

| 要求 | 说明 |
|------|------|
| 实事求是 | 事实材料必须属实，准确可靠 |
| 符合法定条件 | 突出重点，抓住典型事例 |
| 要素完整 | 时间、地点、人物、经过、原因、结果 |

## 输出格式

生成完整的提请减刑建议书，包含：
- 标准的监狱文书格式
- 完整的三部分结构
- 详细的悔改/立功表现事实
- 充分的法律依据论证
